# 文山壮族苗族自治州概况（2025年更新）

> 数据来源：**文山州2025年国民经济和社会发展统计公报**（文山州统计局，2025年4月发布）
> 最新更新：2026年6月 — 补充十五五战略定位与目标展望

## 战略定位（"3+3"发展定位）

迈入"十五五"，文山州确立 **"3+3"发展定位**（2026年6月新闻发布会明确）：

| 定位 | 说明 |
|------|------|
| 民族团结进步示范区 | 深化全域创建"十进"，推进平远片区"三区"建设 |
| 石漠化地区生态文明建设排头兵 | 持续推进石漠化综合治理，筑牢滇东南生态安全屏障 |
| 云南面向东南开放先行区 | 融入西部陆海新通道，富宁港+沿边铁路+3个国家级口岸 |
| 新时代兴边富民示范区 | "两增长三提升"三年行动，438公里边境线全域覆盖 |
| "绿色铝谷"核心区 | 343万吨电解铝产能，2030年铝产业综合产值1800亿 |
| 三七康养旅居目的地 | 千亿级三七产业+康养旅游旅居品牌 |

## 十五五目标展望

| 领域 | 目标 |
|------|------|

## "十四五"成就标注

- 地区生产总值突破 **1600 亿元**（2025年，统计口径 GDP 1632.64 亿元）
- 绿色铝培育为**首个千亿级产业**（2025年产值突破 1000 亿元）
- 文山三七入选**全国优势特色产业集群**和**全国中小企业特色产业集群**
- 创成**全国民族团结进步示范州**、**全国双拥模范城**
- 石漠化治理经验获评**第五届全球最佳生态减贫案例**
