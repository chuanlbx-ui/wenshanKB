# 文山壮族苗族自治州2025年国民经济和社会发展统计公报

> 来源：文山州统计局 | 发布日期：2026年

## 一、综合 — 核心数据

| 指标 | 数值 |
|------|------|
| GDP | **1632.64 亿元** |
| GDP 增速 | **+5.3%** |
| 常住人口 | **335.4 万人** |
| 城镇化率 | **41.19%**（城镇 138.1 万 / 乡村 197.3 万） |
| 性别比 | 男性 51.34%，女性 48.66% |
| 60 周岁及以上 | 61.6 万人（18.37%） |
| 65 周岁及以上 | 39.3 万人（11.72%） |

- 居民消费价格总水平较上年下降 **0.4%**
