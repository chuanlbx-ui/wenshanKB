# 文山近现代大事记（1840—1992）

> 从鸦片战争到中越关系正常化，记录文山一个半世纪的关键历史节点。

---

## 一、晚清时期（1840—1911）

| 年份 | 事件 |
|------|------|
| 1840 | 鸦片战争爆发，文山地处西南边疆，未直接卷入，但清朝国力衰落的影响逐步传导至边疆 |
| 1856—1873 | 云南杜文秀起义期间，文山地区受到波及，地方团练兴起 |
| 1883—1885 | 中法战争，文山南部边境成为军事前哨；战后中法划界，都龙、猛硐等地经勘界归中国 |
| 1887 | 中法《续议界务专条》签订，划定滇越边界文山段 |
| 1895 | 中法《续议界务专条附章》进一步明确边界，麻栗坡设副督办管理边境事务 |
| 1901 | 富州厅设立（今富宁县） |
| 1905 | 清末新政，文山地区开始兴办新式学堂 |
| 1911 | 辛亥革命爆发，云南响应"重九起义"，文山随之光复 |

---

## 二、民国时期（1912—1949）

| 年份 | 事件 |
|------|------|
| 1913 | 全国废府存县；文山地区设文山县、马关县、广南县等 |
| 1914 | 安平县改名为马关县 |
| 1920 | 析文山县、马关县地设西畴县 |
| 1927 | **中共在文山建立党组织**，富宁、广南等地成为滇黔桂革命根据地的重要组成部分 |
| 1929—1930 | 邓小平、张云逸等领导百色起义，红七军开辟右江革命根据地，文山东部（富宁、广南）受其影响，成为革命游击区 |
| 1933 | 设砚山设治局 |
| 1935 | 砚山正式设县 |
| 1937 | 富州县改名为富宁县 |
| 1937—1945 | 抗日战争期间，文山虽处大后方，但作为滇越通道承担了大量物资转运和支前任务 |
| 1942 | 设云南省第二行政督察区，驻文山县 |
| 1946—1949 | 解放战争期间，中共滇桂黔边纵队在文山地区开展游击斗争 |

---

## 三、建国初期至改革开放前（1949—1978）

| 年份 | 事件 |
|------|------|
| 1949.12 | 文山地区和平解放（部分县经边纵武装解放） |
| 1950 | 设文山专区，辖文山、砚山、西畴、马关、丘北、广南、富宁、麻栗坡 8 县 |
| 1950—1954 | **援越抗法**：文山作为支援越南抗击法国殖民者的重要后方基地，提供物资、医疗和交通保障 |
| 1950—1952 | 土地改革、清匪反霸，文山地区社会秩序重建 |
| 1958.4.1 | **文山僮族苗族自治州成立**，辖 8 县 |
| 1958—1960 | "大跃进"期间，文山也经历了浮夸风和后续的困难时期 |
| 1964—1973 | **援越抗美**：文山再次成为支援越南抗击美国的前沿基地，大量部队和物资经文山中转 |
| 1965 | "僮族"改称"壮族"，自治州更名为**文山壮族苗族自治州** |
| 1966—1976 | "文革"期间，文山经济社会秩序受到严重冲击 |

---

## 四、改革开放与边境战时时期（1978—1992）

| 年份 | 事件 |
|------|------|
| 1978 | 十一届三中全会开启改革开放，但文山因边境战事未能同步推进 |
| 1979.2 | **对越自卫还击战**爆发，文山作为主战场之一，老山、者阴山成为焦点 |
| 1980—1989 | 两山轮战持续，文山处于"支前参战"最前沿 |
| 1984 | 老山战役——对越自卫还击战中最著名的战役之一，文山军民全力支前 |
| 1990 | 边境局势逐步缓和 |
| 1991 | 中越关系正常化谈判取得突破 |
| **1992** | **中越关系正式正常化**，文山从此结束长达近半个世纪的战争与备战状态，全面转入经济建设轨道 |

---

## 五、"晚了 14 年"的改革开放

文山因地处对越斗争最前沿，经历了从 **1950 年援越抗法 → 1964 年援越抗美 → 1979 年对越自卫还击战** 长达 40 余年的连续战争和备战。这导致：

- 1978 年全国开启改革开放时，文山正值边境作战最激烈时期
- 国家投资和基础设施建设长期向内地倾斜，文山错失发展机遇
- 直到 **1992 年中越关系正常化**，文山才真正转入以经济建设为中心的轨道
- 改革开放比全国整整晚了 **14 年**

---

## 六、老山精神

在长期的边境斗争中，文山军民铸就了以 **"艰苦奋斗、无私奉献、不怕苦、不怕死、不怕亏"** 为核心的**"老山精神"**，被列为中国革命精神谱系的重要组成部分。
