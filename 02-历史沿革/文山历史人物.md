# 文山州历史人物

## 一、概述

文山州虽地处西南边陲，但自古以来人文荟萃。从明清科举进士到近代革命先驱，从对越自卫还击战的战斗英雄到当代杰出人物，文山儿女在历史长河中书写了壮丽篇章。

## 四、主要人物传略

### 4.1 楚图南（1899-1994）

楚图南是文山州最具全国影响力的近现代历史人物。

**生平大事**：

| 年份 | 事件 |
|------|------|
| 1899年 | 生于文山县城一个书香门第 |
| 1919年 | 考入北京高等师范学校（现北京师范大学），参加五四运动 |
| 1923-1926年 | 在云南、东北等地任教，传播进步思想 |
| 1926年 | 加入中国共产党 |
| 1930年代 | 被捕入狱，坚贞不屈；出狱后从事文化教育工作 |
| 1945年 | 加入中国民主同盟 |
| 1949年后 | 历任西南军政委员会文教部部长、中国人民对外文化协会会长 |
| 1986年 | 当选全国人大常委会副委员长 |
| 1994年 | 在北京逝世，享年95岁 |

**主要贡献**：
- **文学翻译**：翻译了大量俄国和苏联文学作品，包括惠特曼《草叶集》选译
- **外交事业**：长期从事对外文化交流工作，曾陪同周恩来总理出访多国
- **民主事业**：中国民主同盟的重要组织者和领导人
- **学界影响**：为文山州教育事业发展提供精神标杆

**故居与纪念**：
- 文山市楚图南故居（省级文物保护单位）
- 文山州图书馆楚图南专柜
- 文山市楚图南文化广场

### 4.2 柯仲平（1902-1964）

柯仲平是中国现代文学史上"狂飙诗人"的代表人物。

**生平大事**：

| 年份 | 事件 |
|------|------|
| 1902年 | 生于广南县 |
| 1920年代 | 在北京求学期间开始诗歌创作，加入"狂飙社" |
| 1930年 | 加入中国共产党 |
| 1937年 | 赴延安，任陕甘宁边区文化协会主任 |
| 1940年代 | 创作长诗《边区自卫军》，毛泽东称赞其为"大众化诗歌的方向" |
| 1950年后 | 历任全国文联副主席、中国作家协会副主席 |
| 1964年 | 逝世 |

**代表作**：
- 《海夜歌声》（早期诗集）
- 《边区自卫军》（长篇叙事诗，延安时期最重要作品之一）
- 《平汉路工人破坏大队》（长篇叙事诗）
- 《风火山》（诗集）

**评价**：
> 毛泽东："他（柯仲平）把诗歌真的做到了大众化。"

## 五、文山人物精神特征

### 5.1 边地品格

| 精神特质 | 体现 |
|----------|------|
| 坚韧不屈 | 从侬智高抗交趾到老山守卫战，保家卫国的精神一脉相承 |
| 开拓进取 | 从科举进士到当代企业家，在边地开创事业 |

| 心系家国 | 楚图南投身民主革命、柯仲平奔赴延安，家国情怀深厚 |
