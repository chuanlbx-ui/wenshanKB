# 富宁港建设与平陆运河联动

> **来源**：中国新闻网 2026-06-04 | **记者**：罗婕
> **核心**：富宁港年内建成，与平陆运河同步投入使用，成为云南出海最近"黄金港口"。

## 一、富宁港概况

| 指标 | 数据 |
|------|------|
| 位置 | 文山州富宁县剥隘镇（云南"东大门"） |
| 定位 | 云南规划建设的**最大内陆港口**、"珠江第一港" |
| 投用时间 | **2026年内建成** |
| 联动 | 与广西平陆运河**同步投用** |

## 二、平陆运河联动

| 指标 | 数据 |
|------|------|
| 全长 | 134.2公里 |
| 地位 | 新中国成立以来第一条通江达海的大运河 |
| 最新进展 | **2026年6月4日全线通水** |
| 预计通航 | **2026年9月** |
| 联通 | 珠江水系 ↔ 北部湾 |
| 航线优势 | 比经由广州港出海缩短内河航程**560公里以上** |

## 三、战略意义

| 维度 | 内容 |
|------|------|
| 对外开放 | 文山主动融入**西部陆海新通道**的关键举措 |
| 出海水运 | 为云南新增一条最便捷水运通道，加快"通江达海" |
| 产业联动 | 大宗货物"公铁水"联运，服务绿色铝产业 |
| 区域发展 | "港产城"协同发展 |

## 四、配套交通

| 项目 | 时间节点 |
|------|----------|
| 文蒙铁路 | 2028年建成通车 |
| 文靖铁路 | 2030年建成通车 |
| 铝产业园区铁路专线 | 规划布局中 |
| 公铁水联运 | 推动与物流业深度融合 |

## 五、富宁港与绿色铝产业联动

富宁港建成后，将打通绿色铝产业大宗原料与产品的水运出口通道，与各港口通道联动潜力巨大，持续推动绿色铝产业高质量发展。
