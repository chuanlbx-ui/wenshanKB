# 丘砚高速公路 2026 年建设进展

丘北至砚山高速公路是文山州连接丘北县与砚山县的重要骨架交通干线。2026 年以来，各参建单位抢抓施工黄金期，隧道、桥梁、高边坡等重难点工程稳步推进。

## 项目概况

| 项目 | 数据 |
|------|------|
| 线路全长 | **63.78 km** |
| 技术标准 | 双向四车道 |
| 设计时速 | **100 km/h** |
| 概算总投资 | **102.77 亿元** |
| 起点 | 丘北县双龙营镇东侧坝子村（衔接师宗至丘北高速） |
| 终点 | 砚山县城北部（连通蒙自—文山—砚山高速） |

## 二标段进展（中铁二十四局）

| 指标 | 数据 |
|------|------|
| 合同额 | 17 亿元 |
| 建设里程 | 16 km |
| 涵盖隧道 | 9 座 |

## 重点控制性工程：花桂隧道

| 指标 | 数据 |
|------|------|
| 左洞长度 | 1.17 km |
| 右洞长度 | 1.2 km |
| 已进洞 | **300 米** |
| 日均掘进 | 2-3 米 |
| 预计贯通 | **2027 年 3 月底** |

## 高边坡工程

全线最高八级挖方边坡，总开挖高度 **64 米**。目前已完成六级边坡开挖，剩余两级有序施工，预计 **2026 年 6 月底**全面完成。

## 三、项目意义

丘砚高速是完善文山州路网布局、畅通内外交通的核心重点工程，衔接师宗至丘北高速与蒙自—文山—砚山高速，对推动滇东南区域经济协同高质量发展具有重要意义。
