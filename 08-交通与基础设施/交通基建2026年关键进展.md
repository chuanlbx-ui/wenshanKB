## 一、高速公路

### 师丘高速（文山段）— 冲刺通车

| 指标 | 数值 | 截至日期 |
|------|------|----------|
| 总体形象进度 | **79%** | 2026年5月 |
| 隧道工程 | 全线贯通 | 2026年5月 |
| 路基工程 | 完成率 90% | 2026年5月 |
| 桥梁工程 | 完成率 84% | 2026年5月 |
| 路面工程 | 完成率 23% | 2026年5月 |

- 投资建设方：云南交投集团
- 路线：省道 S11 师宗至丘北（文山段）
- 通车意义：彻底打通文山州北部山区交通瓶颈，串联沿线文旅与特色产业资源
- 来源：文山州政务网（2026-06-03）

### 天麻高速 — 2026年建成投运

- 与师丘高速同为 2026 年确保通车的重点项目
- 来源：2026年政府工作报告

## 三、水运

### 富宁港

- **2026 年内建成**，与广西平陆运河同步投用
- 成为云南出海最近的"黄金港口"
- 依托富宁港区位优势，构建"绿色铝—航运物流—内外市场"一体化发展格局
- 来源：政府工作报告 + 文旅头条

## 四、旅游交通

### 普者黑—老山圣地景区直通车

| 项目 | 详情 |
|------|------|
| 开通日期 | **2026年3月22日** |
| 运营方 | 云南普者黑文化旅游开发有限公司 |
| 战略定位 | 丘北—广南—麻栗坡文旅黄金三角 |
| 串联景区 | 5A 普者黑 + 英雄老山圣地 |
| 运营模式 | 定时发车、一站直达 |

- 打通"绿水青山"自然生态与"红色基因"精神文脉的无缝接驳
- 实现"观光+研学+教育"复合型旅游闭环
- 来源：文旅头条（2026-03-22）

### 麻栗坡国门水境度假区

| 指标 | 数值 |
|------|------|
| 总体面积 | 约 6,304.6 亩（4,203,087 ㎡） |
| 水上乐园面积 | 约 4,225 亩（2,816,683 ㎡） |
| 预计全面运营 | **2026年6月底** |
| 主题 | "红、绿、边" |

- 重点项目：花涧泉谷度假片区、大王岩水上运动片区、峡谷森林休闲片区
- 体验项目：水上乐园、峡谷花海、沙滩运动、水上夜游、玻璃栈道、悬索吊桥
- 来源：携程攻略 + 麻栗坡县政府办
