# 文蒙铁路专题

> 文山至蒙自铁路 · 国铁Ⅰ级双线电气化 · 滇东南出海大通道

---

## 一、项目概览

| 参数 | 数值 |
|------|------|
| 线路全长 | 约 115.3 km（新建正线约 101 km） |
| 设计时速 | **160 km/h** |
| 铁路等级 | 国铁Ⅰ级 |
| 轨道类型 | 双线，电力牵引 |
| 桥隧比 | 约 **74.45%** |
| 总投资 | 文山段约 **64 亿元** |
| 建设工期 | 4.5 年 |
| 预计通车 | **2028 年 4-12 月** |

---

## 二、线路走向

自东向西横贯文山州与红河州，途经文山市、蒙自市、屏边县：

```
文山站（起点）
  → 马塘镇 → 薄竹镇 → 老寨乡
  → 查尼皮 → 蒙自站（接既有昆玉河铁路）
  → 红河站（终点，接入弥蒙高铁）
```

---

## 三、车站设置（共 7 座）

| 车站 | 性质 | 位置与功能 |
|------|------|------------|
| **文山站** | 新建中间站（客运） | 文山市，线路起点，配套综合交通枢纽 |
| **马塘站** | 新建中间站（货运） | 文山市马塘镇，定位为区域**货运物流中心** |
| **薄竹站** | 新建中间站（客货） | 文山市薄竹镇北侧，配套综合交通枢纽 |
| **老寨站** | 新建中间站（客货） | 蒙自市老寨乡 |
| **查尼皮站** | 新建越行站 | 蒙自市，供列车越行/会让 |
| **蒙自站** | 既有改建 | 蒙自市东郊，滇南铁路枢纽 |
| **红河站** | 既有改建 | 弥蒙高铁终点，增建二线 |

另设 **永宁线路所**（线路衔接与调度）。

---

## 四、建设进展（截至 2026 年初）

### 4.1 前期工作（2016 — 2023）

完成可研、环评、初步设计及施工图审核等全部前期工作。

### 4.2 全线开工

- **2024 年 1 月 30 日**：文蒙铁路正式开工建设

### 4.3 2025 年关键节点

| 节点 | 进度 |
|------|------|
| 文山段产值 | 完成约 **4.3 亿元**（占年度计划 48.22%） |
| 中岭山隧道 | 首模二衬浇筑完成 |
| 对门山隧道 | **2025 年 10 月 24 日安全贯通**（首座贯通隧道） |
| 新回龙特大桥 | 下部结构接近尾声，计划 2026 年 6 月合龙 |
| 大东山特大桥 | 应用云南省首座铁路**装配式桥墩**技术 |
| 州界隧道（Ⅰ级高风险） | 由斜井转入正洞施工 |

### 4.4 2026 年初动态

- **州界隧道**：掘进突破 **1000 米**（截至 1 月 27 日）
- **大东山特大桥**：首个装配式桥墩完成吊装
- **老尖坡特大桥**：进入墩身主体施工阶段

> 总体判断：项目按 2028 年前后通车目标稳步推进，全线控制性工程均已进入实质性施工阶段。

## 五、战略意义

### 5.1 打通滇东南出海通道

文蒙铁路是**泛亚铁路东线**的重要组成部分，建成后将：
- 连接云桂铁路（文山方向）与昆玉河铁路（蒙自方向）
- 与**蒙自—河口**铁路贯通，直达中越边境河口口岸
- 形成文山 → 蒙自 → 河口 → 越南海防港的**出海大通道**

### 5.2 结束文山不通铁路的历史

文山州是云南省唯一尚未通铁路的州市。文蒙铁路通车后，文山将接入全国铁路网，实现：
- 文山至昆明铁路旅行时间大幅缩短
- 与滇中城市群（昆明、曲靖、玉溪）直连
- 文山州内客货运输成本显著降低

### 5.3 服务绿色铝产业

文山州绿色铝产能 343 万吨，是全国最大的绿色铝产业基地之一。铁路货运能力将：
- 大幅降低铝土矿、氧化铝、电解铝等大宗物资运输成本
- 保障绿色铝产业园（砚山、富宁）供应链稳定
- 提升"绿色铝谷"产品外运效率

### 5.4 促进沿边开放

文蒙铁路与河口口岸铁路联通后，将形成"**内联滇中、外接东盟**"的铁路通道，助力文山从"交通末梢"转变为"开放前沿"。
