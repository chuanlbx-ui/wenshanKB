# 富宁港建设

## 一、战略定位

富宁港是云南省**对接珠江西江黄金水道**的关键节点，是文山州实现"通江达海"战略的核心工程。建成后，云南将打通经右江→西江→珠江直达粤港澳大湾区和北部湾的**千吨级水运大通道**，使文山成为云南面向东盟的水路门户。

## 二、工程概况

| 项目 | 详情 |
|------|------|
| 位置 | 富宁县剥隘镇，右江上游 |
| 设计通航等级 | **1000 吨级**（远期 2000 吨级） |
| 设计年吞吐能力 | **500 万吨**（一期） |
| 泊位数量 | **8 个**（1000 吨级泊位） |
| 总占地 | 约 2,500 亩 |
| 项目总投资 | 约 **58 亿元**（含航道整治） |

## 三、建设进展

### 3.1 关键节点

| 时间 | 里程碑 |
|------|--------|
| 2019年 | 富宁港项目纳入《西部陆海新通道总体规划》 |
| 2021年 | 百色水利枢纽通航设施工程正式开工 |
| 2022年 | 富宁港一期工程项目可研获批 |
| 2023年 | 富宁港一期主体工程全面开工 |
| 2024年 | 码头水工结构基本完工 |
| 2025年 | 陆域仓储堆场、进港公路等全面施工 |
| **2026年（预期）** | **富宁港建成，与平陆运河同步投用** |
| **2027年（预期）** | **百色水利枢纽通航设施建成，全线贯通** |

### 3.2 百色水利枢纽通航设施（关键配套）

| 项目 | 详情 |
|------|------|
| 位置 | 广西百色市，百色水利枢纽 |
| 功能 | 克服富宁—百色间 30 m 水位落差 |
| 建设内容 | 船闸+升船机组合式通航设施 |
| 设计通过能力 | 年双向 600 万吨 |
| 投资 | 约 30 亿元 |
| 状态 | 主体工程 2025年完成 **70%+**，预计 2027 年建成 |

> 百色通航设施是富宁港的"咽喉"，建成前港口无法联通下游。

## 五、货源腹地与产业联动

### 5.1 直接腹地

| 货类 | 年运量预测（初期） | 来源 |
|------|-------------------|------|
| 氧化铝/铝产品 | 200 万吨 | 砚山绿色铝产业园 |
| 矿产（锰、锑、锌） | 50 万吨 | 砚山斗南、广南 |
| 建材（水泥、石材） | 80 万吨 | 文山、广南 |
| 农副产品（八宝米、辣椒） | 30 万吨 | 广南、丘北 |
| 集装箱货物 | 40 万吨 | 综合 |
| **合计** | **约 400 万吨** | |

### 5.2 产业联动链

| 环节 | 说明 |
|------|------|
| 铝产业 | 氧化铝（文山→大湾区）+进口铝土矿（越南经田蓬口岸→富宁港转运） |
| 矿产出口 | 锰矿、锑矿经珠江运往广东佛山、深圳 |
| 农产品 | 文山特色农产品水运直达粤港澳大湾区 |
| 跨境水陆联运 | 田蓬口岸（公路）+富宁港（水路）复合通道 |

## 六、经济效益预测

| 指标 | 预测数据 |
|------|----------|
| 开港首年（2028年）吞吐量 | 约 100 万吨 |
| 达产年（2033年）吞吐量 | 500 万吨 |
| 开港首年综合收入 | 约 2 亿元 |
| 物流成本降低（对比陆路） | **40%—60%** |
| 直接就业 | 3,000+ 人 |
| 间接带动就业 | 10,000+ 人 |

## 七、挑战与风险

| 挑战 | 说明 |
|------|------|
| 通航设施进度 | 百色水利枢纽通航设施工程复杂，存在工期延迟风险 |
| 货源培育 | 初期货源不足可能导致运营亏损 |
| 季节性水位 | 枯水期（11月—4月）右江水位偏低，影响大吨位通航 |
| 铁路竞争 | 文靖铁路建成后可能分流部分货源 |
| 生态约束 | 右江流域生态保护对航道等级的制约 |

## 八、"十五五"展望

| 方向 | 目标 |
|------|------|
| 开港运营 | 2026年建成并与平陆运河同步投用 |
| 富宁港二期 | 规划新增2000吨级泊位，年吞吐能力达 **1,000 万吨** |
| 富宁—田蓬口岸联络线 | 30 km 高等级公路，口岸—港口 30 分钟通勤 |
| 物流园区 | 富宁港临港物流产业园，仓储面积 10 万 m² |
| 智慧港口 | 无人化装卸、智能调度系统 |

> 更新说明：根据2026年6月最新信息，富宁港将于2026年内建成，与平陆运河同步投用，开港时间较原预期（2028年）大幅提前。
