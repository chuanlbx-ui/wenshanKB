## 一、政务宣传

### 1.1 政策解读通俗版
```
你是一位政务新媒体编辑，请将以下政策文件用文山普通老百姓能听懂的话解读一遍，
控制在 500 字以内，语气亲切但不失权威，适当用文山本地的生活例子说明。
政策要点：[粘贴政策原文]
```

### 1.2 基层工作亮点报道
```
请你以文山州某县融媒体中心记者的身份，根据以下素材撰写一篇基层工作亮点报道。
标题抓人眼球，正文采用"总-分-总"结构，穿插一位基层干部的引语（可虚构但合理），
结尾要有展望。字数 800-1200 字。
素材：[粘贴工作素材]
```

### 1.3 民族团结主题宣传文案
```
你是文山州民族团结宣传员。文山州有壮族、苗族、彝族、瑶族等 11 个世居民族，
少数民族人口占比约 58%。请撰写一段 300 字左右的民族团结主题宣传文案，
体现"中华民族一家亲"理念，融入坡芽歌书、铜鼓文化等文山非遗元素，
适合用于政府公众号推文或社区宣传栏。
```

### 1.4 正能量短视频脚本
```
你是一位短视频策划，请为文山州创作一个 60 秒正能量短视频脚本。
主题（三选一）：①西畴精神——"搬家不如搬石头"；②边境守边人日常；
③乡村振兴中的致富带头人。请给出：分镜表（含画面描述、旁白/字幕、时长）、
BGM 建议、封面文案。
```

### 1.5 政策一张图（信息图文案）
```
请将以下政策要点整理为一张信息图文案。每条内容控制在 15 字以内，
用"数字+效果"的格式呈现，适合文山群众在微信朋友圈转发。
政策要点：[粘贴]
```

## 二、文旅推广

### 2.1 景区种草文案（普者黑）
```
你是文山文旅推荐官。普者黑是国家 5A 级景区，拥有 312 座孤峰、83 个溶洞、
54 个湖泊，是《三生三世十里桃花》取景地。2025 年文山全州接待游客 5004 万人次，
旅游收入 541 亿元。请创作一篇小红书风格的普者黑种草文案，
包含：吸睛标题、3 个必打卡点、最佳季节（6-8 月荷花季）、
一个本地人才知道的小众机位、实用 tips。语气活泼有网感，加 emoji 和 tag。
```

### 2.2 文旅短视频解说词
```
你是一位旅行博主，请为文山"喀斯特山水+民族风情"三日游撰写一条 2 分钟短视频解说词。
行程：Day1 普者黑（柳叶小舟+仙人洞村）→ Day2 坝美世外桃源（乘船穿洞）→ Day3 老山圣地+天保口岸。
要求：解说词有画面感，每段标注对应画面建议，总字数 400 字左右，
结尾引导点赞关注。
```

### 2.3 文旅数据海报文案
```
根据以下文山文旅数据，创作 5 条适合做成海报的短文案（每条 ≤20 字）：
- 2025 年接待游客 5004 万人次
- 旅游综合收入 541 亿元
- 1 个 5A 级景区（普者黑）、多个 4A 级景区
- 10 个世居少数民族的节庆文化
- 中越边境风情线路
要求：每条文案有冲击力，适合配景区大片做海报。
```

### 2.4 民族节庆邀约文案
```
文山州每年有苗族花山节（正月）、壮族三月三、彝族火把节（六月）、
瑶族盘王节（十月）等重大民族节庆。请撰写一篇邀请省内外游客来文山过节的公众号推文，
突出"沉浸式体验"和"美食+民俗"两大卖点，附一份全年节庆日历表。
字数 800 字左右。
```

### 2.5 负面提示词（避免文旅宣传踩坑）
```
【文旅宣传负面提示词】
⚠️ 避免使用以下表述及场景：
- "原始落后"、"未开发"（易引发负面联想）
- 对少数民族服饰/习俗的猎奇化描述
- 军事禁区附近拍照引导（老山等边境地区）
- 夸大景区接待能力（如"可容纳十万人"）
- 忽视季节性风险的宣传（如雨季塌方、涨水）
- 未经核实的传说或民间故事（优先使用有文献记载的内容）
请确保所有宣传内容准确反映文山真实风貌。
```

## 三、农产品带货

### 3.1 三七带货脚本
```
你是文山三七带货主播。文山三七产量占全国 90% 以上，是国家地理标志产品。
核心卖点：①道地药材——"北有人参，南有三七"；②活血化瘀、"金不换"；
③文山独特气候（北回归线黄金种植带）。
请撰写一段 90 秒直播带货口播脚本，包含：
开场 hook、"三七 vs 普通田七"区分技巧、冲泡/煲汤方法、今日福利价引导。
语气真诚接地气，不要过度夸张功效。
```

### 3.2 农产品产地故事
```
请为以下文山农产品创作一个 300 字的"产地故事"，突出"一方水土养一方风物"：
- 丘北辣椒：中国辣椒之乡，色泽鲜艳、香辣可口，出口 20 多个国家
- 广南八宝米：明清贡米，因八宝河水质独特，米饭香软油润
- 富宁八角：中国八角之乡，产量占全国 60% 以上
格式：标题 + 产地故事 + 一句话购买理由。
```

### 3.3 短视频助农脚本
```
你是乡村振兴助农主播，为文山某村农产品（自选：三七/辣椒/八宝米/八角）策划一条 60 秒短视频。
要求：
- 前 3 秒必须有视觉冲击（如航拍梯田、农民采摘特写）
- 中间展示产品优势和真实使用场景
- 结尾加"助农链接在评论区"引导
- 文案适合方言口播（亲切感）
```

### 3.4 农产品电商详情页
```
请为文山三七粉撰写一份电商详情页文案框架：
① 主标题（10 字以内）
② 副标题（一句话打动用户）
③ 核心卖点 3 条（每条配 icon 建议）
④ 产地实拍文案（200 字）
⑤ 用户痛点+解决方案
⑥ 5 条买家好评模板
⑦ 食用建议 + 禁忌提示
要求：真实不夸大，符合《广告法》，不涉及疗效宣称。
```

### 3.5 负面提示词（避免农产品带货踩坑）
```
【农产品带货负面提示词】
⚠️ 严禁：
- 宣称三七/其他农产品"治疗"疾病（违反《广告法》，应使用"辅助""调理"等词）
- 使用"最""第一""唯一"等绝对化用语（除非有官方认证）
- 虚假原产地（非文山产三七不得称"文山三七"）
- 价格误导（如"原价 999 现价 99"，需有真实交易记录支撑）
- 伪造农民身份或贫困故事骗取同情消费
- 图片过度美化导致实物与展示严重不符
⚠️ 谨慎：
- 引用未经核实的中医古方
- 暗示产品有美容、减肥等特殊功效
```

```

① 坡芽歌书（国家级非遗，81 个图画符号记录壮族情歌）

③ 苗族蜡染技艺
④ 彝族刺绣
要求：15-30 秒口播文本，用一个悬念开头（如"世界上唯一用图画写的情书，就在文山"），
中间展示技艺过程，结尾呼吁关注非遗传承。
```

### 4.2 民族节日科普图文

```
请为文山"壮族三月三"节日创作一篇微信公众号科普图文大纲：
```

### 4.4 负面提示词（避免民族文化内容踩坑）

```
【民族文化内容负面提示词】
```

## 五、新闻通讯

### 5.1 政务新闻通稿模板
```
你是文山州某政府部门新闻通讯员。请根据以下要素撰写一篇标准新闻通稿：
- 事件：[填写]
- 时间：[填写]
- 地点：[填写]
- 参与领导：[填写]
- 核心成果/数据：[填写]
- 下一步工作：[填写]

要求：倒金字塔结构，导语 60 字以内，全文 400-600 字，引用一位领导的表态（可虚构但符合身份），
适合向文山日报、文山新闻网投稿。
```

### 5.2 典型人物通讯
```
请根据以下线索，撰写一篇文山基层典型人物通讯（1500 字左右）：
人物背景：[填写，如"麻栗坡县某边境村守边人，坚守 30 年"]
要求：用 3 个小标题组织全文，每个小标题下至少有一个具体场景描写，
穿插直接引语（人物原话），结尾要有升华但不拔高过度。
风格参考：《人民日报》"走基层"栏目。
```

### 5.3 突发事件通报
```
你是文山州某单位的舆情应对人员。请根据以下突发事件信息，撰写一份面向公众的官方通报：
事件概述：[填写]
已采取措施：[填写]
下一步安排：[填写]

要求：措辞严谨，不回避问题，不推卸责任，体现政府担当。
200-300 字，适合官方公众号/微博发布。
```

### 5.4 数据新闻转化
```
请将以下文山经济数据转化为一段通俗易懂的新闻导语，让普通读者也能理解数据含义：
数据：[粘贴数据]

要求：使用类比（如"相当于 XX 个足球场""可以绕地球 XX 圈"）、
对比（如"同比增长 XX，是 2019 年的 XX 倍"）、
可视化建议（如"建议做成柱状图对比"）。
```

## 六、乡土故事

### 6.1 "我的家乡文山"叙事模板
```
你是一位文山籍大学生（或在外务工者）。请以"我的家乡文山"为题，
撰写一篇 800-1000 字的乡土散文。
框架：①一个具体的童年记忆场景（如赶集、采三七花、端午药市）
②家乡近年的变化（路通了、旅游火了、三七价格涨了）
③一个让你骄傲的家乡符号（普者黑/坡芽歌书/西畴精神）
④结尾：无论走多远，家乡的味道/声音/风物让我知道根在哪里。
语气真诚自然，避免空洞的赞美，多用具体的感官描写（气味、声音、画面）。
```

### 6.2 乡村匠人故事
```
请根据以下线索，为文山一位乡村匠人撰写一篇"凡人故事"：
匠人姓名/绰号：[填写]
手艺：[填写，如"打了 50 年铜鼓的老匠人""三代传承的壮族刺绣"]
所在村寨：[填写]

要求：平实的叙事，至少引用一段匠人原话，
描述手艺过程（至少 3 个具体步骤），体现"慢工出细活"的价值观。
适合发布在"文山文旅"微信公众号或《文山日报》副刊。
500-800 字。
```

### 6.3 赶集故事脚本
```
文山州每个县城都有赶集日（街天），热闹非凡。请以"文山街天"为主题，
创作一个 60 秒 vlog 脚本：
要求：镜头从全景（熙熙攘攘的集市）→ 中景（某个摊位，如三七、野生菌、手工银饰）→
特写（老人数钱、孩童吃小吃、热气腾腾的米线锅）→ 结尾全景。
旁白要有生活气息，可夹一两句方言词汇（如"赶该"=赶集），
BGM 建议本地民乐或轻快吉他。
```

### 6.4 边境村庄日常
```
请以文山边境某村庄（如麻栗坡天保镇）为背景，创作一篇 500 字的"边境生活日常"短文。
内容：平静的小村庄、界碑旁升国旗的日常、边民互市的烟火气、
"一脚跨两国"的独特体验。
语气：宁静而有力量，不煽情。
适合用于短视频旁白或图集配文。
```

3. **调优生成**：将组装好的 Prompt 输入 AI 工具，根据首次结果微调
4. **合规检查**：生成后对照 [[合规指南]] 检查，尤其边境/民族/军事话题
5. **风格润色**：如需统一文风，参考 [[风格指南]] 二次润色
