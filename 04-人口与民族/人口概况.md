# 人口概况

## 一、人口总量与趋势

| 年份 | 常住人口（万） | 城镇化率 |
|------|:-----------:|:------:|
| 2020（七普） | 350.32 | 37.20% |
| 2023 | 339.7 | 39.67% |
| 2024 | 337.2 | 40.57% |
| 2025 | 335.4 | 41.19% |
| 2025（十五五规划预计） | — | **41%** |
| 2030（十五五目标） | — | **45%左右** |

人口呈逐年小幅下降趋势，2025 年末常住人口 335.4 万人，较 2020 年减少约 15 万人，年均减少约 3 万人。2025 年人口自然增长率 **-1.49‰**（出生率 8.47‰，死亡率 9.96‰），已进入负增长区间。

## 二、人口结构（2025 年）

| 指标 | 数值 | 占比 |
|------|------|:---:|
| 城镇人口 | 138.1 万 | 41.19% |
| 乡村人口 | 197.3 万 | 58.81% |

- **性别比**：男性 175.0 万（52.2%），女性 160.4 万（47.8%），性别比约 109.1（高于全国平均水平）
- **年龄结构**：0-14 岁约 22%，15-59 岁约 62%，60 岁及以上约 16%（老龄化程度低于全省平均）
- **人口密度**：约 107 人/km²（全州面积 31,456 km²）

## 三、民族构成

文山州有汉、壮、苗、彝、瑶、回、傣、布依、蒙古、白、仡佬共 **11 个世居民族**。少数民族人口占总人口的 57.53%（2020 七普）。

| 民族 | 人口（2020 七普） | 占比 |
|------|:-------------:|:---:|
| 汉族 | 1,487,947 | 42.47% |
| 壮族 | 987,585 | 28.19% |
| 苗族 | 506,550 | 14.46% |
| 彝族 | 362,218 | 10.34% |
| 瑶族 | 82,617 | 2.36% |
| 回族 | 26,900 | 0.77% |
| 傣族 | 16,107 | 0.46% |
| 白族 | 10,845 | 0.31% |
| 布依族 | 7,228 | 0.21% |
| 蒙古族 | 5,463 | 0.16% |
| 其他民族 | 9,758 | 0.28% |

壮族和苗族合计占少数民族人口的 74.15%，是文山州最主要的两个少数民族。详细民族分布见 [[民族构成]]。

## 四、劳动与教育

- **劳动年龄人口**：约 208 万人（15-59 岁），占 62%
- **就业结构**：第一产业约 55%，第二产业约 15%，第三产业约 30%
- **受教育程度**：15 岁及以上人口平均受教育年限约 8.5 年，低于全省平均水平
- **高中阶段毛入学率**：90.42%（2024 年）

## 五、人口流动

- 外出务工人口约 60-80 万人，以广东、浙江为主要流向
- 近年受粤港澳大湾区产业协作带动，返乡创业逐渐增加
- 城镇化率年均提升约 1 个百分点，预计 2030 年突破 41.19%

> 人口数据以 2020 年第七次全国人口普查为基准，更新至 2025 年末统计公报。2020 年后的分民族数据待第六次人口普查完整公布后更新。

## 六、2025年人口动态更新

> 来源：文山州2025年统计公报（2026年5月发布）

| 指标 | 2025年数据 | 同比/备注 |
|------|-----------|-----------|
| 年末常住人口 | 335.4 万人 | 较2024年337.2万减少1.8万 |
| 城镇人口 | 138.1 万人 | 占比41.19% |
| 乡村人口 | 197.3 万人 | 占比58.81% |
| 出生人口 | 2.85 万人 | 出生率 8.47‰ |
| 死亡人口 | 3.35 万人 | 死亡率 9.96‰ |
| 自然增长率 | **-1.49‰** | 首次确认进入负增长 |
| 城镇化率 | 41.19% | 比上年提高0.62个百分点 |

**关键信号**：2025年文山州人口自然增长率首次转负，但城镇化率提升0.62个百分点至41.19%，城镇人口较上年增加约1.2万人，城镇化进程仍在加速。
