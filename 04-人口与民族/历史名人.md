# 文山州历史名人

## 一、楚图南（1899–1994）

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 籍贯 | 云南文山 |
| 生卒 | 1899年 – 1994年 |
| 身份 | 教育家、翻译家、社会活动家、国家领导人 |

### 1.2 主要贡献

#### 民主革命时期
- 早年投身民主革命，是**云南早期中共党员**之一
- 长期从事革命宣传与组织工作

#### 教育事业
- 曾任教于多所高校，培养了大批人才
- 推动中国现代教育事业的发展

#### 国家领导职务
- 新中国成立后，历任：
  - **中国人民对外友好协会会长**
  - **全国人大常委会副委员长**
  - 民盟中央主席等职

#### 文化贡献
- 翻译惠特曼《**草叶集**》等世界文学名著
- 推动中外文化交流，被誉为"民间外交家"
- 书法造诣深厚，题字遍布全国多地

## 二、柯仲平（1902–1964）

### 2.1 基本信息

| 项目 | 内容 |
|------|------|
| 籍贯 | 云南广南 |
| 生卒 | 1902年 – 1964年 |
| 身份 | 著名诗人、剧作家 |

### 2.2 主要贡献

#### 左翼文化运动
- 20世纪30年代"**左联**"（中国左翼作家联盟）重要诗人
- 作品充满革命激情与战斗精神

#### 延安文艺
- 创作秧歌剧《**边区自卫军**》等作品，影响广泛
- 是延安文艺座谈会后践行"文艺为工农兵服务"的代表人物

#### 文化领导工作
- 新中国成立后，长期担任文艺界领导职务
- 曾任中国作家协会副主席等职
