## 使用方式

### 在 Obsidian 中打开

1. 打开 Obsidian，选择「打开本地仓库」
2. 选择 `D:\WenShanKB` 文件夹
3. 从 `00-总览/文山KB总索引.md` 开始浏览，支持 wikilink 双向链接跳转

### 查看导出文件

| 导出物 | 路径 |
|--------|------|
| 静态 HTML 站点 | `output/publish/index.html`（浏览器打开） |
| 核心笔记 PDF 合集 | `output/publish/文山KB核心笔记合集.pdf` |
| 总索引精美 HTML | `output/publish/文山KB总索引.html` |

### 阅读建议

- **快速了解**：读 `00-总览/文山KB总索引`，一览全库结构
- **经济研究**：读 `synthesis/文山经济全景`，掌握"资源→产业→通道"核心框架
- **旅游规划**：读 `synthesis/文山文旅地图`，含景区、节日、线路、美食
- **历史脉络**：读 `synthesis/文山发展时间线`，5 万年关键节点一目了然
- **数据查询**：读 `wiki/synthesis/十年趋势数据2015-2025`，十年核心指标趋势
- **内容创作**：读 `ai-hub/提示词库` + `ai-hub/风格指南`，快速上手 AI 辅助创作

## 技术说明

- 格式：Markdown + YAML frontmatter
- 链接：Obsidian 风格 wikilink（`[[路径/笔记名]]`）
- 图表：matplotlib 生成的 180dpi PNG，存放于对应 `assets/` 目录
- 知识库通过脚本构建为静态 HTML 站点和 PDF 合集
- 总索引 `index.md` 和总导航 `00-MOC.md` 持续同步更新
