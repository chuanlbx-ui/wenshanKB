# 广南八宝米产业深度

## 一、产业概况

广南八宝米因原产于文山州广南县**八宝镇**而得名，素有"**云南六大名米**"之美誉。八宝米以"松软、清香、润泽、回味甘甜"的独特口感，自明清时期即作为朝廷贡米进献，至今已有 300 余年栽培历史。

| 指标 | 数据 | 来源 |
|------|------|------|
| 种植面积 | 约 15 万亩 | 广南县农业农村局 |
| 年产量 | 约 5 万吨（稻谷） | 广南县农业农村局 |
| 综合产值 | 约 8 亿元 | 广南县政府工作报告 |
| 从业人员 | 约 8 万人 | 广南县人社局 |
| 地理标志 | 2009 年获农产品地理标志登记保护 | 农业农村部 |

## 二、品质特征

### 2.1 独特成因

八宝米的优良品质源于独特的**环境禀赋**：

| 要素 | 特征 |
|------|------|
| 海拔 | 1100-1300 米（山区河谷地带） |
| 气候 | 亚热带低纬高原气候，年均温 16-17°C |
| 水源 | 八宝河天然自流灌溉，水质纯净 |
| 土壤 | 河谷冲积土，有机质含量高 |
| 昼夜温差 | 约 10-12°C（利于养分积累） |
| 日照 | 年日照约 1800-2000 小时 |

### 2.2 核心品种

| 品种 | 特点 | 米粒长宽比 |
|------|------|------------|
| 八宝香 | 经典贡品品种，米香浓郁，饭粒油润 | 3.2:1 |
| 广籼 2 号 | 优质常规稻，口感软糯，冷饭不回生 | 3.0:1 |
| 红八宝 | 特色有色米，富含花青素 | 2.8:1 |
| 紫八宝 | 紫黑色米，铁锌含量高 | 2.7:1 |

### 2.3 营养指标

| 成分 | 八宝香米含量 | 普通籼米对比 |
|------|-------------|-------------|
| 直链淀粉 | 15-17% | 18-22%（更软糯） |
| 蛋白质 | 7.5-8.0% | 6.5-7.0%（高于普通米） |
| 胶稠度 | ≥ 70 mm | 40-60 mm（更适口） |
| 碱消值 | ≥ 6.0 级 | 4.0-5.5 级 |

## 三、地理标志保护范围

根据农业农村部 2009 年公告，广南八宝米地理标志保护范围为：

> 广南县八宝镇、莲城镇、那洒镇、珠琳镇、者兔乡、者太乡、底圩乡等 7 个乡镇现辖行政区域

其中以**八宝镇**为核心产区，面积约 8 万亩，产量和质量最为突出。

## 四、历史文化价值

### 4.1 贡米历史

| 时期 | 事件 |
|------|------|
| 明末清初 | 八宝米被列为**朝廷贡米**，由土司组织上贡 |
| 清代 | 通过茶马古道运销昆明、四川、贵州等地 |
| 民国 | 在昆明市场被评为"滇南第一米" |
| 1959年 | 广南八宝米被选送北京，开始全国范围推介 |

### 4.2 文化载体

- **八宝河畔稻作文化**：壮族"那"文化（稻作文化）核心区
- **传统稻田节**：每年农历四月举行的壮族农耕祭祀活动
- **贡米文化节**：近年由广南县政府主导打造的文化旅游节事

## 五、产业链分析

### 5.1 种植模式

| 模式 | 特点 | 占比 |
|------|------|------|
| 传统散户 | 农户自有田地，传统种植 | 约 65% |
| 合作社基地 | "合作社+农户"标准化种植 | 约 25% |
| 企业订单 | 龙头企业订单种植，统一收购 | 约 10% |

### 5.2 加工体系

```
品种选育（广南八宝米研究所） → 绿色种植 → 稻谷收购 → 精米加工 → 品牌包装 → 销售
```

**核心加工企业**：
- 广南县八宝米业有限公司（龙头企业）
- 广南凯鑫生态农牧有限公司
- 多家专业合作社精米加工点

### 5.3 产品体系

| 产品类型 | 代表产品 | 定位 |
|----------|----------|------|
| 高端礼盒 | 贡米礼盒、八宝香精品装 | 礼品市场、电商 |
| 中端精品 | 八宝香 5kg/10kg 装 | 商超、专卖店 |
| 功能米 | 红八宝、紫八宝营养米 | 健康食品市场 |
| 深加工 | 八宝米线、八宝米粉 | 餐饮供应链 |

## 八、产业挑战与发展方向

### 8.1 主要挑战

| 挑战 | 具体表现 |
|------|----------|
| 核心产区保护 | 城镇化扩展挤压八宝镇核心稻田面积 |
| 灌溉水源保障 | 八宝河上游开发对水源的潜在影响 |
| 品牌外溢风险 | 非核心产区大米冒充"八宝米"销售 |
| 价格竞争力 | 中低端市场被东北米、湖南米挤压 |

### 8.2 发展方向

1. **核心产区保护**：划定八宝镇核心稻田保护区，纳入县域国土空间规划
2. **原产地认证体系**：建立"广南八宝米"可追溯防伪体系，一袋一码
3. **品质分级定价**：建立贡品级/精品级/普品级三级分级标准
4. **品种创新**：开发香型更强、口感更优的新品种
5. **农文旅融合**：结合坝美景区、句町文化，打造"稻田观光+米食体验"旅游线路

## 九、与坝美景区联动

广南八宝米与**坝美世外桃源**景区形成文化深度联动：
- 坝美景区周边即八宝米核心产区，可设计"坝美游览 + 八宝稻田观光"一日游线路
- 开发"八宝米 + 壮族稻作文化"体验项目
- 将八宝米精品礼盒作为坝美景区官方伴手礼
