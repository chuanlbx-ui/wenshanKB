# 文山三七

## 一、道地属性

- **学名**：*Panax notoginseng* (Burk.) F.H. Chen
- **类别**：五加科人参属
- **核心成分**：三七总皂苷（PNS），含量为人参的 2-3 倍
- **道地性依据**：文山是三七的**原产地和原生地**，地质形成于约 6500 万年前，海拔高、纬度低、降雨多，土壤钾钙含量高，是全球最适宜三七生长的区域
- **栽培历史**：400 年以上，中国最早实现三七人工栽培的地区
- **保护认证**：国家地理标志保护产品；中国重要农业文化遗产；入选 2025 年首批全国名特优新农产品目录

## 二、产业规模（2025 年）

| 指标 | 数值 |
|------|------|
| 以三七为主的中药材种植面积 | 254 万亩 |
| 三七综合产值 | **220 亿元**（+7%，仅三七品种口径；中药材全口径 456 亿元） |
| 三七自身贡献产值 | 约 206 亿元 |
| GAP 标准化种植基地 | 1.5 万亩（全省前列） |
| 林下三七生态种植 | 8,000 余亩 |
| 老地复种推广面积 | 超 5 万亩（"十四五"期间） |
| 品牌价值 | 90.8 亿元（2024 年） |
| 数智云药交易中心交易额 | 突破 31 亿元 |
| 2030 年目标综合产值 | 1,000 亿元 |

产业地位：全国产量第一、质量第一、产值第一，荣获全国优势特色产业集群等 **9 个国家级荣誉**。

## 三、产业链条

```
种子种苗 → 订单种植 → 规范种植 → 公开购销 → 产地初加工 → 产地仓储 → 地标赋能 → 精深加工 → 市场流通
```

### 加工与产品开发

- 17 家以上以三七为原料的生产企业
- 开发三七系列产品 55 种
- 28 种在国际和全国博览会上获奖
- 高新区三七产业园入驻 50 余家加工企业
- 引进华润三九、漳州片仔癀、云南白药集团等龙头企业
- 华润集团为"链主"整合三七生产企业

## 四、科研与保障体系

- **立法保障**：全国首个单一中药材品种地方条例——《文山三七发展条例》
- **科研平台**：文山三七研究院、文山学院三七医药学院、全国首个三七资源保护与利用技术国家地方联合工程研究中心
- **质量标准**：覆盖全产业链的国际/国家/行业/地方质量标准体系
- **种源保护**：云药种谷·文山三七种源基地（云南白药），到 2030 年良种覆盖率提升至 50% 以上
- **产业孵化**：文山三七新质生产力孵化园，总投资超 1 亿元，年产优质种苗超 1 亿株，年产值约 2.5 亿元，带动周边 50 万亩种植

## 五、生态挑战与绿色转型

### 连作障碍

三七根系向土壤分泌自毒物质，导致传统模式下种过三七的土地需间隔 **20 年**才能再次种植。文山州构建了「病原阻断 → 土壤调理 → 微生物重构 → 综合防控」生态种植体系，攻克连作障碍。

### 绿色种植

- GAP 标准化基地可获每亩 300-400 元财政奖励
- 2025 年下达 5 万亩种植用地指标
- 全面推广"产地仓 + 数智云药交易中心"全链条数字化模式
