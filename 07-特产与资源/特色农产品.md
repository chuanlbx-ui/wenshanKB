# 特色农产品

文山州地处低纬高原，立体气候显著，生物资源丰富，特色农产品种类多、品质优。以下按产业规模与影响力分列八大核心产品。

## 一、丘北辣椒（地理标志产品）

| 指标 | 数据 |
|------|------|
| **种植面积** | 全州约 50 万亩，丘北核心区约 35 万亩 |
| **年产量** | 干椒约 6—7 万吨 |
| **年产值** | 超 20 亿元 |
| **出口** | 欧美、东南亚等 20 多个国家和地区 |

**品质特色**：色艳、肉厚、籽多、油丰、味香、辣红素含量高，香辣味浓。丘北县是全国重要的辣椒出口基地之一，产品以干椒出口为主，兼顾国内调味品市场。

## 二、富宁八角（地理标志产品）

| 指标 | 数据 |
|------|------|
| **种植面积** | 约 65 万亩 |
| **投产面积** | 超 50 万亩 |
| **年产量** | 干八角约 1.5—2 万吨 |
| **年产值** | 超 15 亿元 |

**品质特色**：色泽红润、香味浓郁、茴香油含量高，面积和产量均占云南省八角总产量的 70% 以上。富宁县是"中国八角之乡"，产品销往国内调味品市场及东南亚、欧美等地。

## 三、广南八宝米（地理标志产品）

| 指标 | 数据 |
|------|------|
| **种植面积** | 全州约 10 万亩，核心区八宝镇约 3 万亩 |
| **年产量** | 总产量约 4—5 万吨，八宝镇约 1.5 万吨 |
| **年产值** | 超 8 亿元 |

**品质特色**：颗粒饱满、色泽白润、饭香浓郁、口感软糯，历史上为朝廷贡米，目前作为高端优质米供应省内外，部分经电商渠道销往全国。

## 四、烤烟

| 指标 | 数据 |
|------|------|
| **种植面积** | 全州年均约 50 万亩，覆盖 8 县（市） |
| **年收购量** | 约 130—150 万担（6.5—7.5 万吨） |
| **烟农售烟收入** | 常年稳定超 20 亿元 |

**品质特色**：色泽金黄、油分充足，是云南中烟等企业的核心原料基地，烟叶质量全国领先，是文山州传统经济支柱和地方财政收入重要来源。

## 五、草果

| 指标 | 数据 |
|------|------|
| **种植面积** | 全州约 40 万亩（林下种植为主） |
| **投产面积** | 约 25 万亩 |
| **年产量** | 干草果约 1.2—1.5 万吨 |
| **年产值** | 超 10 亿元 |

**品质特色**：果大、香气浓郁，主产区马关、麻栗坡、富宁等县。文山草果面积和产量均居云南省前列，产品主要用于食品调味和中药材，销往国内市场并部分出口东南亚。

## 六、油茶

| 指标 | 数据 |
|------|------|
| **种植面积** | 全州约 80 万亩 |
| **投产面积** | 约 40 万亩 |
| **年产量** | 茶油约 2000—3000 吨 |
| **年产值** | 超 5 亿元 |

**品质特色**：广南县是"中国油茶之乡"，油茶林多为老林改造而来，茶油品质优良。目前正加大精深加工投入，延伸高端食用油、化妆品等高附加值产品链。

## 七、蒜头果（文山特有珍稀物种）

| 指标 | 数据 |
|------|------|
| **野生资源** | 约 10 万亩 |
| **人工种植** | 约 5 万亩 |
| **年产量** | 鲜果约 500—800 吨 |
| **年产值** | 神经酸产品超 1 亿元 |

**核心价值**：蒜头果种子含神经酸高达 50% 以上，是目前发现的自然界中神经酸含量最高的植物之一。神经酸对修复受损神经细胞有独特功效，属高端保健品和药品原料，市场前景广阔。该物种为国家二级保护植物，兼具生态与产业双重价值。

## 八、阳荷

| 指标 | 数据 |
|------|------|
| **种植面积** | 全州约 8 万亩，丘北核心区约 5 万亩 |
| **年产量** | 鲜阳荷约 10—12 万吨 |
| **年产值** | 超 6 亿元 |

**品质特色**：又名阳荷姜，口感鲜嫩、风味独特，既可鲜食也可加工为腌菜、罐头。传统以本地市场为主，近年来通过电商逐步走向全国，丘北县为核心产区。

## 产业结构总结

| 梯次 | 代表产品 | 年产值级别 |
|------|---------|------------|
| **第一梯队** | 丘北辣椒、烤烟 | 20 亿元级 |
| **第二梯队** | 富宁八角、草果 | 10—15 亿元级 |
| **第三梯队** | 八宝米、阳荷、油茶 | 5—10 亿元级 |
| **新兴潜力** | 蒜头果 | 1 亿元级（高速增长期） |

文山州特色农产品呈现"传统烟辣稳固、林下草果崛起、木本油料提速、珍稀物种破圈"的多元化格局，是高原特色农业的重要实践样本。
