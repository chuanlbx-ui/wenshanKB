# 广南八宝米

> 中国八宝贡米之乡 · 中国重要农业文化遗产 · 明清贡米

---

## 一、产地环境

广南八宝米核心产区位于文山州**广南县八宝镇**一带，地处"高原小桂林"腹地。

| 环境要素 | 特征 |
|----------|------|
| 海拔 | 1100-1600 米 |
| 气候 | 亚热带季风气候，年均温 16.5-17℃，年降雨约 1100mm |
| 日照 | 年日照超 1800 小时 |
| 无霜期 | 330 天以上 |
| 相对湿度 | 70%-80% |
| 土壤 | 岩溶地貌，富含钾元素，有机质含量高 |
| 水源 | 清澈八宝河水灌溉 |

"好山好水好田"的独特组合，是八宝米品质形成的关键。

---

## 二、品种特点

### 2.1 品种类型

以优质籼稻为主，代表性品种包括"八宝谷"系列、"文稻"系列等，在传统**贡米品种**基础上改良选育而成。

### 2.2 外观与口感

| 维度 | 特征 |
|------|------|
| 米粒形态 | 长椭圆形，白里透青或洁白油亮 |
| 蒸煮后 | 软糯适中、富有弹性、油润光亮 |
| 口感 | 清香回甘，**冷却后不易回生变硬**（区别于普通大米的关键优势） |

### 2.3 营养价值

- 蛋白质和脂肪含量高于普通大米
- 富含多种**氨基酸**及**维生素 E、B1**
- 富含**磷、钾、镁、硒**等微量元素

---

## 三、历史文化

### 3.1 悠久种植史

八宝米种植历史可追溯至距今约 **3200 年**的商周时期，是**云南最早栽培的水稻品种**之一。

### 3.2 贡米地位

明清时期被列为**朝廷贡米**，有"**每岁贡百担**"的记载，被誉为：

> "洁白无瑕，优质慧中，米中之花"

### 3.3 荣誉时间线

| 年份 | 荣誉 |
|------|------|
| 1981 年 | 被列为**国家级名贵米种** |
| 2001 年 | 广南县被授予"**中国八宝贡米之乡**" |
| 2009 年 | 获**农产品地理标志**登记 |
| 2014 年 | "云南广南八宝稻作生态系统"入选**中国重要农业文化遗产** |
| 2015 年 | 获**国家地理标志产品保护** |
| — | 多次获"绿色食品"认证，入选云南"**六大名米**" |

## 四、产业发展

### 4.1 产业规模（截至 2025 年）

| 指标 | 数值 |
|------|------|
| 种植面积 | **20 万亩** |
| 年产量 | **约 10 万吨** |
| 综合产值 | 突破 **20 亿元** |
| 带动农户 | 约 **7 万户**增收 |

### 4.2 发展模式

| 维度 | 内容 |
|------|------|
| 科技驱动 | 与中科院合作建立良种繁育基地，推广"稻+鱼/鸭"绿色生态种养 |
| 组织模式 | 推广"公司 + 基地 + 农户"订单农业，保底收购保障农户收益 |
| 加工升级 | 引进龙头企业，建设智能化加工生产线，提升产品附加值 |

### 4.3 品牌矩阵

| 品牌 | 定位 |
|------|------|
| "**八宝贡**" | 核心高端品牌 |
| "**云之香**" | 中高端市场 |
| "**句町润米**" | 地域文化品牌 |
| "**那糇芒**" | 壮族文化品牌 |

产品覆盖有机米、功能米、即食米、文创礼品米等多个系列，通过电商和直播销往北上广深等一线城市。

## 五、农文旅融合

依托"八宝贡米之乡"和"世外桃源"**坝美**等旅游资源：

- 举办**稻作文化节**、丰收节
- 发展稻田观光、农耕体验
- 实现"**卖米 + 卖风景**"双轮驱动

## 六、地理标志保护

| 保护类型 | 批准时间 | 保护内容 |
|----------|----------|----------|
| 农产品地理标志 | 2009 年 | 八宝米产品及产地 |
| 国家地理标志产品 | 2015 年 | 广南县 15 个乡镇指定区域，品种/栽培/加工全链条 |
| 中国重要农业文化遗产 | 2014 年 | 整个稻作生态系统（含传统知识、耕作制度及壮族文化） |

## 七、产业前景

1. **品牌溢价**：深化"贡米"文化 IP，提升高端市场占有率
2. **绿色有机认证**：扩大有机种植面积，对接高端消费市场
3. **深加工突破**：开发米粉、米线、速食米饭等衍生产品
4. **文旅深度融合**：串联坝美景区，打造"稻田 + 桃源"精品线路
