# 富宁八角产业

> 中国八角之乡 · 全国县域种植面积第一 · 73 万亩 · 年产鲜果 15-21 万吨

---

## 一、种植环境

富宁县位于云南东南部，属**低纬高原季风气候**，拥有八角的天然适生条件：

| 条件 | 富宁实际 | 八角适生要求 |
|------|----------|--------------|
| 气候 | 低纬高原季风气候 | 温暖湿润 |
| 土壤 | 山地红壤、黄壤（pH 4.5–6.5） | 酸性土壤 |
| 年均温 | 16 – 21 °C | 16 – 23 °C |
| 年降雨量 | 1000 – 1900 mm | >1000 mm |
| 无霜期 | >300 天 | >250 天 |

---

## 二、种植历史

| 时期 | 事件 |
|------|------|
| 清康熙末年（约 1722 年） | 从广西引种，开启八角栽培 |
| 清光绪年间 | 种植遍及全县，成为云南八角栽培**发源地**，并向周边扩散 |
| 20 世纪 60 年代 | 列为山区经济支柱产业，从粗放经营向集约经营转变 |
| **1999 年** | 被正式命名为**"中国八角之乡"** |
| 21 世纪 | 持续扩种、科技赋能、产业链一体化发展 |

---

## 三、规模与产量

| 年份 | 种植面积（万亩） | 鲜果产量（万吨） | 农业产值（亿元） | 综合产值（亿元） |
|------|-----------------|------------------|------------------|------------------|
| 2013 | 5.87 | ~1（干果） | — | — |
| 2022 | 69.2 | 18 | 27+ | — |
| 2023 | 70.8 | 15.2 | 18.24 | — |
| **2024** | — | **21** | 12.6 | **15.6** |
| **2025** | **73.61** | 14.55 | 16.56 | — |

**核心地位：**

- 种植面积稳定在 **70 万亩以上**，占全国约 **11%–12%**，长期位居**全国县级行政区第一位**
- 鲜果年产量约 15–21 万吨，约占全国总产量的 **三分之一**
- 覆盖农户 6.6 万户，惠及 30.1 万人口

## 四、产业链全景

### 4.1 上游——种苗与种植

| 环节 | 内容 |
|------|------|
| 良种繁育 | 建成采穗圃和苗圃基地，年出圃优质苗木数百万株 |
| 标准化种植 | 推广"降冠—疏密—精管"等良法 |
| 科技赋能 | 院士/专家工作站、物联网、大数据智慧种植 |

### 4.2 中游——加工与研发

| 项目 | 内容 |
|------|------|
| 龙头企业 | 云南众木农林发展有限公司，投资约 **2.5 亿元** |
| 加工能力 | 年加工鲜果数万吨 |
| 精深加工产品 | 八角精粉、茴香油、八角啤酒、冰淇淋、香水等 **10 余种** |
| 合作伙伴 | 海底捞、盒马鲜生等 |
| 科研平台 | 云南八角产业研究院，已获多项专利 |

### 4.3 下游——交易、品牌与文旅

| 环节 | 内容 |
|------|------|
| 交易平台 | 中国八角数字化服务平台 + 线下交易中心 |
| 品牌认证 | 国家农产品地理标志、地理标志证明商标、国家级农耕农品记忆名录 |
| 农文旅 | "八角古树十里长廊"、采摘节、乡村旅游融合 |

### 4.4 联农带农

| 模式 | 内容 |
|------|------|
| 组织模式 | "党组织 + 合作社 + 公司 + 基地 + 农户" |
| 利益分配 | "4 个五分钱"、"85:15" 等多种分配模式 |
| 增收效果 | 订单收购、保底分红，带动村集体和农户增收 |

## 五、市场地位

| 维度 | 评价 |
|------|------|
| 品质特征 | **"色红、香浓、味醇、型好"**，全国领先 |
| 全国份额 | 产量约占全国 1/3 |
| 市场覆盖 | 国内知名调味品企业重要原料基地，产品远销海内外 |

## 八、挑战与展望

1. **单产提升**：70 万亩面积中部分老林产量偏低，需科技改造
2. **价格波动**：八角作为香料原料，受国际市场行情影响较大
3. **深加工突破**：从原料输出向终端消费品转型，提升附加值
4. **品牌国际化**：借 RCEP 东风开拓东南亚及国际市场
