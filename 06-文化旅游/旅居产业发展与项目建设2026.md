> 数据来源：文山发布、人民网、云南网、文山发布网

## 一、品牌定位

- **核心品牌**："旅居云南·好在文山" / "三七之乡·康养文山"
- **五大体验**：好在文山（生态宜居）→ 游在文山（山河盛景）→ 居在文山（梦里乡愁）→ 养在文山（颐养身心）→ 乐在文山（烟火人间）
- 推动"中医药+文旅"融合，探索具有文山特色的中医药康旅新模式

## 三、品牌定位

- 核心品牌：**"旅居云南·好在文山"**
- 转型方向：从"**观光游**"向"**旅居游**"跨越升级
- 两大核心 IP：「**三七之乡·康养文山**」「**旅居云南·好在文山**」

### 五大主题

| 主题 | 方向 |
|------|------|
| 文旅体验 | 沉浸式文化旅游体验 |
| 康养度假 | 三七中医药康养度假 |
| 边境风情 | 中越边境特色风情 |
| 田园旅居 | 乡村田园慢生活 |
| 红色研学 | 老山等红色研学旅游 |

### 空间格局

**"一核引领、三区联动、两带串联"**

- **一核**：文山市核心引领区
- **三区**：丘北-砚山片区、广南-富宁片区、麻栗坡-马关-西畴片区联动
- **两带**：高铁旅游经济带、沿边跨境旅游带

### 五项重点工程

康养旅居示范样板工程 / 旅游品牌创建工程 / 研学旅游业态丰富工程 / 边境旅游发展工程 / 数字化旅游业态培育工程 + 精品业态提质升级

## 四、8 大旅居业态

| 业态 | 说明 |
|------|------|
| **乡村旅居** | 普者黑村、石家湾村等乡村田园生活体验 |
| **城镇旅居** | 文山市区及县城的城镇慢生活 |
| **医养融合旅居** | 三七中医药康养、砚山康养中心 |
| **度假旅居** | 普者黑喀斯特山水、坝美世外桃源 |
| **文艺旅居** | 民族非遗体验、坡芽歌书等文艺创作 |
| **边境旅居** | 中越边境特色文化、跨境体验 |
| **体育旅居** | 滇超联赛等赛事+旅居融合 |
| **养老旅居** | 暖冬康养、气候避寒 |
