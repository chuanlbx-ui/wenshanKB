# 红色旅游线路深度

## 一、红色旅游资源全景

文山州是**对越自卫还击战**的主战场之一，也是"老山精神"的发源地。全州共有红色旅游资源点 **20+** 处，涵盖战场遗址、烈士陵园、纪念馆、革命旧址等类型。

## 三、者阴山专题（1 天）

### 3.1 基本概况

- **位置**：麻栗坡县杨万乡，主峰海拔 1250+ m
- **历史**：者阴山是 1984 年"两山轮战"主战场之一，与老山齐名
- **现状**：战场遗址保存完好，尚未大规模旅游开发，游客量较小，体验更原真

### 3.2 游览要点

| 点位 | 内容 |
|------|------|
| 者阴山主峰 | 战场工事、猫耳洞遗址 |
| 杨万边贸市场 | 边民互市体验 |
| 者阴山前线指挥部旧址 | 历史陈列 |

### 3.3 与老山的区别

| 维度 | 老山 | 者阴山 |
|------|------|--------|
| 开发程度 | 国家 4A，设施完善 | 待开发，更为原生态 |
| 游客量 | 年均 100 万+ 人次 | 数万人次 |
| 交通 | 全程柏油路 | 部分路段为弹石路 |
| 体验 | 标准化参观 | 深度探访、沉浸式 |

## 四、其他红色资源整合

### 4.1 马关罗家坪大山

- 1980 年收复战战场遗址
- 主峰海拔 2002 m，可俯瞰中越边境风光
- 适合与都龙口岸串联游览

### 4.2 西畴精神教育基地

| 点位 | 内容 |
|------|------|
| 西畴石漠化展览馆 | 展示西畴人民炸石造地、脱贫历程 |
| 三光片区 | 石漠化治理示范区，红色精神当代传承 |
| 南疆烈士陵园 | 与老山烈士陵园互补 |

### 4.3 富宁红色资源

- 富宁是**左右江革命根据地**组成部分，有早期革命活动遗址
- 革命先驱李国柱故居（富宁县里达镇）
