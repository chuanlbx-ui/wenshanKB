# 文山州主要景点介绍

## 七大核心景区

| 景区 | 等级 | 位置 | 核心特色 |
|------|:--:|------|----------|
| 普者黑 | 5A | 丘北县 | 312 座孤峰、54 个湖泊、喀斯特湿地 |
| 坝美 | 4A | 广南县 | "世外桃源"、溶洞进出、壮族村落 |
| 老山圣地 | 4A | 麻栗坡县 | 对越自卫还击战遗址、红色旅游 |
| 砚山浴仙湖 | — | 砚山县 | 温泉康养、溶洞奇观 |

## 四大地质奇观

- **普者黑喀斯特峰林**：中国南方喀斯特世界自然遗产辐射区
- **坝美溶洞**：穿洞而过的桃源秘境
- **麻栗坡天保口岸**：中越边境口岸风光
- **富宁驮娘江**：峡谷漂流与原始森林

## 四条精品线路

1. **喀斯特山水游**：文山 → 普者黑 → 坝美（2-3 天）
2. **红色记忆游**：文山 → 麻栗坡老山 → 天保口岸（2 天）
3. **边境风情游**：文山 → 马关 → 麻栗坡 → 富宁（3 天）
4. **康养休闲游**：文山 → 砚山浴仙湖 → 丘北（2-3 天）
