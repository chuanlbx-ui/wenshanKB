## 活动亮点

### 1. "百人直播"模式
集结 **100余位新媒体达人**，包括 @昆明老外史蒂夫、@跟着嘉宁去旅游、@韦朵朵.麦唱的小女孩 等，同步联动中央及省州主流媒体，通过镜头与直播将万亩荷塘、喀斯特山水、民族风情实时传递给全国观众。

### 2. 文山州文旅推荐官授牌
现场举行文山州文旅推荐官授牌仪式，新媒体达人化身家乡代言人，以歌声、才艺、康养美食等多元视角呈现"有一种叫云南的生活"。

### 3. 配套活动矩阵

| 板块 | 内容 |
|------|------|
| **线路发布** | 精品旅游线路与惠民政策集中发布 |
| **文艺展演** | 民族打跳、弦子舞等轮番上演 |
| **非遗体验** | 非遗市集、彝族服饰体验 |
| **公共服务** | 流动图书进景区 |
| **直播带货** | 丘北辣椒、特色白葡萄酒等好物推向云端 |

### 4. 一站式体验闭环
实现"赏美景、品文化、购好物、享生活"的全链路文旅消费体验。

## 品牌战略

近年来，丘北县依托优越的旅游资源、地理位置和民族文化底蕴，以节庆为载体，联动企业、官方媒体、新媒体逐步构建 **全媒体矩阵**，创新传播力赋能文旅高质量发展，让"世界喀斯特 中国普者黑""旅居普者黑"品牌深入人心。
