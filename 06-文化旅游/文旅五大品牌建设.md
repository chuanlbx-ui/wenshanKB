# 文山州文旅五大品牌建设

> 2026年6月3日云南省"开局起步'十五五'"新闻发布会文山专场上，副州长田燕介绍文山将打造五大文旅品牌。央广网6月4日刊发深度报道。

## 一、五大品牌体系

| 品牌 | 核心内涵 |
|------|---------|
| **好在文山** | 北回归线穿境而过的温润气候，四季宜居、生态优良 |
| **游在文山** | 普者黑和坝美一汪碧水与千峰叠翠，"水陆空铁"通江达海立体交通 |
| **居在文山** | 旅居业态百花齐放，村村可旅居、四季可康养的全域格局 |
| **养在文山** | "中医药+文旅"融合，医康养健食药游赏学购十维康养体系 |
| **乐在文山** | 民族节庆、体育赛事、非遗体验等沉浸式文化活动 |

## 二、五大核心业态

| 业态 | 代表项目 |
|------|---------|
| 山水游 | 普者黑、坝美 |
| 红色游 | 老山 |
| 边境游 | 天保口岸、田蓬口岸 |
| 文化游 | 广南古城南街、非遗项目进景区 |
| 赛事游 | 四季主题文旅活动、体育赛事 |

## 三、旅游黄金三角

构建**丘北—广南—麻栗坡**"旅游黄金三角"：

| 节点 | 特色定位 |
|------|---------|
| 丘北 | 普者黑山水田园为核心，辐射乡村旅游 |
| 广南 | 坝美世外桃源+六郎城三产融合旅居+古城文化 |
| 麻栗坡 | 老山红色旅游+边境风情体验 |

## 四、"中医药+文旅"十维融合

在"医、康、养、健、食、药、游、赏、学、购"十个维度中找到康养兴奋点：

| 维度 | 内容 |
|------|------|
| 医 | 中医诊疗、民族医药体验 |
| 康 | 康复理疗、慢病管理 |
| 养 | 气候康养、森林康养 |
| 健 | 运动健身、户外徒步 |
| 食 | 三七药膳、民族特色饮食 |
| 药 | 三七系列产品、道地药材 |
| 游 | 康养旅游线路 |
| 赏 | 自然风光、民族风情 |
| 学 | 中医药研学、非遗传承 |
| 购 | 三七制品、民族手工艺品 |

## 五、旅居示范样板

| 样板 | 模式 |
|------|------|
| 广南六郎城 | 三产融合旅居新模式，全省旅居示范 |
| 砚山康养中心 | "医、养、游"一体化 |
| 富宁 | 艺术旅居 |
| 麻栗坡 | 红色旅居 |
| 普者黑 | 乡村旅居 |

## 六、战略定位

"十五五"期间，文山州围绕"三七康养旅居目的地"定位，将文山打造为全省康养旅居胜地，推动从"路过式旅游"向"留下来旅居"转变。
