# 文山州"十五五"文旅发展规划

> **原标题**：守正创新 奋力开创文山"十五五"文旅发展新局面

## 二、战略定位

从"**旅游过境地**"向"**旅居目的地**"跨越升级。

- 两大核心 IP：「**三七之乡·康养文山**」「**旅居云南·好在文山**」
- 规划文本：14 个板块 + 18 个专栏
- 重点项目：**88 个**（20 个申请省级、21 个申报州级）

## 三、黄金三角

构建「**丘北—广南—麻栗坡**」旅游黄金三角，打造核心增长极：

- **丘北（普者黑）**：喀斯特山水田园，近 500 家特色民宿，年接待约 60 万人次
- **广南（坝美）**：原生态自然美，引入轻餐饮、文创零售、灯光秀、实景演艺等
- **麻栗坡**：红色旅游 + 乡村旅游

## 四、A级景区与品牌建设

全州 **64 个** A 级景区同步推出专属福利。四季主题精准导流：

| 主题 | 内容 | 
|------|------|
| 春花茶咖 | 春季踏青赏花 |
| 夏凉菌香 | 夏季避暑康养 |
| 秋丰送爽 | 秋季丰收体验 |
| 冬暖花开 | 冬季暖冬旅居 |

省外客源占比达 **51.24%**。

## 五、康养旅居体系

深化「康养+文旅」融合：
- 「**1+1+10**」中医药康养模式
- 「**1+4+2**」医养中心提质
- 森林、湿地、草原康养基地建设
- 推出大健康产品，升级智慧生态系统

## 六、联农带农（典型案例）

| 案例 | 模式 | 成效 |
|------|------|------|
| 六郎城 | 「五金增收」模式 | 全国全省示范典型 |
| 普者黑村 | 多元旅游产品 + 民宿 | 近 500 家民宿，人均纯收入 5 万余元 |
| 石家湾村 | 「乡村旅居运营人 + 村集体 + 农户」+ KPI 机制 | 户均创收 3 万余元 |

## 七、数字文旅

- 旅居地图小程序：汇聚 A 级景区、旅居村、非遗工坊、文化场馆、精品线路等 **600 余个**旅游资源
- 官方新媒体账号集群（抖音、小红书等）
- 滇超联赛「赛事+文旅」融合宣传

## 八、十五五重点方向

1. **景区强基焕新**：普者黑聚焦喀斯特山水田园、坝美突出原生态自然美，开发露营、研学等新消费场景
2. **文艺精品创作**：《坡芽歌书》歌舞剧为"头号工程"
3. **历史文脉赓续**：第四次全国文物普查、资源总目录及数字化基因库、"拯救老屋"行动、大王岩岩画保护、铜鼓舞传承
4. **非遗活态传承**：策划非遗旅游线路、谋划建设"非遗大观园"
5. **宣推矩阵**：主流媒体 + 新媒体账号集群，两大核心 IP 持续打造
