## 四、五维竞争力综合评估

| 维度 | 领跑者 | 追赶者 | 说明 |
|------|--------|--------|------|
| **经济总量** | 文山市、砚山县 | 广南县 | 文山-砚山双核占全州 GDP 约 43% |
| **人均水平** | 文山市、砚山县 | 广南县 | 广南人口多但人均垫底（~28,000） |
| **财政健康度** | 文山市 | 西畴县 | 西畴财政自给率仅 ~15% |
| **产业多元性** | 砚山县、文山市 | 西畴县、麻栗坡县 | 砚山铝+建材+食品多元，西畴过于依赖生态 |
| **增长潜力** | 麻栗坡县、砚山县 | 广南县 | 麻栗坡口岸经济爆发（外贸+38.8%），砚山工业持续扩产 |

### 五维雷达概要

```
            经济总量
              ▲
              |
      增长潜力---人均水平
              |
      产业多元---财政健康

领跑者：文山市（总量+人均+财政）
潜力股：麻栗坡县（口岸爆发）、砚山县（工业扩产）
需关注：广南县（人均低+财政弱）、西畴县（体量小+产业单一）
```

## 五、趋势判断

1. **双核格局稳定**：文山市 + 砚山县合计占全州 GDP 约 43%，工业化和城镇化继续向双核集中。
2. **边境县崛起**：麻栗坡（天保口岸）、马关（都龙口岸）、富宁（田蓬口岸）在新一轮口岸经济政策红利下将加速追赶。
3. **文旅县差异化**：丘北（普者黑）+ 广南（坝美）形成"喀斯特山水文旅双城"，但需解决旅游季节性波动和人均偏低问题。
4. **西畴路径独特**：不走规模竞争，以"西畴精神"+"石漠化治理"+"康养"走小而精的差异化路线。
