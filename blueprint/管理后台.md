## 二、内容管理

### 2.1 笔记 CRUD

**创建笔记流程**：

```
会员/编辑 点击"新建笔记"
        │
        ▼
┌──────────────────┐
│  编辑器界面       │
│  - Markdown 编辑  │
│  - Frontmatter 表 │
│  - 实时预览       │
│  - wikilink 自动  │
│    补全           │
└────────┬─────────┘
         │ 保存草稿
         ▼
┌──────────────────┐
│  自动检查         │
│  ✓ frontmatter   │
│    完整性         │
│  ✓ 链接有效性     │
│  ✓ 敏感词过滤     │
│  ✓ 合规规则引擎   │
└────────┬─────────┘
         │
    ┌────┴────┐
    ▼         ▼
  通过      警告/拦截
    │         │
    ▼         └──→ 返回修改
┌──────────┐
│ 提交审核  │
└──────────┘
```

**状态流转**：

```
草稿 ──→ 待审核 ──→ 已发布 ──→ 已归档
  │         │         │
  │         └──→ 已驳回 ──→ 草稿（修改后重新提交）
  │                   │
  └───────────────────┘
```

**编辑与版本管理**：
- 每次编辑保存生成新版本（Git 式 diff 存储）
- 支持版本对比和回滚
- 已发布内容的编辑自动进入"待审核"状态，原版本保持在线

### 2.2 分类与标签管理

**分类体系**：继承现有 13 个主题分区，新增分类需超管审批

```
管理界面
├── 分类列表（拖拽排序）
│   ├── 00-总览
│   ├── 01-地理与自然环境
│   ├── ...
│   ├── 12-跨分类合成笔记
│   └── 13-AI 创作赋能中心
├── 标签云（管理）
│   ├── 合并重复标签
│   ├── 标签别名（如 "三七" → "文山三七"）
│   └── 标签使用统计
└── wikilink 图谱
    ├── 孤立笔记检测
    ├── 断链检测
    └── 推荐关联
```

### 2.3 审核工作流

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  待审核队列   │────▶│  审核详情页   │────▶│  通过/驳回   │
│              │     │              │     │              │
│  - 按时间排序 │     │  左侧: 渲染   │     │  通过 → 发布 │
│  - 按提交者   │     │  右侧: 检查   │     │  驳回 → 附   │
│  - 按分类     │     │  清单         │     │  理由通知   │
│              │     │              │     │  提交者     │
└─────────────┘     └─────────────┘     └─────────────┘

检查清单（审核员必查）：
☐ frontmatter 完整（title / created / category / tags）
☐ 数据引用的来源和年份标注
☐ 边境/民族/军事内容符合 [[../ai-hub/合规指南]]
☐ wikilink 指向的笔记存在
☐ 无敏感词（自动检查辅助）
☐ 内容原创性或引用有标注
```

### 2.4 批量操作

| 操作 | 说明 |
|------|------|
| 批量导入 | 上传 Markdown 文件/zip，自动解析 frontmatter |
| 批量修改标签 | 选择多篇笔记 → 统一添加/移除标签 |
| 批量归档 | 将过期笔记批量标记为"已归档" |
| 批量导出 | 选择笔记导出为 Markdown / PDF / JSON |
| 从 Obsidian 同步 | 增量同步 D:\WenShanKB 变更（需手动触发） |

| 告警项 | 触发条件 | 通知方式 |
|--------|----------|----------|

| 审核积压 | 待审核 > 20 篇 | 通知审核员 |

## 六、数据库表设计（核心表）

```sql
-- 用户表
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    username VARCHAR(50) UNIQUE,
    phone VARCHAR(20) UNIQUE,
    wechat_openid VARCHAR(100) UNIQUE,
    role VARCHAR(20) NOT NULL DEFAULT 'user',  -- super_admin/editor/reviewer/member/user
    display_name VARCHAR(100),
    avatar_url TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    last_login_at TIMESTAMPTZ,
    is_active BOOLEAN DEFAULT TRUE
);

-- 笔记表
CREATE TABLE notes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title VARCHAR(200) NOT NULL,
    slug VARCHAR(200) UNIQUE,
    content TEXT NOT NULL,           -- Markdown 原文
    content_html TEXT,               -- 渲染后的 HTML（缓存）
    frontmatter JSONB DEFAULT '{}',  -- YAML frontmatter 解析为 JSON
    category VARCHAR(50),
    tags TEXT[] DEFAULT '{}',
    status VARCHAR(20) DEFAULT 'draft',  -- draft/pending_review/published/archived
    author_id UUID REFERENCES users(id),
    reviewer_id UUID REFERENCES users(id),
    version INTEGER DEFAULT 1,
    view_count INTEGER DEFAULT 0,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    published_at TIMESTAMPTZ,
    embedding vector(1536)           -- pgvector，语义检索用
);

-- 笔记版本表
CREATE TABLE note_versions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    note_id UUID REFERENCES notes(id) ON DELETE CASCADE,
    version INTEGER NOT NULL,
    content TEXT NOT NULL,
    diff TEXT,                       -- Git diff 格式
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 操作日志表
CREATE TABLE operation_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    action VARCHAR(50) NOT NULL,     -- create/update/delete/review/...
    target_type VARCHAR(50),         -- note/category/tag/user/...
    target_id UUID,
    detail JSONB,
    ip_address INET,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 搜索日志表
CREATE TABLE search_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    query TEXT NOT NULL,
    result_count INTEGER,
    has_click BOOLEAN DEFAULT FALSE,
    source VARCHAR(20),              -- web/miniapp/api/agent
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```
