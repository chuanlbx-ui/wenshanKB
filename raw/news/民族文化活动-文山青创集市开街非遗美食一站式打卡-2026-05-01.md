# 文山青创集市开街，创业就业非遗美食一站式打卡

4月30日—5月4日，"青创赋能新发展·文青筑梦创未来"——文山州第三届青年创业就业集市暨2026年大学生"毕业季"招聘会及第五届青年文化周活动在文山市举行。

## 活动规模

主会场七花广场与分会场光大步行街双向联动，188个特色展位、八大主题展区集中亮相。

## 八大主题展区

| 展区 | 特色内容 |
|------|----------|
| 青创街区 | 青年创业者的文创设计、特色手作、电商服务 |
| 招聘街区 | 三七医药、现代农业、文化旅游等领域岗位 |
| 国补促消费街区 | 购车优惠、家电以旧换新 |
| 农特产品街区 | 三七、八角、古树茶、生态果蔬等"文品出山" |
| 非遗街区 | 壮族刺绣、苗族银饰、彝族剪纸等非遗技艺展演 |
| 中医体验街区 | 义诊服务 |
| 科技街区 | 科创成果展示 |
| 美食街区 | 特色民族美食 |

## 意义

集创业成果展示、就业精准对接、非遗活态展演、惠民消费助力和文旅融合于一体，以青春活力赋能文山高质量发展。
