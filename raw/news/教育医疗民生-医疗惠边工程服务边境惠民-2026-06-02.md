# 文山州实施医疗惠边工程服务边境惠民生

"十五五"开局之年，文山州紧扣边境地区"两增长三提升"三年行动方案，全面深化"三医"协同发展和治理，以公立医院改革、边境医联体建设为载体，统筹推进医疗惠边工程落地见效。

## 医疗惠边政策稳步推进

| 领域 | 成果 |
|------|------|
| 医改考核 | 全州医改综合考核A+等次，西部170个地市排名第18位 |
| 创新服务 | "一次挂号管三天"入选全省医改典型案例 |
| 基层达标 | 14个边境乡镇卫生院、42个边境村卫生室均达国家基本标准 |
| 急诊急救 | 全州8县市人民医院均达二甲及以上，县级"五大中心"全覆盖 |
| 专病防治 | 构建"州级引领、县级支撑、基层托底"整合型专病防治体系 |
| 医联体 | 组建全域边境医联体，推行"共建科室""共管病房"模式 |

## 分级诊疗体系

- "1+8+N"三级会诊转诊网络，打通州、县、乡、村四级转诊通道
- 转诊申请、专家会诊、医保备案一站式办理、24小时响应

## 健康目标（2030年）

- 每千人口执业（助理）医师数达3.73人
- 孕产妇死亡率下降至9/10万及以下
- 婴儿死亡率控制在5‰以内
- 形成体系完整、功能互补、运行高效的整合型卫生健康服务体系
