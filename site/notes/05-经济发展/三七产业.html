三七产业

经济 产业 三七 中药材 置信度: 低 经济发展

三七产业

产业地位

  • 文山是三七的原产地、主产区，占全国产量、质量、产值第一
  • "中国三七之乡"、"世界三七之都"
  • 国家地理标志保护产品

产业规模

  • 以三七为主的中药材种植面积：254 万亩
  • 2025年综合产值 456 亿元，增长 10.4%
  • 荣获全国优势特色产业集群等 9 个国家级荣誉

产业链

```

标准化基地建设 → 精深加工 → 科技研发 → 市场流通 → 联农带农

```

保障体系

  • 全国首个单一中药材品种地方条例：《文山三七发展条例》
  • 文山三七研究院、文山学院三七医药学院
  • 覆盖全产业链的国际/国家/行业/地方质量标准体系
  • 三七种植系统被认定为中国重要农业文化遗产
  • 人工栽培历史 400 年以上
