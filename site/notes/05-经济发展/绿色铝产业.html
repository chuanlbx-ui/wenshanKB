完整产业链

```

铝土矿 → 氧化铝 → 绿色铝（电解铝）→ 精深加工 → 再生利用

```

  • 铝土矿占全省 90% 以上
  • 全国产能近 1/10、全省 1/2

发展挑战

  1. 优惠电价退出影响成本
  2. "双碳"政策约束加强
  3. 合金化率待提升（2023年仅42.5%）
  4. 下游精深加工产业链仍需延伸
