# 自进化学习规则

## 一、定时增量更新

通过 Marvis 定时任务，每周自动执行以下流程：

1. 使用 AnySearch 搜索 "云南文山 最新 2025 2026" 获取最新资讯
2. 提取重要新闻的全文内容
3. 判断是否属于已有 10 大分类
4. 如属新信息，生成新笔记或更新已有笔记
5. 更新本文件中的 `更新记录`

## 二、交叉引用规则

每篇笔记必须包含：

- `tags` 字段：至少 1 个分类标签
- `related` 字段：至少 2 个双向链接 `[[笔记名]]`
- 正文中嵌入 `[[相关笔记]]` 链接

## 三、知识图谱维护

- 通过 Obsidian Graph View 定期检查孤立节点
- 确保每篇笔记至少有 2 条以上入链/出链

## 四、质量审核

- 每季度人工或 AI 审核知识库内容时效性
- 标注过时内容为 `status: outdated`
- GDP、人口等数据标注数据年份

## 五、AI Agent 调用接口

知识库面向 AI Agent 提供以下调用方式：

- 直接读取 `.md` 文件
- 通过 Obsidian API 查询
- 全文搜索标签或关键词
- 图谱遍历关联知识
