# 文山州医疗深度（2024年数据）

## 医疗规模

| 指标 | 数值 |
|------|------|
| 卫生健康机构 | **1,600 个**（+81） |
| 实有床位数 | **27,173 张**（+2,154） |
| 每千人口床位 | 8.06 张 |
| 卫生技术人员 | **30,173 人**（+2,754） |
| 执业（助理）医师 | 9,217 人（+1,125） |
| 每千人口卫生技术人员 | 8.95 人 |
| 卫生健康支出 | **53.53 亿元**（占一般公共预算 13.8%） |

## 医疗改革进展

- 8县（市）综合医院和 119 个乡镇卫生院全部达到国家服务能力标准
- 紧密型县域医共体建设全部达到评判标准
- 成立城市医联体、中医药联盟
- 院前急救实现统一调度指挥
- 公立医院民族医学科实现"零突破"

## 主要医疗机构

- **文山州人民医院**：全州最大的三级综合医院
- **文山市人民医院**：地址文山市凤凰路31号
- 各县（市）人民医院、中医医院

## 预防接种

全州各乡镇卫生院/社区卫生服务中心均设有预防接种门诊
