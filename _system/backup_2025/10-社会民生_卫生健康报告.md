# 文山州卫生健康报告

## 一、医疗资源（2024年）

| 指标 | 数值 | 较上年变化 |
|------|------|-----------|
| 卫生健康机构 | **1,600 个** | +81 |
| 实有床位数 | **27,173 张** | +2,154 |
| 每千人口床位数 | **8.06 张** | +0.69 |
| 卫生技术人员 | **30,173 人** | +2,754 |
| 执业（助理）医师 | **9,217 人** | +1,125 |
| 每千人口卫生人员 | **8.95 人** | +0.88 |

"十四五"对比：

| 指标 | 2020年 | 2024年 |
|------|--------|--------|
| 每千人口床位数 | 6.54 | **8.06** |
| 每千人口卫生人员 | 7.09 | **8.95** |
| 每千人口执业医师 | 1.87 | **2.73** |
| 每千人口注册护士 | 3.42 | **4.27** |

## 二、医疗服务体系

- **三级医院**：7 个
- 州人民医院：城市医疗集团牵头
- 州中医医院：中医药联盟牵头
- 州皮防所：皮肤病专科联盟牵头
- 8 县（市）人民医院全部达二级以上
- 妇幼保健院全部达二级甲等
- 119 个乡镇卫生院达国家标准（99个基本+20个推荐）
- 急诊急救"五大中心"全覆盖
- 院前急救统一调度

## 三、医改成效

- 2023年全州医改：西部地区 182 地市中排名 **第 13 位**，全国 428 地市中排名 **第 29 位**
- 8 县（市）全部建成紧密型县域医共体
- "文山经验""砚山模式"获省卫健委推广
- 药品耗材集中带量采购：村卫生室药品配备 80 种+
- 基本医保覆盖 350 万+，参保率稳定在 **100%**

## 四、疾病防控

| 指标 | 成就 |
|------|------|
| 法定传染病总发病率 | 连续 **24 年**低于全省平均 |
| 无脊灰 | 连续 **34 年** |
| 无霍乱 | 连续 **26 年** |
| 无鼠疫 | 连续 **22 年** |
| 无疟疾本地感染 | 连续 **12 年** |
| 消除麻风危害 | 获授"全国消除麻风危害示范区" |

### 艾滋病防治
- 治疗率 **95.8%**，有效率 **97.2%**
- "3个95%"目标已提前实现 2 个
- 三病母婴阻断连续 **6 年**达世卫消除认证标准
- 持续 **24 年**无艾滋病输血传播

## 五、居民健康水平

| 指标 | 2020年 | 2024年 |
|------|--------|--------|
| 人均预期寿命 | 74.25 岁 | **75.06 岁** |
| 居民健康素养 | 16.58% | **27.09%** |
| 重大慢病过早死亡率 | 19.35% | **16.95%** |

## 六、对口帮扶

- 省内外 **16 家**三级医院选派 **86 名**专家对口帮扶 11 家县级医院
- 国家级临床重点专科 **1 个**，省级临床重点专科 **56 个**
- 建成全省首个血液科普教育基地
- 上海市胸科医院先心病筛查工作室落户州妇幼保健院

## 七、中医药

- 州中医医院新增 5 个省级名老中医药专家传承工作室
- 县级中医医院"两专科一中心"全覆盖
- 中医药适宜技术推广中心 8 个
- 公立医院设立民族医学科实现"零突破"
