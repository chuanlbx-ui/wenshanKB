# 文山州十五五规划：3+3 发展定位与四大重点产业

> 来源：文山州政务网文章《决胜十四五 启航十五五 谱写中国式现代化文山实践新篇章》（2026年6月16日）

## 一、十四五成就回顾

- **经济总量**：GDP 突破 1600 亿元。
- **绿色铝**：首个千亿级产业。
- **三七**：文山三七入选全国优势特色产业集群。
- **民族团结与双拥**：创成全国民族团结进步示范州、全国双拥模范城。
- **生态减贫**：石漠化治理经验获评第五届全球最佳生态减贫案例。

## 二、十五五"3+3"发展定位

| 维度 | 定位 |
|------|------|
| **三大定位** | 民族团结进步示范区 |
|  | 石漠化地区生态文明建设排头兵 |
|  | 云南面向东南开放先行区 |
| **三个示范区/区/目的地** | 新时代兴边富民示范区 |
|  | "绿色铝谷"核心区 |
|  | 三七康养旅居目的地 |

## 三、四大重点产业

### 1. 绿色铝

- 2025 年电解铝产能占全省 **52.8%**、全国 **7.7%**。
- 文山为全国"北铝南移"标志区。
- 推动绿色铝向新材料、终端制造延伸。
- **目标**：2030 年铝产业综合产值突破 **1800 亿元**，打造云南"绿色铝谷"核心区。

### 2. 三七中药材

- 原产地主产区，向**千亿级产业**目标迈进。
- 打造云南"第二个烟草产业"。

### 3. 高原特色现代农业

- 北回归线贯穿，"天然温床"。
- 建成全国优质绿色现代农产品生产基地和粤港澳大湾区"菜篮子"基地。

### 4. 康养旅游旅居

- 打造"有一种叫云南的生活——三七之乡 康养文山"品牌。
- 推广"旅居云南·好在文山"品牌。

## 四、对外开放

- **富宁港**：年内建成，与平陆运河同步投运，打造云南出海最近"黄金港口"。
- **铁路**：文蒙铁路 2028 年建成通车，文靖铁路 2030 年建成通车。

## 五、生态环境

- 空气优良率 **97.5%**。
- 国省控地表水断面水质优良率 **100%**。
- 马关县、麻栗坡县创成全国生态文明建设示范区。

## 六、兴边富民

- 边境线 438 公里，少数民族占 **57.5%**。
- 创建新一轮全国民族团结进步示范州。
- 边境地区"两增长三提升"三年行动。
