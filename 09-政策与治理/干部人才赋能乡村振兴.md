# 干部人才赋能乡村振兴工作机制

> 2026年6月3日云南省"开局起步'十五五'"新闻发布会文山专场上，州委常委、组织部部长陈治介绍"十五五"时期文山州干部人才赋能乡村振兴的四项举措。央广网6月4日刊发详细报道。

## 一、总体思路

通过"选、育、用、留"四措并举，引导激励各类干部人才奔赴乡野、服务乡村、干事创业：

| 环节 | 核心举措 |
|------|---------|
| 聚焦需求**选** | 精准引才、按需匹配 |
| 因材施策**育** | 分类培养、实践锻炼 |
| 搭建平台**用** | 下沉一线、发挥作用 |
| 优化环境**留** | 激励保障、正向反馈 |

## 二、"721"干部工作法与"五访"制度

健全完善干部识别与考察机制：

| 制度 | 内容 |
|------|------|
| **"721"干部工作法** | 七分看实绩、两分看评价、一分看潜力，建立科学干部评价体系 |
| **"五访"制度** | 常态化走访考察，多渠道了解干部在乡村振兴一线的实际表现 |

## 三、"智汇文山·菁英相约"行动

深入实施人才引进专项行动：
- 面向全国引进各类急需紧缺人才
- 重点覆盖农业科技、乡村规划、教育医疗、文旅产业等领域
- 强化柔性引才机制，鼓励"候鸟式"专家服务

## 四、"三个计划"下沉体系

建立三级干部挂联帮扶机制：

| 计划 | 覆盖范围 | 说明 |
|------|---------|------|
| **百名科级干部挂百乡** | 科级干部 → 乡镇 | 百名科级干部挂钩联系百个乡镇 |
| **千名处级干部联千村** | 处级干部 → 行政村 | 千名处级干部联系千个行政村 |
| **万名各级干部帮万户** | 各级干部 → 农户 | 万名干部结对帮扶万户群众 |

## 五、"兴边人才集市"

深化拓展边境地区人才平台建设：
- 打造边境地区"人才超市"，精准匹配供需
- 聚焦沿边产业发展和边境治理需求
- 鼓励专业技术人才、返乡创业人才、乡土人才向边境流动

## 六、基层干部激励保障

严格执行一线优先导向：

| 指标 | 比例要求 |
|------|---------|
| 州、县两级每年从一线提拔重用干部比例 | **不低于总数 60%** |
| 州、县两级每年从一线晋升职级干部比例 | **不低于总数 60%** |

- 树立"干部从基层来、到基层去"的鲜明导向
- 强化乡村振兴一线干部的待遇保障和成长通道
