# 云南文山：聚力招商 以产业突破激活发展动能

> **来源**：新华网 | **发布时间**：2026-06-08

## 一、2026年1-4月招商引资成果

2026年以来，文山州全面落实云南省委"3815"战略发展目标，把招商引资作为推动高质量发展的"一号工程"。

| 指标 | 数据 |
|------|------|
| 新签约项目 | **28个** |
| 协议总投资 | **27.81亿元** |
| 其中亿元以上项目 | **9个** |
| 新落地项目 | **39个** |
| 其中亿元以上项目 | **14个** |
| 产业招商项目到位资金 | **52.97亿元** |

## 二、政策框架与工作机制

文山州出台《进一步加强新形势下招商引资工作的实施意见》，构建权责清晰、靶向精准、闭环推进的招商工作体系：

- **四方联动机制**：省州县三级联动，争取省级政策、平台、项目、培训全方位支持
- **强招商促发展六条措施**：常态化开展联合招商、企业座谈、项目对接
- **项目引进全流程管理**：闭环推进，强化全流程跟踪服务

## 三、2026年首批重点招商项目

2026年首批谋划重点产业招商项目 **133个**，总投资额 **669.87亿元**。筛选 **10个** 牵引性强、带动性大、产业链长的标杆项目重点推介、重点攻坚，以点带面推动产业集群发展。

## 四、十五五产业招商方向

文山州锚定"产业强州"战略，聚焦延链补链强链，四大重点产业招商方向：

### 绿色铝
- 重点招引：电池铝箔、动力电池壳体、再生铝综合利用、航空航天、低空经济
- 十五五目标：力争**2030年**铝产业综合产值突破 **1800亿元**（锚点：当前绿色铝产值1000亿元）

### 三七
- 重点招引：三七提取物、中药配方颗粒、三七健康食品
- 转型路径：从"原料基地"向"精深加工高地"转型
- 十五五目标：力争**2030年**实现千亿级产业目标（锚点：当前三七综合产值456亿元）

### 高原特色现代农业
- "1+4+N"产业体系：三七中药材、蔬菜、水果、肉牛 + N个特色集群
- 重点招引：绿色食品加工、冷链物流
- 定位：粤港澳大湾区"菜篮子"基地、绿色食品加工基地

### 文旅康养
- 五大主题：文旅体验、康养度假、边境风情、田园旅居、红色研学
- 重点招引：大型文旅综合体、医养融合旅居、边关特色民宿集群
- 目标：从"旅游过境地"向"旅居目的地"转型升级
