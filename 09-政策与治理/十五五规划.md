# 文山州"十五五"规划纲要

> **数据来源**：2026年6月3日云南省"开局起步'十五五'"主题系列新闻发布会·文山专场

## 总体成就（"十四五"收官）

- 地区生产总值达 **1632.64 亿元**，年均增速高于全国全省
- 规模以上工业总产值突破千亿元
- 研发经费投入年均增长 **28.38%**
- **民营经济占 GDP 比重、产业投资占固投比重均超 62%**

## 发展定位："3+3"

| 三大定位 | 三大特色定位 |
|----------|-------------|
| 民族团结进步示范区 | 新时代兴边富民示范区 |
| 石漠化地区生态文明建设排头兵 | "绿色铝谷"核心区 |
| 云南面向东南开放先行区 | 三七康养旅居目的地 |

## 四大重点："产业、项目、干部、环境"

## 产业基础

| 领域 | 关键指标 |
|------|---------|
| 绿色铝 | 电解铝产能 343 万吨（占全省52.8%、全国7.7%），铝加工产能 418 万吨，2025年产值1000亿，再生铝实现零突破 |
| 三七 | 入选全国优势特色产业集群，综合产值456亿（2025，广义口径），核心产业口径220亿（+7%），7个种质资源圃，GAP基地7000亩→2026年扩至1.5万亩→2028年全覆盖 |
| 农业 | 552 家龙头企业，583 个绿色优质农产品，高标准农田 290.42 万亩 |
| 旅游 | 64个A级景区，2026年1-4月接待游客1643.45万人次 |
| 数字经济 | 百度智能云（广南）基地建成投运 |

### 重大项目储备

- 共谋划储备重点产业类项目 **2845 个**，总投资接近万亿
- 同步布局风电光伏、储能、大数据等新兴产业

### 十五五核心目标

| 产业 | 目标 |
|------|------|
| 绿色铝 | 2030年目标综合产值突破 **1800 亿** |
| 三七中药材 | 向 **千亿级** 产业迈进 |
| 高原现代农业 | 总产值突破 **550 亿** |
| 康养旅游旅居 | 打造"三七之乡 康养文山""旅居云南·好在文山"品牌 |
