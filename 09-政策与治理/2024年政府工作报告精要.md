# 文山州2024年政府工作报告精要

> 州第十五届人大四次会议（2024年1月18日），州长马忠俊

## 2023年回顾

| 指标 | 数值 |
|------|------|
| 地区生产总值(2023) | **1462.3 亿元**（+5.1%） |
| 规上工业增加值增速 | 12.8%（全省第2） |
| 地方一般公共预算收入增速 | 10.3% |
| 全社会用电量 | 420.29 亿千瓦时（全省第1） |
| 存款增速 | 18.9%（全省第1） |
| 减税降费 | 21.87 亿元 |
| 净增经营主体 | 10.3 万户 |

### 产业

- 343 万吨铝产业产能全部建成，产值 **1000 亿元**（2024年数据），度电产值 1.82 元
- 中药材综合产值 **456 亿元**（2024年数据，+13.3%）
- 高原特色农业全产业链产值 **1,456 亿元**（+14.3%）
- 园区工业总产值 **720 亿元**，占全州 73.3%
- 外贸 27.99 亿元 [置信度: 低]（+386%），边民互市 20.75 亿元
- 旅游接待 4,649 万人次 [置信度: 低]，收入 504.75 亿元

### 民生

- 民生支出占 83.3%
- 脱贫人口人均纯收入 **17,388 元**（+13%）
- 高考一本率 9.2%→**18.4%**，州一中 **58.7%**
- 文山职业技术学院正式招生

## 2024年目标

| 指标 | 目标 |
|------|------|
| GDP 增速 | 6% 左右 |
| 规上工业增速 | 10.5% |
| 社消零售增速 | 9% |
| 城乡居民收入增速 | 3.5% / 8% 以上 |

### 八大任务

1. 抓产业兴工业：铝产业→1000亿+、中药材→400亿+、农业→1,600亿+（2024年数据）
2. 抓投资促消费：文蒙铁路开工、那兴高速通车、新增新能源装机120万千瓦+
3. 抓改革促开放：RCEP、中越合作、自贸区联动创新区
4. 抓科教促创新：文山学院迁建、铝产业技术研究院
5. 抓统筹重协调："强州府"、文砚同城、乡村振兴"百千万"
6. 抓生态优环境：碳达峰碳中和、绿美建设
7. 抓民生办实事：就业/保障/医疗/文体
8. 抓治理提效能：民族团结、边疆治理
