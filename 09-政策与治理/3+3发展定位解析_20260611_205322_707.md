# 文山州"3+3"发展定位解析

> **来源**：云南省"开局起步'十五五'"主题系列新闻发布会·文山专场（2026-06-03，云南海埂会堂）
> **发布人**：州长马忠俊及相关部门负责人

## 一、"3+3"完整表述

"3+3"定位是文山州"十五五"时期的核心发展纲目：

### 三大定位（国家/省级战略层面）

| 序号 | 定位 | 内涵 |
|:---:|------|------|
| 1 | **民族团结进步示范区** | 铸牢中华民族共同体意识，巩固边疆民族团结大局 |
| 2 | **石漠化地区生态文明建设排头兵** | 探索石漠化治理与绿色发展协同路径 |
| 3 | **云南面向东南开放先行区** | 发挥沿边区位优势，融入西部陆海新通道 |

### 三大目标（产业/民生落地层面）

| 序号 | 目标 | 内涵 |
|:---:|------|------|
| 1 | **新时代兴边富民示范区** | 推动边疆繁荣稳定、边民富裕 |
| 2 | **"绿色铝谷"核心区** | 打造全国绿色铝产能最大、产业链最完整的核心区 |
| 3 | **三七康养旅居目的地** | 以三七为核心的健康旅居品牌 |

## 二、四项重点

文山州将突出 **"产业、项目、干部、环境"** 四项重点，系统推进"十五五"发展：

- **产业**：实施"产业强州"战略，构建文山特色现代化产业集群
- **项目**：坚持"项目为王、产业为基"，集中推进标志性项目
- **干部**：解放思想、改革创新、奋发进取、真抓实干
- **环境**：优化营商环境，提升发展软实力

## 三、三大经济路径

以 **资源经济、园区经济、口岸经济** 为三大路径：

| 路径 | 核心逻辑 |
|------|----------|
| **资源经济** | 依托绿色水电、矿产、三七等优势资源，发展精深加工 |
| **园区经济** | 错位布局三大产业园区，集聚联动、提质增效 |
| **口岸经济** | 发挥天保、都龙、田蓬等国家级口岸优势，扩大沿边开放 |

## 四、四大重点产业及目标

### 1. 绿色铝——千亿龙头

| 指标 | 数据 |
|------|------|
| 2025年产值 | **突破 1000 亿元**（文山首个千亿级产业） |
| 电解铝产能 | **343 万吨**（占全省 52.8%、全国 7.7%） |
| 铝加工产能 | **418 万吨** |
| 2030年目标 | 综合产值突破 **1800 亿元** |
| 产业体系 | "1+1+3+N"（电解铝 + 再生铝 + 3 园区 + N 项目） |

### 2. 三七中药材——向千亿级迈进

| 指标 | 数据 |
|------|------|
| 2025年综合产值 | **220 亿元**（增长 7%） |
| 目标 | 向**千亿级产业**迈进 |
| 2028年目标 | 州内三七种植实现 **GAP 全覆盖** |
| 核心合作 | 云南白药、华润三九等头部企业 |
| 质控体系 | "七统一"：种子种苗、订单种植、规范种植、公开购销、产地加工、仓储物流、地标赋能 |
| 国家级荣誉 | 入选全国优势特色产业集群、全国中小企业特色产业集群 |

### 3. 高原特色现代农业——粤港澳大湾区"菜篮子"

| 指标 | 数据 |
|------|------|
| 定位 | 全国优质绿色现代农产品生产基地和绿色食品加工基地 |
| 方向 | 生态农业、设施农业、高效农业、共享农业 |
| 目标 | 一、二、三产深度融合 |

### 4. 康养旅游旅居——新名片

| 指标 | 数据 |
|------|------|
| 品牌 | "有一种叫云南的生活——三七之乡 康养文山""旅居云南·好在文山" |
| 2025年旅居人数 | **16.63 万人**，增长 **49.82%** |
| 四大板块 | 山水田园、康养旅居、边境风情、红色文化 |
