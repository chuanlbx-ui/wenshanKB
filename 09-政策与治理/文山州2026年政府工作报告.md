# 文山州2026年政府工作报告

> **报告人**：州长 马忠俊
> **时间**：2026年1月20日
> **场合**：文山州第十五届人民代表大会第六次会议
> **来源**：文山州人民政府

---

## 一、2025年工作回顾

### 1.1 综合经济

| 指标 | 2025年实际 | 增速 |
|------|-----------|------|
| 地区生产总值（GDP） | 1632.64 亿元 | **5.3%** |
| 规上工业增加值 | — | **12.9%** |
| 一般公共预算收入 | 74.12 亿元 | **6.8%** |
| 社会消费品零售总额 | 720.42 亿元 | 3.7% |
| 农村居民人均可支配收入 | 17,501 元 | **6.1%** |
| 城镇居民人均可支配收入 | 41,927 元 | — |

### 1.2 绿色铝产业 — 首个千亿产业

- **绿色铝产值突破 1000 亿元**，占全省半壁江山
- 电解铝产能 343 万吨，占全省 52.8%、全国 7.7%
- 合金化率提升至 69.92%

### 1.3 三七及中药材产业

- **三七综合产值 456 亿元**，同比增长 10.4%
- 加工产值占综合产值 26%
- 中药材种植面积稳步扩大

### 1.4 文旅消费

| 指标 | 2025年 | 增速 |
|------|--------|------|
| 接待游客 | 5,256.7 万人次 | 5.1% |
| 旅游花费 | 425.7 亿元 | 8.4% |

### 1.5 对外开放

- 外贸进出口总额 39 亿元，增长 38.8%

### 1.6 人口与城镇化

- 常住人口 335.4 万人
- 城镇化率 41.19%

---

## 二、2026年主要预期目标

| 指标 | 目标 |
|------|------|
| GDP 增速 | **5% 以上** |
| 规上工业增加值增速 | **10% 以上** |
| 绿色铝产值增速 | **20% 以上** |
| 三七中药材综合产值（2026年目标） | **500 亿元以上** |
| 固定资产投资增速 | 正增长 |
| 社会消费品零售总额增速 | 5% 左右 |
| 城乡居民收入增长 | 与经济增长同步 |

---

## 三、"十五五"远景目标

- **铝产业**：2030 年综合产值突破 **1,800 亿元**
- **三七及中药材**：冲刺**千亿级**产业
- **富宁港**：2026 年内建成
- **高速公路**：天麻高速、师丘高速 2026 年通车
- **铁路**：文蒙铁路建设中，文靖铁路开工建设

---

## 四、重点任务方向

1. 巩固绿色铝产能优势，延伸精深加工链条
2. 推进三七及中药材全产业链升级
3. 加快口岸经济建设，提升对外开放水平
4. 统筹城乡发展，稳步提高城镇化率
5. 强化民生保障，促进居民增收
6. 加快推进综合交通基础设施建设

## 五、2025年工作回顾（补充细节）

### 5.1 产业发展

- 马关、丘北、广南3个园区获批省级产业园区
- 锌、铟、钨、锑、硅等有色金属产业产值达 **140.62 亿元**
- 新增 A 级景区 7 个；仙人洞、六郎城入选首批"旅居云南"高质量发展试点
- 新增省级龙头企业 12 户，农林牧渔业总产值增长 2.6%

### 5.2 改革开放

- 政务服务网上可办率 **100%**，全程网办率 **91.4%**
- 招引省外产业项目 **140 个**、到位资金 **158.7 亿元**
- 外贸进出口总额 **39 亿元**，增长 **38.8%**

### 5.3 城乡发展

- 改造老旧小区 18 个，新建改造城市地下管网 200 公里
- 新建改造高标准农田 31.34 万亩，新改建农村公路 1,210 公里
- 那兴高速建成通车，文蒙铁路全线铺开
- 新增新能源装机 129.05 万千瓦，建成新型储能装机 49.1 万千瓦

### 5.4 生态建设

- 空气质量优良率 **97.5%**，27 个国省控断面水质优良 **100%**
- 省对州污染防治攻坚战成效考核**全省第一**
- 非化石能源消费占比 **71%**

### 5.5 社会民生

- 城镇新增就业 **2.2 万人**，农村劳动力转移就业 **127.6 万人**
- 高考特控上线率 **26.2%**；全国医改监测评价 **A+** 等级
- 集中整治殡葬领域收费乱象，取消、降低收费项目 224 个

### 5.6 安全稳定

- 5 个单位列为全省首批全国民族团结进步示范区示范单位培育对象
- 生产安全事故四项指标"四下降"
- 68 起成功避险避灾案例获应急管理部通报表扬
- 化解矛盾纠纷 4.13 万件，刑事、治安案件分别下降 10.8%、10.2%

## 七、"十五五""3+3"发展定位

| 定位 | 说明 |
|------|------|
| **民族团结进步示范区** | 铸牢中华民族共同体意识为主线 |
| **石漠化生态文明排头兵** | 筑牢滇东南生态安全屏障 |
| **东南开放先行区** | 沿边开放 + "通江达海" |
| **兴边富民示范区** | 边境地区"两增长三提升" |
| **"绿色铝谷"核心区** | 全国绿色铝产能最大州市 |
| **三七康养旅居目的地** | 千亿三七 + 全域旅居 |

同时打造六大目标：平安文山、康养文山、文明文山、美丽文山、幸福文山、清廉文山。

## 八、2026年重点工作（八项）

### 8.1 扩大有效需求
- 实施亿元以上重大项目 **166 个**
- 开工：文靖铁路、清水河水利枢纽、500kV 南屏变
- 建成：天麻高速、师丘高速、富宁港、百色至文山天然气管道
- 社会消费品零售总额增长 4%

### 8.2 产业强州战略
- 绿色铝产值增长 20%+；中铝绿材、信特项目 + 50 万吨再生铝投产
- 三七中药材综合产值目标 **500 亿元以上**；新建 GAP 基地 1.5 万亩
- 培育农业龙头企业 50 户、绿色优质农产品 70 个
- 文旅品牌："三七之乡 康养文山" + "旅居云南·好在文山"

### 8.3 科教兴州
- 新增高新技术企业 5 户、科技型中小企业 30 户
- 新增城区学位 8,800 个；推进免费学前教育
- 文山学院三七大健康产业学院、绿色铝产业学院建成省级特色学院

### 8.4 改革开放
- 探索农村集体经营性建设用地入市改革
- 天保口岸跨界交通工程、都龙口岸联检楼改扩建
- 力争引进亿元产业项目 50 个以上

### 8.5 城乡融合
- 城市更新 + "城管革命"巩固
- 乡村振兴：培养 100 名乡村 CEO、1,000 名乡村运营人才

### 8.6 美丽文山（生态）
- 推进"无废城市"建设
- 零碳园区、低碳社区创建
- 石漠化综合治理，山水林田湖草沙一体化保护

### 8.7 社会民生
- 职业技能培训 3 万人次以上
- 发放创业担保贷款 3.4 亿元以上
- "健康文山"：全民健康信息"一张网" + 边境医联体

### 8.8 边疆治理
- 新一轮全国民族团结进步示范州创建
- 新时代兴边富民行动
- 实施命案防控巩固提升三年行动
