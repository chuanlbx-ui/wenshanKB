# 文山州"十五五"规划纲要要点

> **来源**：2026年6月3日云南省"开局起步'十五五'"主题系列新闻发布会·文山专场（州长马忠俊主发布）

## 一、总体成就

- "十四五"GDP达1632.64亿元，年均增速高于全国全省
- 培育绿色铝首个千亿级产业：2025年产值 **1000 亿元**
- 文山三七入选全国优势特色产业集群和全国中小企业特色产业集群
- 创成全国民族团结进步示范州、全国双拥模范城
- 石漠化治理经验获评第五届全球最佳生态减贫案例
- 研发经费投入年均增长 **28.38%**
- 民营经济占 GDP 比重、产业投资占固投比重均超 **62%**

## 二、发展定位："3+3"

| 三大定位 | 三大特色定位 |
|----------|-------------|
| 民族团结进步示范区 | 新时代兴边富民示范区 |
| 石漠化地区生态文明建设排头兵 | "绿色铝谷"核心区 |
| 云南面向东南开放先行区 | 三七康养旅居目的地 |

## 三、四项重点："产业、项目、干部、环境"

## 四、四大重点产业

### 绿色铝产业

| 指标 | 数据 |
|------|------|
| 电解铝产能 | 343 万吨（占全省 52.8%、全国 7.7%） |
| 铝深加工产能 | 418 万吨 |
| 2025年产值 | 1016 亿元（首次突破千亿，锚点值1000亿） |
| 产业链 | "铝土矿—氧化铝—绿色铝—铝材精深加工—再生铝—综合利用" |
| 重大项目 | 中铝20万吨绿色低碳铸轧、砚山智铝二期、云南神火超薄铝箔、文山华疆再生铝循环利用 |
| 2026年1-4月招商引资 | 省外到位资金超51亿元 |
| 十五五目标 | 2030年目标综合产值突破 1800 亿元 |

### 三七中药材产业

| 指标 | 数据 |
|------|------|

### 高原特色现代农业

- 发展"四大农业"：生态农业、设施农业、高效农业、共享农业
- 建成粤港澳大湾区"菜篮子"基地
- "十五五"末总产值突破 **550 亿元**

## 五、对外开放

- 主动融入**西部陆海新通道**
- 富宁港年内建成，与广西平陆运河同步投用→云南出海最近"黄金港口"
- 文蒙铁路 2028 年通车、文靖铁路 2030 年通车
- 师丘高速（文山段）隧道全线贯通，总体进度 79%
- 3个国家级口岸优势，推动"口岸+园区+物流"联动
