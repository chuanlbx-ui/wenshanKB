# 边境治理（文山段）

> 438 公里中越边境线 · 3 个国家一类口岸 · 24 个边民互市点

---

## 一、边境概况

文山州与越南接壤的**麻栗坡、马关、富宁**三县，构成云南省对越开放的前沿阵地：

| 县 | 边境线长度 | 对应越方省份 | 核心口岸 |
|------|------------|------------|----------|
| **麻栗坡县** | 277 km | 河江省 | 天保口岸（国家一类） |
| **马关县** | 91 km | 河江省 | 都龙口岸（国家一类） |
| **富宁县** | 70 km | 河江省 | 田蓬口岸（国家一类） |

全州共有 **3 个国家一类口岸**、**24 个边民互市点**，占云南省对越口岸的 **50%**。

---

## 二、边境管理机制

### 2.1 立体化防控

| 维度 | 措施 |
|------|------|
| **人防** | 党政军警民"五位一体"合力强边固防机制 |
| **物防** | 边境铁丝网、巡逻道、检查站等物理隔离设施 |
| **技防** | 视频监控、无人机巡查、智能感知系统全覆盖 |
| **重点打击** | 电诈、偷渡、走私等跨境违法犯罪 |

### 2.2 基层治理创新

| 举措 | 内容 |
|------|------|
| "红旗飘飘"工程 | 边境村寨悬挂国旗，增强国家意识 |
| 边疆党建长廊 | 基层党组织引领边境治理 |
| "一户一明白人" | 每户培养一名懂政策、懂法律的明白人 |
| 涉外法律服务 | 边境一线设立涉外法律服务工作站 |

### 2.3 法治保障

- "**国门 + 村寨 + 集市**"立体普法格局
- 涉外法律咨询、纠纷调解
- 边民与企业的合法权益保障

## 三、跨境合作

文山州将越南作为对外工作的优先方向，与越南**宣光省、老街省、河江省**等建立常态化互访机制。

### 3.1 合作领域

| 领域 | 内容 |
|------|------|
| **经贸** | 签署数十个合作文本、缔结多对友好城市与村寨 |
| **交通** | 推动"中国天保—越南清水国际口岸跨界交通工程" |
| **文旅** | "文山·睦邻"品牌、中越人文交流周、边民大联欢 |
| **跨境旅游** | "中国麻栗坡—越南河江 2 日游"线路 |
| **卫生** | 跨境传染病联防联控 |
| **教育** | 留学生交流、职业教育合作 |

### 3.2 互联互通

| 项目 | 状态 |
|------|------|
| 云桂高铁 + 7 条高速公路 | 已通车 |
| 文蒙铁路 | 在建（预计 2028 年通车） |
| 富宁港 | 在建 |
| 中越跨境直达运输 | "一箱到底"已实现，24 小时通关 |

## 四、口岸经济

### 4.1 总体数据

| 指标 | 2023 年 | 2025 年 | 增速 |
|------|---------|---------|------|
| 口岸进出口货值 | 26.4 亿元 | **69.5 亿元** | +136.7% |
| 对越贸易额 | 24.1 亿元 | **29.1 亿元** | +20.7% |
| 外贸增速 | 全省第 2 | 全省第 2 | — |

### 4.2 天保口岸——核心引擎

位于麻栗坡县，与越南清水口岸相邻，是云南"1+3+4+N"重点口岸布局之一。

| 维度 | 详情 |
|------|------|
| 2025 年货值 | **70.1 亿元**，全省公路口岸第 4 位 |
| 核心品类 | 榴莲进口占比 68.85%，全国第三大陆路榴莲进口口岸 |
| 设施 | 国际货场、保税仓、进境指定监管场地 |
| 吞吐能力 | 日吞吐 1500 辆次 |
| 通关效率 | 非查验车辆最快 **6 秒**放行，车辆平均通关压缩 30% |

贸易格局——"**金色榴莲 + 银色铝品**"双轮驱动。

### 4.3 "落地加工"转型

从"通道经济"走向"产业经济"：
- 进口腰果、咖啡豆、榴莲等**本地加工**
- "边民互市 + 落地加工"模式
- 引进多家加工企业，产值数千万元
- 带动边民就近就业

## 五、边民互市

### 5.1 创新模式

| 模式 | 内容 |
|------|------|
| **+合作社** | "党支部 + 合作社 + 互助组 + 边民"模式 |
| **+互联网** | "边互通"APP 数字化申报 |
| **+落地加工** | 进口商品本地加工，以贸带工 |
| **+集体经济** | "532"利益分配模式（合作社收益部分扶持村集体和公益） |

### 5.2 政策保障

- 边民每人每日 **8000 元**进口免税额度
- 边民互市贸易"绿色通道"
- 特色金融产品：惠边卡、边民互市贷

### 5.3 富民成效

| 指标 | 数据 |
|------|------|
| 麻栗坡边民互市累计货值 | 超 **65 亿元** |
| 边民直接纯收入 | 约 **3200 万元** |
| 边民收入提升 | 参与互市 + 落地加工双重增收 |

## 六、从战场到市场

文山 438 公里边境线，40 年前是老山战区的烽火前线，如今是对外开放的最前沿。

| 对比维度 | 昨天（1984-1993） | 今天（2025） |
|----------|-------------------|-------------|
| 状态 | 战区、雷场 | 口岸、市场 |
| 人流 | 军人、民兵 | 边民、游客、商人 |
| 物流 | 弹药、补给 | 榴莲、电解铝 |
| 关键词 | 老山精神 | 口岸经济 |

天保口岸所在地，正是当年老山战区的核心地带。从"枪炮声"到"装卸声"，从"封锁线"到"通关道"，文山边境完成了历史性转身。

## 七、未来展望

1. **文蒙铁路通车后**（2028 年）：铁路直通河口口岸，形成"铁路+公路+水运"立体口岸体系
2. **天保—清水跨界交通工程**：中越已签署协定，建成后将大幅提升通关效率
3. **边境旅游试验区**：推动跨境旅游常态化、便利化
4. **产业升级**：从榴莲进口集散地向水果加工产业集群转型
