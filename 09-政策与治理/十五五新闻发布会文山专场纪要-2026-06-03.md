# 云南省"开局起步'十五五'"主题系列新闻发布会文山专场纪要

> **时间**：2026年6月3日 | **地点**：昆明海埂会堂
> **主发布人**：州委副书记、州长马忠俊
> **出席领导**：李永忠、陈治、李松涛、田燕
> **参会媒体**：人民日报、新华社等30余家媒体

## 一、"十四五"成就概要

- 地区生产总值**突破1600亿元**（锚点值1632.64亿元）
- 培育**绿色铝首个千亿级产业**
- 文山三七入选**全国优势特色产业集群**和**全国中小企业特色产业集群**
- 创成**全国民族团结进步示范州**、**全国双拥模范城**
- 石漠化治理经验获评**第五届全球最佳生态减贫案例**

## 二、"十五五"五大战略方向

马忠俊提出五个"坚定不移"：

| 方向 | 核心内容 |
|------|----------|
| 产业强州 | 以资源经济、园区经济、口岸经济为路径，壮大四大重点产业，构建特色现代化产业集群 |
| 高水平对外开放 | 融入西部陆海新通道，加快云桂沿边铁路建设，打造云南面向东南开放先行区 |
| 生态优先绿色发展 | 持续推进石漠化综合治理，加强普者黑湖和盘龙河"一湖一河"河湖保护，筑牢滇东南生态安全屏障 |
| 保障和改善民生 | 树立和践行正确政绩观学习教育，用心用情办好民生实事 |
| 新时代兴边富民 | 深化民族团结进步全域创建"十进"，推进"两增长三提升"三年行动，打造边境幸福村升级版 |

## 三、三大经济路径

| 路径 | 内涵 |
|------|------|
| 资源经济 | 以三七、绿色铝、高原特色农业等优势资源为基础 |
| 园区经济 | 砚山千亿级园区、文山马塘工业园区、富宁绿色铝创新产业园 |
| 口岸经济 | 天保、都龙、田蓬3个国家级口岸，"口岸+园区+物流"联动 |

## 四、"3+3"发展定位

| 三大定位 | 三大特色定位 |
|----------|-------------|
| 民族团结进步示范区 | 新时代兴边富民示范区 |
| 石漠化地区生态文明建设排头兵 | "绿色铝谷"核心区 |
| 云南面向东南开放先行区 | 三七康养旅居目的地 |

## 五、四大重点产业框架

- **绿色铝**：2025年产值1000亿元（首次突破千亿），锚点值1016亿；2030年目标1800亿
- **三七中药材**：2025年综合产值220亿元（同比+7%，口径：三七产业核心产值），锚点值456亿（广义口径）；向千亿级目标迈进
- **高原特色现代农业**：生态农业、设施农业、高效农业、共享农业"四大农业"
- **康养旅游旅居**："三七之乡 康养文山""旅居云南·好在文山"品牌

## 六、四项重点

> "产业、项目、干部、环境"——"十五五"工作主线

## 七、基础设施重大节点

| 项目 | 时间节点 | 说明 |
|------|----------|------|
| **富宁港** | **2026 年内建成** | 云南出海最近"黄金港口"，融入西部陆海新通道 |
| **文蒙铁路** | **2028 年建成通车** | 云桂沿边铁路关键段 |
| **文靖铁路** | **2030 年建成通车** | 云桂沿边铁路南段 |
| 曲靖至文山铁路 | 争取突破 | 规划中 |

## 八、民生保障与生态建设

- **财政支出**：80%以上财政支出用于民生
- **农村居民收入**：人均可支配收入增速高于全省平均水平
- **生态建设**：持续推进石漠化综合治理，加强普者黑湖、盘龙河"一湖一河"保护
- **全域绿化美化**：推动减污、降碳协同增效，发展方式绿色低碳转型

## 九、项目储备

- 谋划储备重点产业类项目 **2845 个**，总投资**接近万亿**
- 边境线长 **438 公里**，少数民族人口占 **57.5%**
- 深入实施新时代兴边富民行动，"两增长三提升"三年行动，边境幸福村升级版
