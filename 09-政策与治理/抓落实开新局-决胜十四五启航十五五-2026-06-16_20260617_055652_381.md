# 【抓落实 开新局】决胜"十四五" 启航"十五五" 谱写中国式现代化文山实践新篇章

> 发布日期：2026-06-16 | 来源：文山州政府办

## 一、"十四五"收官成就

"十四五"以来，文山各族干部群众牢记嘱托、感恩奋进，坚定不移沿着习近平总书记指引的方向前进：

- **GDP**：突破 **1632.64 亿元**
- **绿色铝**：培育了首个**千亿级产业**
- **三七**：入选全国优势特色产业集群和全国中小企业特色产业集群
- **民族团结**：创成全国民族团结进步示范州、全国双拥模范城
- **石漠化治理**：经验获评第五届全球最佳生态减贫案例
- 全州经济社会发展迈上新台阶，为"十五五"发展奠定了坚实基础

## 二、"十五五"总体框架

### 指导思想

以习近平新时代中国特色社会主义思想为指导，围绕"一个跨越""三个定位""开创发展新局面"总纲。

### "3＋3"发展定位

| 定位 | 说明 |
|------|------|
| 民族团结进步示范区 | 铸牢中华民族共同体意识为主线 |
| 石漠化地区生态文明建设排头兵 | 筑牢滇东南生态安全屏障 |
| 云南面向东南开放先行区 | 沿边开放 + 通江达海 |
| 新时代兴边富民示范区 | 边境"两增长三提升" |
| "绿色铝谷"核心区 | 全国绿色铝产能最大州市 |
| 三七康养旅居目的地 | 千亿三七 + 全域旅居 |

### 四项重点

突出**产业、项目、干部、环境**，解放思想、改革创新，确保与全国全省同步基本实现社会主义现代化取得决定性进展。

## 三、实施"产业强州"战略

以资源经济、园区经济、口岸经济为路径，持续发展壮大"四大重点产业"。

### 3.1 绿色铝产业（重中之重）

| 指标 | 数据 |
|------|------|
| 2025年电解铝产能 | 占全省 **52.8%**、全国 **7.7%** |
| 产业地位 | 全国"北铝南移"标志区 |
| 发展方向 | 向新材料、终端制造等价值链高端延伸 |
| 2030年目标 | 铝产业综合产值突破 **1800 亿元** |
| 定位 | 打造云南"绿色铝谷"核心区 |

充分发挥"绿电＋先进制造业"新优势。

### 3.2 三七中药材产业

- 三七是文山独具优势的生物医药资源，文山是三七的原产地、主产区
- 紧扣把中药材产业打造成为云南"第二个烟草产业"的要求
- 做大做强三七全产业链，全力以赴向**千亿级产业**目标迈进

### 3.3 高原特色现代农业

- 文山海拔相对高、纬度相对低、降雨相对多，北回归线贯穿全境，是农作物生长的"天然温床"
- 持续发展壮大生态农业、设施农业、高效农业、共享农业
- 推动一二三产业深度融合
- 建成全国优质绿色现代农产品生产基地和绿色食品加工基地、粤港澳大湾区"菜篮子"基地

### 3.4 康养旅游旅居产业

- 系统推进旅游产品、模式、业态创新和服务创优
- 接续打造"有一种叫云南的生活——三七之乡 康养文山""旅居云南·好在文山"品牌
- 让文山成为令人向往的诗和远方

## 四、推进高水平对外开放

文山是云南省东南大门，一头接粤港澳大湾区，一头接东南亚国家，是云南通往珠三角、粤港澳大湾区最近最便捷的通道。

### 交通重大项目时间表

| 项目 | 目标时间 |
|------|---------|
| **富宁港** | **2026年内建成**，与广西平陆运河同步投入使用 |
| **文蒙铁路** | **2028年建成通车** |
| **文靖铁路** | **2030年建成通车** |
| 曲靖至文山铁路 | 争取取得新突破 |

- 主动融入西部陆海新通道建设
- 打造云南出海最近的"黄金港口"
- 服务融入构建具有战略意义的中越命运共同体建设
- 充分发挥 3 个国家级口岸优势
- 一体推进口岸功能提升、口岸经济发展、口岸城市建设
- **以大通道带动大物流、促进大产业、推进大发展**

## 五、生态优先、绿色发展

| 指标 | 数据 |
|------|------|
| 空气优良率 | **97.5%** |
| 国控省控地表水监测断面水质优良率 | **100%** |

"十四五"生态文明荣誉：

- 马关县、麻栗坡县创成**全国生态文明建设示范区**
- 西畴县被命名为**"绿水青山就是金山银山"实践创新基地**

"十五五"方向：

- 树牢"绿水青山就是金山银山"理念
- 持续推进石漠化综合治理
- 加强以普者黑湖、盘龙河"一湖一河"为重点的河湖保护
- 推进全域绿化美化
- 推动减污、降碳协同增效
- 加快发展方式绿色低碳转型

## 六、民生福祉

- "十四五"期间，**80% 以上**财政支出用于民生
- 认真答好群众持续增收"五问"
- 农村居民人均可支配收入增速高于全省平均水平
- 牢牢守住了不发生规模性返贫致贫的底线
- 以更大力度"投资于人"
- 让各族群众有更稳定的工作、更满意的收入、更优质的教育医疗、更可靠的社会保障
