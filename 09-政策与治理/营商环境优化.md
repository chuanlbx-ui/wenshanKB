# 营商环境优化

> 文山州放管服改革、招商引资政策与企业服务体系全貌。

---

## 一、总体目标

- 到 2026 年，营商环境整体水平进入**云南省第一方阵**，部分指标达到省内一流
- 企业满意度持续提升，形成具有文山特色的"**文山效率**""**文山服务**"品牌
- 主动融入粤港澳大湾区、成渝双城经济圈，提升区域开放合作水平

---

## 二、放管服改革

### 简政放权（"放"）

| 举措 | 内容 |
|------|------|
| 行政审批精简 | 持续取消、下放和调整行政审批事项，赋予县级和园区更多自主权 |
| "证照分离" | 深化"证照分离"改革，推进"一业一证""一照多址" |
| 极简审批 | 推广"容缺受理""告知承诺"，推进"互联网+政务服务" |
| 基层延伸 | 政务服务事项向基层延伸，实现"就近能办、多点可办" |

### 放管结合（"管"）

| 举措 | 内容 |
|------|------|
| "双随机、一公开" | 监管常态化、全覆盖，推进部门联合抽查 |
| 信用监管 | 完善信用记录和信用承诺制度，实施分级分类监管 |
| 包容审慎 | 对新业态实施包容审慎监管 |
| "综合查一次" | 推行涉企行政执法检查统筹合并，避免多头执法 |

### 优化服务（"服"）

| 举措 | 内容 |
|------|------|
| "一站式"服务 | 提升政务服务大厅功能，推广"一窗受理、集成服务" |
| 智能化 | 利用大数据、AI 提升政务服务智能化、精准化水平 |
| 无障碍 | 关注老年人、残疾人等特殊群体，保留并优化线下服务渠道 |

---

## 三、招商引资政策

### 产业招商方向

| 重点产业 | 招商重点 |
|----------|----------|
| **绿色铝** | 精深加工、汽车零部件、建筑铝材、终端消费品 |
| **稀有金属新材料** | 铟锗等战略金属开发利用 |
| **生物医药** | 三七 GAP 种植、医药制造、大健康 |
| **现代农业** | 三七、辣椒、八角等特色优势产业深加工 |
| **文旅康养** | 普者黑、坝美、老山景区及康养项目 |
| **现代物流** | 依托口岸和交通枢纽发展跨境物流 |

### 政策工具箱

- 制定和完善招商引资优惠政策，覆盖土地、税收、投融资、人才引进等领域
- 重大项目实行"**一企一策**""**一事一议**"定制化支持
- 保障项目用地、用能、用工等要素需求
- 拓宽融资渠道，鼓励金融机构加大对招商项目的支持

### 招商方式创新

- 线上招商、云招商（利用数字经济手段）
- 长三角、珠三角、京津冀、成渝等重点区域对接
- 产业链招商、以商招商、精准招商
- 依托商会、协会、驻外办事处

---

## 四、企业服务体系

### 全生命周期服务

- 覆盖企业**准入—生产—经营—退出**全生命周期
- 企业诉求快速响应、高效办结

### 政企沟通机制

| 机制 | 内容 |
|------|------|
| 企业服务平台 | 完善"文山州中小企业公共服务平台"，整合政策发布、诉求受理、融资对接、人才培训 |
| 企业服务专员 | 推广"企业服务专员"制度，一个窗口对外 |
| 政企座谈 | 定期召开政企座谈会、早餐会、恳谈会 |
| 领导挂钩 | 领导干部挂钩联系重点企业制度 |

### 政策落地保障

- 加大惠企政策宣传解读力度
- 推行"**免申即享**""**快申快享**"，确保政策红利直达快享
- 建立政策落实督查机制，防止政策"空转"
- 严格规范涉企收费，清理拖欠企业账款

---

## 五、法治环境建设

- 严格规范行政执法，维护公平竞争秩序
- 加强知识产权保护
- 完善矛盾纠纷多元化解机制
- 建立健全企业投诉举报处理机制
- 构建"**亲**""**清**"政商关系
