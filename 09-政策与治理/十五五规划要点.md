# 文山州"十五五"规划要点

> 正式名称：《文山壮族苗族自治州国民经济和社会发展第十五个五年规划纲要》，2026年1月22日州第十五届人大六次会议审议通过，2026年4月10日由州人民政府印发实施。

---

## 一、"3+3"发展定位

| 三大定位 | 说明 |
|----------|------|
| 民族团结进步示范区 | 铸牢中华民族共同体意识 |
| 石漠化地区生态文明建设排头兵 | 生态修复与绿色发展 |
| 云南面向东南开放先行区 | 向东融入粤港澳大湾区、向南辐射南亚东南亚 |

| 三大特色定位 | 说明 |
|--------------|------|
| 新时代兴边富民示范区 | 边境幸福村、边民增收 |
| "绿色铝谷"核心区 | 全国铝产业标杆 |
| 三七康养旅居目的地 | 康养旅游旅居品牌 |

> 突出"**产业、项目、干部、环境**"四项重点，解放思想、改革创新、奋发进取、真抓实干。

## 四、四大重点产业布局

### 绿色铝
- **目标**：2030年铝产业综合产值突破 **1,800 亿元**
- **体系**："1+1+3+N"（电解铝 + 再生铝 + 3个园区 + N个项目）
- **方向**：高端化、终端化、集群化、绿色化
- 铝水 100% 就地转化、铝材 100% 精深加工
- 再生铝产能 2030 年达 150 万吨

### 三七中药材
- **目标**：向千亿级产业迈进
- 2028 年实现州内三七种植 GAP 全覆盖
- 2030 年基本实现良种全覆盖

### 高原特色现代农业
- **目标**：总产值超 **550 亿元**（十五五末）
- **体系**："1+4+N"（粮食 + 三七/蔬菜/水果/肉牛 + N个特色产业）
- 粤港澳大湾区"菜篮子"基地

### 康养旅游旅居
- **品牌**："旅居云南·好在文山" / "三七之乡·康养文山"
- 构建丘北、广南、麻栗坡"旅游黄金三角"

## 五、其他关键领域

| 领域 | 重点方向 |
|------|----------|
| 基础设施 | 文蒙铁路、富宁港、清洁能源基地、新型储能 |
| 新型城镇化 | 2030年城镇化率目标约45%、农业转移人口市民化 |
| 生态文明 | 石漠化治理排头兵、森林覆盖率 45% |
| 开放合作 | 面向东南开放先行区、口岸经济 |
| 民生福祉 | 居民收入与经济增长同步、人均预期寿命 75.22 岁 |
| 安全稳定 | 粮食产能 182 万吨、能源综合产能 750 万吨标准煤 |

## 六、2026年"项目攻坚年"

- 聚焦绿色铝、三七中药材、现代农业、康养旅游旅居四大重点产业
- 同步布局新能源、新材料、数字经济等新兴产业
- 集中推进一批投资大、链条长、带动强的标志性项目
- 开工一批、建设一批、投产一批、储备一批良性循环
