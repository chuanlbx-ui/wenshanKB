# 守正创新践嘱托 石漠蝶变绘新卷

> **来源**：文山州林业和草原局
> **日期**：2026年4月10日
> **主题**：统筹推进自然保护地体系建设与石漠化综合治理，探索生态保护、治理增效、民生改善相得益彰的高质量发展路径

## 一、自然保护地整合优化

| 指标 | 整合前 | 整合后 |
|------|--------|--------|
| 数量 | 11 个（交叉重叠） | **15 个** |
| 总面积 | — | **12.81 万公顷** |
| 自然保护区 | — | 3 个，3.67 万公顷 |
| 自然公园 | — | 13 个（含 1 个新设立），9.14 万公顷 |

- 优化转型 4 个，新设立 1 个
- 启动 3 个自然保护区、5 个风景名胜区总体规划编制（修编）
- 累计投入 **3,344.87 万元**实施能力提升、基础设施项目 **23 个**
- 联合多部门建立常态化排查整治机制

## 二、物种保护

| 地点 | 保护对象 |
|------|---------|
| 马关古林箐 | 蜂猴、倭蜂猴、望天树等 |
| 麻栗坡老山 | 滇桐等 5 种珍稀濒危植物繁育回归 |
| 富宁驮娘江 | 叉孢苏铁繁育回归 |
| 广南八宝 | 管护站建设 |
| 丘北普者黑 | 越冬水鸟监测 |

## 三、石漠化综合治理项目

| 指标 | 数据 |
|------|------|
| 项目名称 | 文山州滇东南山地石漠化综合治理项目（2023-2025年） |
| 总投资 | **4.8 亿元** |
| 生态修复面积 | **71 万亩** |
| 管理模式 | EPC 总承包 + 项目法人/招投标/监理/合同"四制" |

### 完工明细

| 工程类型 | 完成量 |
|----------|--------|
| 人工造乔木林 | **20.83 万亩** |
| 退化林修复 | **28.2 万亩** |
| 封山育林 | **22 万亩** |
| 草原改良 | **0.6 万亩** |
| 小型水利水保设施 | **1,280 处** |

### 产业融合

- 油茶、八角、木姜子、澳洲坚果等经济树种 **12.94 万亩**纳入建设
- "生态治理+产业发展"模式

## 四、以工代赈

| 指标 | 数据 |
|------|------|
| 培训群众 | **14,959 人次** |
| 发放劳务报酬 | **8,345.56 万元** |
| 人均增收 | 约 **5,600 元** |

- 全环节嵌入"以工代赈"政策，"项目建设+技能培训+劳务报酬"模式
- 群众从"旁观者"转变为"建设者""受益者"

- 石漠化治理经验获评 **第五届全球最佳生态减贫案例**（"六子登科"综合治理模式）
- 为全球生态治理与减贫事业贡献"文山样本"
