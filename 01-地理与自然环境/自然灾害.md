# 自然灾害

## 总体特征

文山州地处云南高原与广西丘陵过渡带，地质灾害高中易发区占全州面积的 **68.1%**。主要灾害：**地震、泥石流/滑坡、旱涝**。"十年九旱、暴雨成灾"、"旱涝急转"现象突出。

## 隐患规模

| 指标 | 数据 |
|------|------|
| 地质灾害隐患点 | **1,210 个**（截至 2020 年底） |
| 威胁人口 | **7.9 万余人** |
| 威胁财产 | **24 亿余元** |
| 泥石流隐患点 | **71 处** |
| 滑坡隐患点 | **860 处** |

## 地震灾害

文山整体属**少震、弱震区**，无特大破坏性地震记录。2021 年 9 月 25 日文山市发生 2.9 级地震。

| 监测设施 | 数量 |
|----------|------|
| 基准站 | **3 个** |
| 基本站 | **11 个** |
| 一般站 | **98 个** |
| 预警终端 | **28 套** |

**"十三五"期间**（2016-2020）：地质灾害 48 起，1 人死亡，直接损失 0.11 亿元。成功避险 15 起，避免 383 人伤亡。

**2025 年 9 月**：文山市举行模拟 5.6 级地震综合应急演练，检验"123 快速响应机制"。

**主要诱因**：强降雨、台风、地震及人类工程活动（切坡建房）。

## 综合防控体系

### "1262"精细化预报与响应联动
根据 12/6/2 小时强降雨预报，分级落实巡查、转移、抢险措施。

### "15445"工作机制

| 要素 | 内容 |
|------|------|
| 1 个目标 | "不死人、少损失" |
| 5 情 | 雨情、水情、汛情、险情、灾情 |
| 5 查 | 群众自查、干部帮查、行管巡查、督查检查、专业核查 |
| 4 停 | 停课、停工、停产、停运、停业 |
| 5 级联动 | 县、乡、村、组四级联防 |

### 临灾转移"四个一律"
雨量达阈值 / 险情异动 / 风险难判 / 夜间难判 → **一律转移**。

### 基础设施保障
- 全州 **432 座水库**、**1,059 个**山洪灾害危险区落实防汛责任人
- 农村自来水普及率 **88.58%**
- 实施 **57 件**抗旱保供水工程
