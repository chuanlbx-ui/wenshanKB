# 石漠化治理

## 石漠化概况

文山州是滇黔桂石漠化集中连片特困地区的核心区域。

| 指标 | 数据 |
|------|------|
| 岩溶地貌占全州面积 | **53.4%** |
| 石漠化曾占岩溶面积 | **71.1%** |
| 严重程度 | 全国石漠化最严重地区之一 |

## 治理成效

| 层面 | 治理规模 | 成效 |
|------|----------|------|
| **全州** | 累计治理约 **255 万亩**（约 1,700 km²，十八大以来） | 石漠化土地净减少 **22.68 万公顷**（约 340 万亩，与 2016 年相比） |
| **全州** | — | 森林覆盖率从 37.6% 提高到 **50% 以上** |
| **西畴县** | 累计治理 **252 km²** | 森林覆盖率从 25% 左右提高到约 **60%** |
| **西畴县** | — | 人均耕地从不足 0.8 亩提高到约 **2 亩** |

## "六子登科"综合治理模式

文山州在实践中探索出的石漠化综合治理模式，入选**全球减贫最佳案例**。

| 模式 | 具体内容 |
|------|----------|
| **🏔️ 山顶戴帽子** | 封山育林、植树造林，恢复山顶森林植被，涵养水源 |
| **⛰️ 山腰系带子** | 在山坡发展核桃、油茶等特色经济林，兼顾生态与增收 |
| **🏞️ 山脚搭台子** | 通过"坡改梯"、炸石垒埂，建设"保土、保水、保肥"的"三保"台地 |
| **🌾 平地铺毯子** | 在条件较好的平地建设高稳产农田，提高土地产出 |
| **💧 入户建池子** | 建设水窖、沼气池等，解决农户饮水、能源和肥料问题 |
| **🏡 村庄移位子** | 对生存条件极差的村寨实施易地搬迁，减轻生态压力 |

## 生态与经济协同成效

### 生态环境改善
- 西畴县空气质量优良率 **100%**
- 国省控地表水断面水质优良率 **100%**
- 林草覆盖率 **66.66%**，获评"国家水土保持示范县"
- 农村生活污水治理率从 25.71% 提升至 **86.11%**

### 产业发展
- **"八林经济"**：林木、林果、林花、林药、林禽、林菌、林游、林草
- 高原生态农业总产值 **27 亿元**，林草总产值 **8.7 亿元**
- 三光片区猕猴桃产业：预计 2025 年产值突破 **2,400 万元**

### 生态旅游
- "三光片区"从"三光村"蝶变为**国家 AAAA 级景区**
- 2023 年西畴县接待游客 **190 万人次**，旅游综合收入 **20 亿元**
- "十四五"期间全县接待游客超 **900 万人次**，旅游总收入超 **90 亿元**

### 生态价值量化
- 西畴县生态产品总值（GEP）核算为 **249.03 亿元**
- 实现了生态价值的可量化评估
