# 生物多样性

## 总体特征

文山州地处北回归线南侧，核心保护区——云南文山国家级自然保护区（老君山片区）是滇东南生物多样性最集中的区域，被誉为 **"北回归线绿洲"** 和 **"天然物种基因库"**。

## 云南文山国家级自然保护区

| 项目 | 数据 |
|------|------|
| 级别 | **国家级**（2003 年晋升） |
| 片区 | 老君山 + 小桥沟 |
| 主峰 | 薄竹山，海拔 **2,991.2 m**（滇东南第一高峰） |
| 生态功能 | 盘龙河、那么果河源头，年涵养水量超 **1 亿 m³** |
| 保护对象 | 岩溶中山南亚热带常绿阔叶林生态系统 |

## 保护成效

- 2023 年列入国家 **"兽类及鸟类重要栖息地"**
- 两个"老君山"保护区体系：文山国家级（老君山片区）+ 老君山省级（马关、麻栗坡）
- 持续发现格彩臂金龟（国家 Ⅱ 级昆虫）、长喙兰（濒危植物）等新记录
