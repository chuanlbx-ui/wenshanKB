# 广南县深度档案（2025年）

## 基本信息

| 项目 | 数据 |
|------|------|
| 面积 | 7,983 km²（全州最大） |
| 常住人口 | 335.4 万人 |
| 政府驻地 | 莲城镇 |
| 下辖 | 7 镇、11 乡 |
| 气候 | 中亚热带湿润气候 |
| 区位 | 文山州东北部，东邻广西百色市 |

## 文旅资源

### 坝美景区（世外桃源）
以"林尽水源，便得一山"的天然溶洞通道为特色，乘船穿洞而入，被誉为"最后的世外桃源"。

| 指标 | 2025年数据 |
|------|-----------|
| 全年游客 | **8.94 万人次** |
| 营业收入 | **1,245.50 万元** |
| 景区提质投资 | 1,131 万元 |
| 年旅居者 | 约 7,000 人，旅居收入 840 万元 |

**节假日**：春节接待 8,076 人次（收入 121.16 万元），五一接待 5,472 人次（票务收入 90.65 万元）。

### 其他文旅
- **广南八宝**：壮族"三月三"歌圩，国家级非遗。
- **句町古国遗址**：壮族历史文化遗址。
- **侬氏土司衙署**：保存完好的土司衙署建筑群。

## 经济特色

### 坝美镇："2+N"产业体系

**两大主导产业**：

| 产业 | 2025年数据 |
|------|-----------|
| 🍊 柑橘 | 种植 2.15 万亩，投产 1.43 万亩，总产量 4.28 万吨，户均年收入 10.3 万元 |
| 🍵 茶叶 | 种植 6 万余亩（可采 5.6 万余亩），预计产值超 1,787 万元 |

**多元产业**：甘蔗、百香果、贵族南瓜（226 亩，产值 112.5 万元）、铁皮石斛（61.2 万丛）、肉牛（补助 133.69 万元，惠及 467 户）。

### 绿色工业
- 生物质颗粒生产线 4 条，投资 1,400 万元，利用农林废弃物生产生物质颗粒，带动增收超 300 万元。

## 交通

- **铁路**：云桂铁路设**广南县站**，动车直达昆明、南宁。
- **公路**：G80 广昆高速贯穿全境，G5615 天猴高速连接丘北。
- **水路**：驮娘江可通航（小吨位）。
