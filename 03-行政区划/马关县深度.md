# 马关县深度档案（2025年）

## 基本信息

| 项目 | 数据 |
|------|------|
| 面积 | 2,666 km² |
| 常住人口 | 335.4 万人 |
| 政府驻地 | 马白镇 |
| 下辖 | 4 镇、9 乡 |
| 边境线 | 约 138 公里 |
| 区位 | 文山州南部，与越南河江省接壤 |

## 历史沿革

清雍正八年（1730年）设马白同知。清嘉庆二十五年（1820年）设安平厅。民国 3 年（1914年）改称马关县。

## 边境口岸

**都龙口岸**：中越边境国家级口岸，2025年核心数据：

| 指标 | 数据 |
|------|------|
| 园区工业总产值 | **98.30 亿元** |
| 进出口贸易额 (园区口径) | 400.59 万元 |
| 在园企业 | 32 家（带动就业 5,000+ 人） |
| 边民互市贸易量 | 110.45 吨（截至 11 月） |
| 进出境人员 (目标) | 超 25 万人 |
| 进出口货值 (目标) | 超 2 亿元 |

推动 **"边民互市 + 落地加工"** 产业链，2025 年 11 月首次进口越南芒果干、菠萝蜜脯等水果制品。

## 经济特色

### 产业园区
云南马关产业园区（一园三片）主导产业：
- **有色金属精深加工**（核心）
- **中药材**：砂仁、草果等 87 种，总面积超 **42 万亩**，砂仁产值 **5.39 亿元**，草果产值 **1.02 亿元**
- **农特产品加工**

### 特色农业

| 产业 | 关键数据 (2025年) |
|------|-------------------|
| 林草产业 | 总产值 **35.4 亿元**，林下经济产值 19.08 亿元 |
| 百香果 | 约 7,000 亩，带动 600 余户人均增收约 1 万元 |
| 茶叶 | 都龙镇龙井茶约 1,000 亩，产值突破 900 万元 |
| 中药材 | 42 万亩，砂仁 5.39 亿元，草果 1.02 亿元 |

马关县于 2026 年入选云南省林下经济重点县。

### 边境就业
- 都龙镇茅坪社区引进数据线加工企业，日均产量超 2 万条，年产值预估 **450 万元**，带动 500 余人就业。
- 沿边行政村人均可支配收入 **20,346 元**，同比增长 1,281 元。

## 文旅资源

- **花山节**：民歌对唱、服装走秀、芦笙舞、爬花杆，苗族传统节庆。
- **边境风情**：G219 国道穿境而过，沿线边关风光独特。
- **民族村寨**：壮族、苗族、瑶族等多民族聚居，民俗文化丰富。

## 交通

- **公路**：G219 国道纵贯全境，马兴高速连接文山市区。
- **口岸通道**：都龙口岸通关效率持续提升，连接越南河江省。
- **铁路**：距云桂铁路普者黑站约 140 公里。
