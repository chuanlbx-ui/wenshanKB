# 中医药康旅"文山模式"

> **来源**：搜狐 / 人民日报客户端 | **发布时间**：2026-06-12

---

## 一、核心框架：「1+1+10」模式

文山州构建 **"1+1+10"** 中医药康旅模式，从 **十个维度** 系统推进：

| 维度 | 内容 |
|------|------|
| **医** | 中医药诊疗服务体系 |
| **康** | 康复理疗 |
| **养** | 养生保健 |
| **健** | 健康管理 |
| **食** | 药膳美食 |
| **药** | 三七等道地药材 |
| **游** | 康养旅游线路 |
| **赏** | 中医药文化展示 |
| **学** | 研学教育 |
| **购** | 中医药产品消费 |

---

## 二、典型案例

### 2.1 砚山康养中心

- 模式：「**中医 + 康养 + 旅居**」
- 荣誉：入选 **"旅居云南"典型案例**

### 2.2 西畴"医共体+森林康养"

- 模式：「**医共体 + 森林康养**」睡眠新模式
- 荣誉：入选 **全国医共体建设优秀成果**
