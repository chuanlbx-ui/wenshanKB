# 柳井乡以工代赈与冷链仓储项目

> 来源：云南网 | 发布时间：2026-06-08
> 通讯员：沈静娴
> 地点：文山市柳井乡

## 一、项目概况

2026年6月，文山市柳井乡两个百万级项目先后开工，构建"基础设施兜底 + 产业配套赋能 + 集体经济壮大 + 群众稳定增收"的发展链条，实现从被动"输血"到主动"造血"的重大转机。

| 项目 | 类型 | 投资模式 | 投资额 |
|------|------|----------|--------|
| 通村路硬化 | 以工代赈 | 村民自建 | **349 万元** |
| 冷链仓储厂房 | 东西部协作 | 集体经济 | **544 万元** |

## 二、以工代赈通村路

| 指标 | 数据 |
|------|------|
| 全长 | **8.56 公里** |
| 覆盖自然村 | 马边塘、彭家凹塘、团田、麦子地、下打铁寨（4个） |
| 投资 | **349 万元** |
| 模式 | **"村民自建"** |
| 预计完工 | **2026年9月** |
| 带动务工 | **80 人** |
| 助农增收 | **145 万元** |

### 运作机制

- 采取"村民自建"模式，成立以工代赈项目理事会
- 用工安排、材料采购、资金管理、质量验收由村民民主推选小组负责
- 建成后货车可直达田间地头，农资运输成本大幅降低
- 彻底改变"晴天一身灰，雨天一身泥"的出行困境

## 三、冷链仓储厂房

| 指标 | 数据 |
|------|------|
| 仓储厂房 | **2750 平方米** |
| 冷库容量 | **1000 立方米** |
| 投资 | **544 万元**（全部来自东西部协作资金） |
| 预计完工 | **2026年11月** |
| 配套 | 标准化分拣场地、给排水系统、电力设施、雨水收集池、污水处理池 |

### 经济效益

| 指标 | 数据 |
|------|------|
| 村集体年租金收入 | **20 万元以上** |
| 长期就业 | **10 人以上** |
| 临时用工 | **30 人以上** |
| 年提供务工岗位 | 超 **50 人次** |
| 间接带动就业 | 约 **200 个**岗位 |
| 辐射周边农贸交易 | **3000 余人次** |

### 产业价值

- 破解农产品产后损耗率高、好产品卖不出好价钱的核心痛点
- 冷链加持下新鲜果蔬保鲜期大幅延长，可实现错峰上市、待价而沽
- 作为村级经营性固定资产对外出租，持续"造血"
- 补齐"产、运、储、销"链条，念好农产品出山和农产品增值两篇致富经

> "通过实施2个项目，将进一步补齐柳井乡基础设施短板，有力促进产业发展、带动群众增收，推动壮大村集体经济，念好全乡农产品出山和农产品增值两篇致富经。"
> —— 柳井乡党委书记 彭帅
