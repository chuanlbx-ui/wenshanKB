# 文山州教育发展深度

## 一、教育概况

文山州教育体系覆盖学前教育、基础教育、职业教育、高等教育全链条。受历史、地理和经济条件制约，文山教育基础相对薄弱，但近年来在国家和省级政策支持下快速发展。

| 指标 | 2025 年数据 | 来源 |
|------|-------------|------|
| 各级各类学校 | 2,211 所 | 文山州统计公报 |
| 在校学生 | 约 85 万人 | 文山州教育体育局 |
| 专任教师 | 约 5.2 万人 | 文山州统计公报 |
| 教育支出 | 91.80 亿元（占一般公共预算支出 24.0%） | 文山州统计公报 |
| 高中阶段毛入学率 | 90.42% | 文山州教育体育局 |

## 三、高等教育

### 3.1 文山学院

文山学院是文山州**唯一一所本科高校**，是文山高等教育的核心载体。

| 项目 | 内容 |
|------|------|
| 成立时间 | 2009 年（升本） |
| 校区 | 文山市区主校区 + 在建新校区 |
| 在校生 | 约 1.2 万人 |
| 本科专业 | 约 40 个 |
| 重点学科 | 三七医药、民族学、教师教育 |
| 特色定位 | 服务地方产业（三七、旅游、民族教育） |

## 四、教育公平与帮扶

### 4.1 学生资助体系

| 资助项目 | 覆盖学段 | 标准 |
|----------|----------|------|
| 学前教育资助 | 幼儿园 | 约 300-500 元/年 |
| 义务教育"两免一补" | 小学+初中 | 免学杂费+寄宿生生活补助 |
| 普通高中国家助学金 | 高中 | 1,000-3,000 元/年 |
| 中职国家助学金 | 中职 | 2,000 元/年 |
| 大学生生源地贷款 | 高校 | 最高 16,000 元/年 |

### 4.2 东西协作与教育帮扶

| 项目 | 内容 |
|------|------|
| 上海对口帮扶 | 教师培训、学校建设、学生资助 |
| 省内外名校帮扶 | 云南师大附中帮扶文山州一中 |
| "组团式"教育帮扶 | 2022 年起中组部牵头，选派优秀校长教师入文山 |

## 五、教育发展挑战

| 挑战 | 具体表现 |
|------|----------|
| 教育质量差距 | 高考一本率、本科上线率低于全省平均 |
| 师资结构性短缺 | 英语、音体美、信息技术等学科教师不足 |
| 山区就学不便 | 教学点分散，寄宿制学校管理压力大 |
| 职业教育吸引力弱 | 社会对职业教育认可度不高，招生困难 |

## 六、教育发展规划与政策方向

### 6.1 "十五五"教育发展目标

| 目标 | 指标 |
|------|------|
| 学前教育毛入园率 | ≥ 92% |
| 九年义务教育巩固率 | ≥ 97% |
| 高中阶段毛入学率 | ≥ 93% |
| 高等教育毛入学率 | 持续提升 |

### 6.2 重点工程

1. **基础教育质量提升工程**：加强教师培训、推进集团化办学
2. **职业教育提质培优**：建设产教融合实训基地，服务绿色铝、三七等产业
3. **高等教育内涵发展**：支持文山学院硕士点建设，提升科研服务能力
4. **教育数字化**：推进"三个课堂"（专递课堂、名师课堂、名校网络课堂）全覆盖
5. **边境地区教育保障**：改善边境学校办学条件，留住优秀教师

## 七、教育服务地方产业

| 产业 | 教育支撑 |
|------|----------|
| 三七产业 | 文山学院三七医药学院、州内中职学校中药材种植与加工专业 |
| 绿色铝产业 | 文山职业技术学院机电、材料工程专业 |
| 文旅产业 | 州内中职、高职旅游服务与管理专业 |
| 现代农业 | 涉农中职学校、农广校培训体系 |
