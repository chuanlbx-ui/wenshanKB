# 庄子田村仙人掌产业：五千亩绿刺"刺"破石漠化困局

> **来源**：中国新闻网 | **发布时间**：2026-04-07
> **核心**：文山市庄子田村通过发展仙人掌种植及深加工产业治理石漠化，五年间从不足700亩扩至5000余亩，综合年产值突破1500万元，成为中国仙人掌种植面积最大、产值最高的村子；2025年实现全国首批可食仙人掌叶片出口越南。

## 一、背景与困境

- **地理位置**：文山市新平街道庄子田村
- **自然条件**：石漠化面积超八成，人均耕地不足1.9亩
- **历史面貌**：曾流传"好女不嫁庄子田"，属典型石漠化贫困村

## 二、产业起步与模式

### 归雁行动引领

- **时间节点**：2021年，政府落实"归雁行动"，回引本村能人返乡
- **关键人物**：鲁春红（"80后"，在外经商近20年），牵头调研论证，确定耐旱耐瘠的仙人掌作为主导产业
- **组织模式**：成立合作社，全村106户土地全部入股，人人既是股东也是员工
- **初期策略**：种苗来自鲁春红多年积累资源，肥料用农家猪粪牛粪替代，销路靠自己一步步跑出

## 三、产业成果

| 指标 | 数值 |
|------|------|
| 种植面积 | 从不足700亩 → **5,000 余亩**（五年） |
| 综合年产值 | **突破 1,500 万元** |
| 人均年增收 | **3.2 万元** |
| 全国地位 | 中国仙人掌种植面积最大、产值最高的村子 |

## 四、深加工与科研合作

鲁春红先后与 **中国农业大学**、**华东理工大学** 等科研团队合作，开发十余款深加工产品：

- 食品类：果干、酵素饮品、精酿啤酒
- 日化类：护手霜等

## 五、出口突破

- **2023年起**：越南考察团多次到村对接合作，韩国商人亦来洽谈采购
- **2025年8月**：20吨可食仙人掌叶片经海关监管发往越南，成为 **全国首批出口的可食仙人掌**

|------|------|-----------|

### 配套业态

- 已开业：**5 家农家乐**、**2 家民宿**
- 规划中：研学基地
- 返乡创业案例：李加文（原昆明厨师）返村后将自家小院改造为集餐饮、休闲于一体的院落，研发仙人掌特色凉菜

## 七、关键启示

1. **因地制宜选产业**：石漠化地区选择耐旱耐瘠的仙人掌，化生态劣势为产业优势
2. **能人返乡带动**：归雁行动精准回引有经验的本村能人，解决资金、技术、销路等起步痛点
3. **全产业链延伸**：从单一种植扩展到深加工（食品+日化）+出口贸易+文旅融合
4. **利益共享机制**：全员入股合作社模式确保村民共享发展红利
