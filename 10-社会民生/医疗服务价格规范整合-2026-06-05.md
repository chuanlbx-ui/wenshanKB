# 文山州规范综合诊查类等医疗服务项目价格及医保支付政策

## 政策背景

2026年6月4日，文山州医疗保障局、文山州卫生健康委员会联合印发《关于规范综合诊查类等医疗服务项目价格及医保支付政策的通知》（以下简称《通知》），贯彻落实云南省（云医保〔2025〕52号）要求。

## 核心数据

| 指标 | 数据 |
|------|------|
| 整合前项目数 | **2840项** |
| 整合后项目数 | **1806项** |
| 规范批次 | **17批** |
| 终止原有项目 | 2480项（含"挂号费"等） |
| 纳入医保支付 | **1499项**（甲类1203项 + 乙类296项） |
| 医保不予支付（丙类） | 307项 |
| 医保覆盖率 | 83% |
| 执行时间 | **2026年5月30日零时起** |

## 主要规范

### 一、统一规范
- 统一项目编码名称，使用全国统一编码
- 统一计价规则（如"床位费"优化整合为9项+1项加收+1项减收+1项拓展）
- 统一价格项目内涵并预留拓展空间

### 二、新增项目
新增"**安宁疗护费**"、"**侵入式脑机接口置入费**"等契合临床需求、符合医疗发展方向的新型医疗服务价格项目。

### 三、价格形成原则
- "先平移后调整"，总体不增负、优化调结构
- 宫腔镜检查：原250+12元=262元，规范后仍为262元
- 乡镇卫生院输液：不再收取输液观察床/椅费用

### 四、民生减免政策
全州各级各类非营利性医疗机构，对经民政部门认定的**最低生活保障对象、特困人员**：
- **免收**门诊诊查费
- **减收20%** 床位费

### 五、特殊刀使用费调整
整合为"可复消手术特殊刀使用费（项目编码33m）"，终止"超声高频外科集成系统使用费""水动力系统使用费"项目。

## 覆盖范围

《通知》覆盖综合诊查类、超声检查类、放射治疗类、精神治疗类、妇科类、眼科类、体被系统类、骨骼肌肉系统类、甲乳类和淋巴结清扫、心血管系统类、泌尿系统类、神经系统类、呼吸系统类、麻醉类、血液系统类、康复类、物理治疗类共17批1806项。

## 非公立医疗机构

按照发改价格〔2014〕503号规定执行市场调节价，医保定点医疗机构按服务协议约定事项接受管理。
