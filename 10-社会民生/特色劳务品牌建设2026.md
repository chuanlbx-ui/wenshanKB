# 文山州特色劳务品牌建设（2026年）

> **来源**：文山州人力资源和社会保障局官网
> **说明**：文山州构建边疆特色劳务品牌体系，激活乡村振兴就业动能

---

## 一、总体数据

| 指标 | 数值 |
|------|------|
| 全州特色劳务品牌数量 | **36 个** |
| 累计撬动就业人数 | **超 80 万人** |
| 年务工总收入 | **151.56 亿元** |
| 劳务品牌经济规模 | **突破 1500 亿元** |

---

## 二、重点劳务品牌

### 2.1 "文山三七产业工"

| 指标 | 数值 |
|------|------|
| 直接从业人员 | **1.46 万人** |
| 带动就业人数 | **4.38 万人** |
| 配套机构 | 30余家国内外科研机构及院校 |
| 院士（专家）工作站 | **32 个** |
| 标准化基地（2026） | 1.5万亩 GAP + 8000亩林下三七 |
| 产业学院 | 三七大健康产业学院 |

> 模式："企业+合作社+农户"，年提供务工岗位20万人次，户均年增收约3万元。

### 2.2 "文山州绿色铝产业工"

| 指标 | 数值 |
|------|------|
| 全产业链直接就业 | **1.2 万人** |
| 带动就业人数 | **2.4 万人** |
| 年务工收入 | **13.18 亿元** |
| 电解铝建成产能 | 343 万吨 |
| 铝产业公共实训基地 | 全国唯一 |

> 配套：文山州绿色铝产业学院，与昆明理工大学、昆明冶金高等专科学校协同合作。

### 2.3 "砚山彝绣工匠"

深挖非遗文化，编织致富之路（详见原文）。

---

## 三、建设举措

1. **产学研协同**：与30余家科研机构及院校合作，建成32个院士（专家）工作站
2. **电商人才培养**：推进三七产业直播电商人才培养
3. **产业学院建设**：三七大健康产业学院、绿色铝产业学院
4. **公共实训基地**：全国唯一的铝产业公共实训基地，助力1100名员工学历、技能"双提升"
5. **龙头企业引进**：魏桥、神火、中铝等世界500强、中国500强企业落地文山
