# 绿色铝产业保障支撑

> 来源：经济日报 / 新浪财经 | 2026-04-23  
> 锚点数据：GDP=1632.64亿 | 常住人口=335.4万 | 城镇化率=41.19% | 绿色铝产值=1000亿 | 三七综合产值=456亿

## 一、产业基本盘

| 指标 | 数据 |
|------|------|
| 电解铝产能 | 343 万吨 |
| 铝加工产能 | 407.4 万吨 |
| 建成及在建再生铝产能 | 210 万吨 |
| 2025年绿色铝产值 | 1016 亿元（文山首个千亿级产业） |
| 2026年目标 | 绿色铝产值增长 20% 以上 |

核心园区：砚山绿色铝产业园区（云南宏泰203万吨为龙头）、富宁产业园区（云南神火90万吨）。

## 四、电力保障

| 事项 | 说明 |
|------|------|
| 服务模式 | 办电一站式服务 + 用电保姆式服务 |
| 线上功能 | 线上报装、电费查询、故障申报一站式 |
| 办理效率 | 园区新装增容业务平均时长压缩至 **20 个工作日** |

| 专属团队 | 常态化上门走访，根据生产用电情况实时建言献策 |

## 五、金融支持

| 事项 | 说明 |
|------|------|
| 服务模式 | "**圈链群**"服务模式（建行文山分行） |
| 供应链金融 | 搭建跨区域集群供应链金融平台，产业产能共享与金融风险共担 |
| 一体化营销 | 构建绿色铝产业生态圈，辐射带动区域铝产业集群 |
| 金融服务港湾 | 在铝产业园区重点打造 |
| 累计支持 | **10 余家**园区企业获贷款 **12.52 亿元** |

## 六、2026年重点推进

- 中铝绿材、信特等重点项目建成投产
- 50 万吨再生铝项目建成投产
- 促进电解铝、再生铝和铝加工融合发展
- 围绕绿电消纳精准招商，吸引高附加值、高用电负荷绿色产业项目

## 七、产业链构成

铝土矿 → 氧化铝 → 绿色铝 → 铝材精深加工 → 再生铝 → 综合利用

砚山园区：以云南宏泰为龙头，配套云南文都新材料（50万吨预焙阳极炭素）、云南创格新材料（高端铝合金/铝棒，铝水20分钟直供）。

富宁园区：以云南神火90万吨电解铝为依托，引进18家涉铝企业、24个项目，签约产能含40万吨炭素、116万吨铝加工、48.5万吨再生铝。

## 八、综合效应

- 带动就业增收，本地群众实现"家门口上班"
- 带动餐饮等服务业发展（如鲁味印象等配套商业）
- 释放锌、铟等有色金属产能，扩大高载能产业用电规模
