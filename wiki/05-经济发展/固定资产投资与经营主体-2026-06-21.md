# 文山州固定资产投资与经营主体（2025年）

> 来源：文山州2025年国民经济和社会发展报告、2026年政府工作报告

---

## 一、固定资产投资

| 指标 | 数值 |
|------|------|
| 2025年固投总额 | 537.69 亿元 |
| 较2024年减少 | 35.84 亿元 |
| 趋势 | 连续三年下降（自2023年起） |
| 十四五年均增速 | -1.6% |
| 亿元以上项目占比 | 22.1% |
| 民间投资增速 | -5.9%（连续六年下降） |

民间投资在交通、卫生、水利、教育、数字经济、生态保护等行业中参与度不高。

## 二、社会消费品零售总额

| 指标 | 数值 |
|------|------|
| 2025年增速 | +3.7% |
| 较2023年回落 | 3.9 个百分点 |
| 较2024年回落 | 1.3 个百分点 |
| 消费品以旧换新交易额 | 2.54 亿元 |
| 趋势 | 增速逐年回落，月度波动收缓 |

## 三、一般公共预算收入

| 指标 | 数值 |

| 超长期特别国债 | 10.55 亿元 |
| 实施重点项目 | 94 个 |

## 四、经营主体

## 五、建筑业与房地产

| 指标 | 说明 |
|------|------|
| 建筑业增加值占GDP | 11.2% |
| 2025年建筑业总产值增速 | 逐季回落，Q1 3.5% → 全年 2.2% |
| 十四五建筑业年均增速 | 5.0%（十三五 13.9%） |
| 房地产投资及销售 | 长期明显下降态势 |

## 六、2026年工作方向

- 将"稳存量"作为基础、"抓项目"作为关键、"挖增量"作为突破、"抓纳规"作为保障
- 2026年预期目标：GDP增长5%以上、规上工业增长10%以上、固投增速与上年持平
- 社消零售总额增长4%、一般公共预算收入增长1%
