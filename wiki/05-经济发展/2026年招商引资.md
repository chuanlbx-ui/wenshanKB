# 2026年招商引资

> 2026年文山州把招商引资作为"一号工程"，以"产业强州"战略为牵引，围绕四大重点产业精准招商。首批谋划133个重点招商项目，筛选10个标杆项目重点推进。

## 一、1—4月招商数据

| 指标 | 数值 |
|------|------|
| 新签约项目 | **28 个** |
| 协议总投资 | **27.81 亿元** |
| 其中亿元以上项目 | **9 个** |
| 新落地项目 | **39 个** |
| 其中亿元以上落地 | **14 个** |
| 产业招商到位资金 | **52.97 亿元** |

## 三、四大产业招商方向

### 3.1 绿色铝产业

| 招商方向 | 具体领域 |
|---------|---------|
| 精深加工 | 电池铝箔、动力电池壳体 |
| 循环经济 | 再生铝回收利用 |
| 高端制造 | 航空航天用铝、低空经济铝材 |
| 配套产业 | 铝加工装备制造、物流仓储 |

### 3.2 三七产业

| 招商方向 | 具体领域 |
|---------|---------|
| 提取物 | 三七总皂苷等高附加值提取 |
| 中药配方颗粒 | 标准化中药饮片及配方颗粒 |
| 健康食品 | 三七功能性食品、保健品 |

### 3.3 现代农业

| 招商方向 | 具体领域 |
|---------|---------|
| 粤港澳大湾区"菜篮子" | 高原特色蔬菜水果直供基地 |
| 绿色食品加工 | 农产品精深加工、冷链物流 |
| 品牌农业 | 地理标志产品开发运营 |

### 3.4 文旅康养

| 招商方向 | 具体领域 |
|---------|---------|
| 大型文旅综合体 | 景区综合开发运营 |
| 医养融合旅居 | 三七中医康养基地 |
| 边关民宿集群 | 边境特色民宿、乡村旅游 |

## 五、第10届南博会招商专场（2026年6月）

> 来源：云南日报 2026-06-13

2026年6月12日，文山州在昆明举行第10届南博会招商专场推介会，邀请50余位全国企业家代表参加。

| 指标 | 数值 |
|------|------|
| 签约协议 | **6 个**（1个框架协议 + 5个投资协议） |
| 投资总额 | **13.46 亿元** |
| 覆盖领域 | 新能源光伏、风力发电、矿产资源开发、农产品冷链物流 |
| 重点推介 | 绿色铝产业（州工信局）、三七和中药材产业（州三七和中医药产业发展中心） |

招商推介会上，文山州明确铝产业「铝水100%就地转化、铝材100%精深加工」双百目标，重点培育汽车零部件、建筑铝材、终端消费品"三个百万吨级"绿色铝精深加工集群。
