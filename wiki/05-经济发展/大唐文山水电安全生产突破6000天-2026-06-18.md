# 大唐文山水电安全生产突破6000天

> 来源：云南网 | 2026-06-18
> 截至2026年6月17日，大唐文山水电公司实现连续安全生产6013天。

## 一、安全生产里程碑

| 指标 | 数值 |
|------|------|
| 连续安全生产天数 | **6013 天** |
| 首台机组投产 | 2009年12月 |
| 累计运行时间 | **16 年余** |

## 二、马鹿塘水电站二期工程

| 项目 | 内容 |
|------|------|
| 定位 | 文山州装机容量最大的水电站 |
| 功能一 | 区域清洁能源重要支撑 |
| 功能二 | 防洪减灾"压舱石" |

## 三、安全管理体系

### "三零"目标

| 维度 | 目标 |
|------|------|
| 人员安全 | 人员零伤害 |
| 设备运行 | 机组零非停 |
| 环境保护 | 环保零事件 |

### "安全生产十条硬措施"

落实以下管控机制确保安全运行：

- **春季安全大检查**：季节性风险排查
- **防汛专项排查**：汛期防洪安全管控
- **重大隐患治理**：系统性消除安全隐患

## 四、产业意义

马鹿塘水电站二期作为文山州最大水电站，在清洁能源供给和流域防洪体系中具有关键地位，其超16年安全运行纪录体现了区域水电运营的成熟能力。
