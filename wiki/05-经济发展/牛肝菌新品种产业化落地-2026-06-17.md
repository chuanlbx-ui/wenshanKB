# 国内首个牛肝菌新品种落地文山

> 来源：人民日报客户端 | 2026-06-17

## 一、签约概况

2026年6月，文山市举办牛肝菌技术授权合作签约暨产业顾问颁证仪式。**中华全国供销合作总社昆明食用菌研究所**与**江苏华绿生物科技集团股份有限公司**正式签订牛肝菌技术授权合作协议，全面落地转化昆明食用菌研究所核心专利及多款自主选育牛肝菌新品种。

## 二、三大品类

| 品种 | 说明 |
|------|------|
| 红牛肝菌（见手青） | 云南标志性珍稀食用菌 |
| 中菌黄见手青1号 | 昆明食用菌研究所自主选育新品种 |
| 黑牛肝菌 | 高价值食用菌 |

## 三、投资与产能

| 指标 | 数值 |
|------|------|
| 总投资 | **2 亿元** |
| 一期占地 | 约 **220 亩** |
| 年产鲜菇（达产） | **18,000 吨** |
| 日产鲜菇（达产） | **50 吨** |
| 定位 | **全国规模最大**的红牛肝菌、中菌黄见手青1号、黑牛肝菌综合生产基地 |

## 四、技术根基

核心技术源自昆明食用菌研究所多项自研成果，双方整合技术、产业、市场优势开展深度协作，推动三大品类菌种与栽培技术实现产业化应用。

## 五、产业意义

- 填补国内牛肝菌工厂化栽培产业化空白
- 助力文山市珍稀食用菌产业规模化、高质量发展
- 为乡村振兴筑牢产业根基
- 拓展文山州"1+4+N"高原特色现代农业产业体系新赛道
