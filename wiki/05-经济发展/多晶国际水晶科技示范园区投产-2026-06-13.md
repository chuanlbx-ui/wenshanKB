# 砚山多晶国际水晶科技示范园区投产

> 砚山县多晶国际水晶科技示范园区（一期）于2026年6月正式投产，创新推行"边调试、边生产"模式。项目全面建成后年产值28亿元。同期第10届南博会上坝心风电场项目签约。

## 一、水晶科技示范园区

| 指标 | 数据 |
|------|------|
| 项目名称 | 多晶国际水晶科技示范园区 |
| 所在地 | 砚山县 |
| 一期状态 | 2026年6月正式投产 |
| 达产规模 | 年产3万吨精品特A水晶钻石 |
| 年产值（达产） | **28 亿元** |
| 首批投产 | 100个机组，日产约10吨 |
| 二期计划 | 2026年6月底逐步投产 |
| 配套工程 | 110千伏输变电工程冲刺收尾 |

**生产模式**：创新推行"边调试、边生产"，加快产能爬坡。

## 二、坝心风电场签约

第10届南博会上，砚山县人民政府与大唐云南发电有限公司签约坝心风电场项目：

- **签约主体**：大唐云南发电有限公司
- **所属板块**：文山州"四大重点产业"中绿色铝及新能源板块
- **定位**：文山绿色能源新引擎

## 三、产业意义

- 水晶科技示范园区是砚山工业经济高质量发展的重要增量，达产后单项目年产值28亿元，契合文山"产业强州"战略
- 坝心风电则进一步充实绿色能源矩阵，为绿色铝等高载能产业提供配套清洁电力
