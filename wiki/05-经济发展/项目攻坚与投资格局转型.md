# 项目攻坚与投资格局转型

> 来源：文山发布网（云南发布）| 2026-06-03
> 文山州把项目建设作为稳增长、调结构、补短板的硬核抓手，过去五年累计实施重点项目 1876 个，投资结构实现历史性优化。

## 一、项目攻坚历程

### "十四五"投资规模

| 指标 | 数值 |
|------|------|
| 累计实施重点项目 | **1876 个** |
| 完成投资 | **3880.46 亿元** |
| 较"十三五" | 逆势正增长 |

### "十五五"储备

| 指标 | 数值 |
|------|------|
| 储备重大项目 | **4867 个** |
| 总投资 | **超 1.6 万亿元** |
| 储备重点产业类项目 | **2845 个**，总投资接近万亿 |

### 2026 年项目攻坚

| 指标 | 数值 |
|------|------|
| 向上争取落地项目 | **71 个** |
| 争取资金 | **13.63 亿元** |
| 亿元以上重大项目 | **166 个**（政府工作报告口径） |

## 二、投资结构历史性转变

| 指标 | "十三五"末 | 当前 | 变化 |
|------|-----------|------|------|
| 产业投资占比 | 32% | **62%** | +30pp |
| 工业投资占比 | 19.6% | **42.6%** | +23pp |
| 民间投资占比 | — | **37.5%** | 活力持续增强 |

> 彻底改变以往依靠基建投资拉动的旧模式，**产业投资成为拉动经济增长的绝对主力**。

## 三、要素保障突破

| 要素 | 成果 |
|------|------|
| **土地** | 优化国土空间布局，调剂优质指标保障园区；富宁铝园区 **1500 亩**新增产业用地获批 |
| **资金** | "十四五"累计争取上级各类资金 **352.48 亿元** |
| **能源** | 全州新能源并网超 **520 万千瓦**；丘北储能项目为**全省唯一国家级新型储能示范项目**，稳定并网；**149 公里**天然气管道贯通园区，用气成本大幅下降 |

## 四、2026 年攻坚机制

- 紧扣"项目攻坚年"，细化九大攻坚任务
- 州级领导包保、清单化推进、常态化调度
- 形成"开工一批、建设一批、投产一批、储备一批"良性循环

## 五、重点基础设施进展

| 项目 | 状态 |
|------|------|
| 那兴高速 | 已通车，山区乡镇迈入高速时代 |
| 文蒙铁路 | 加快建设 |
| 富宁港 | 加快建设，2026 年底前建成 |
| 公铁水空立体交通格局 | 从蓝图变为现实 |

## 六、产业带动就业

- 全州铝产业直接带动 **1.3 万余人**就业
- 本地群众实现"家门口上班、按月拿工资、顾家又增收"

## 七、2845个重点产业项目分解

> 来源：文山发布 2026-06-04《文山州谋划储备重点产业类项目2845个》

| 产业板块 | 重点项目 |
|---------|---------|
| **绿色铝** | 中铝20万吨铸轧、砚山智铝二期、神火超薄铝箔、华疆再生铝循环利用 |
| **三七中药材** | 全产业链升级、健康产业园、日化品加工、华润三九血塞通基地、白药气血康二期 |
| **现代农业** | 砚山蔬菜供应基地、蓝莓产业园、广南铁皮石斛、畜禽精深加工（目标550亿） |
| **康养旅游** | 山水田园、康养旅居、边境风情、红色文化四大板块 |
| **新兴产业** | 风电光伏、储能电站、铝基新材料、大数据中心 |
