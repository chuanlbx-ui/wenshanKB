## 三、四大产业招商方向

### 3.1 绿色铝

| 招商方向 | 具体领域 |
|---------|---------|
| 终端产品 | 电池铝箔、动力电池壳体 |
| 循环经济 | 再生铝综合利用 |
| 高端制造 | 航空航天、低空经济铝材 |
| 目标 | 2030年铝产业综合产值突破 **1800 亿元** |

### 3.2 三七中药材

| 招商方向 | 具体领域 |
|---------|---------|
| 提取物 | 三七提取物精深加工 |
| 中药配方颗粒 | 标准化中药饮片及配方颗粒 |
| 健康食品 | 三七功能性食品、保健品 |
| 转型目标 | 从"原料基地"向"精深加工高地"转型，2030年实现千亿级 |

### 3.3 高原特色现代农业

| 招商方向 | 具体领域 |
|---------|---------|
| 绿色食品加工 | 农产品精深加工 |
| 冷链物流 | 生鲜农产品冷链配送 |
| 目标 | 打造粤港澳大湾区"菜篮子"基地、绿色食品加工基地 |
| 体系 | "1+4+N"现代农业产业体系 |

### 3.4 康养旅游旅居

| 招商方向 | 具体领域 |
|---------|---------|
| 大型文旅综合体 | 景区综合开发运营 |
| 医养融合旅居 | 三七中医康养基地 |
| 边关民宿集群 | 边境特色民宿 |
| 五大主题 | 文旅体验、康养度假、边境风情、田园旅居、红色研学 |
| 转型目标 | 从"旅游过境地"向"旅居目的地"升级 |

## 四、制度保障

- 把招商引资作为推动高质量发展的"一号工程"
- 全面落实云南省委"3815"战略发展目标
- 以产业突破激活发展动能
- 强化全流程跟踪服务，加速推进重点项目从"纸上"落到"地上"
