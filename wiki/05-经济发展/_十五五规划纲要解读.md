# 文山州"十五五"规划纲要解读

> 来源：文山州政府办（2026-06-16）、央广网（2026-06-04）
> 锚点基准：GDP=1632.64亿（2025年）| 绿色铝产值=1016亿（2025年）

## 一、"3+3"发展定位

| 层级 | 定位 |
|------|------|
| 省级三大定位 | 民族团结进步示范区 |
| | 石漠化地区生态文明建设排头兵 |
| | 云南面向东南开放先行区 |
| 州级三大特色 | 新时代兴边富民示范区 |
| | "绿色铝谷"核心区 |
| | 三七康养旅居目的地 |

**四项重点**：产业、项目、干部、环境

## 二、四大重点产业

### 2.1 绿色铝（重中之重）

- 2025年电解铝产能 343 万吨，占全省 52.8%、全国 7.7%，文山成为全国"北铝南移"标志区
- 发挥"绿电+先进制造业"新优势，推动向新材料、终端制造等价值链高端延伸
- **2030年目标**：铝产业综合产值突破 **1800 亿元**，打造云南"绿色铝谷"核心区

### 2.2 三七中药材

- 文山是三七原产地、主产区，独具优势的生物医药资源
- 紧扣将中药材产业打造为云南"第二个烟草产业"的要求
- 做大做强三七全产业链
- **目标**：全力以赴向 **千亿级产业** 迈进

### 2.3 高原特色现代农业

- 海拔相对高、纬度相对低、降雨相对多，北回归线贯穿全境（农作物"天然温床"）
- 发展生态农业、设施农业、高效农业、共享农业，推动一二三产深度融合
- 建成全国优质绿色现代农产品生产基地和绿色食品加工基地、粤港澳大湾区"菜篮子"基地
- **2030年目标**：农林牧渔业总产值 **550 亿元**

### 2.4 康养旅游旅居

- 系统推进旅游产品、模式、业态创新和服务创优
- 打造"有一种叫云南的生活——三七之乡 康养文山"
- "旅居云南·好在文山"品牌

## 三、高水平对外开放

以 **"2条水路、3条铁路、8条高速公路、7条国道改扩建"** 为交通骨架，构建跨境多式联运综合交通网络。

| 项目 | 时间节点 | 意义 |
|------|---------|------|
| 富宁港 | **2026年底与平陆运河同步建成** | 云南出海最近"黄金港口" |
| 文蒙铁路 | **2028年建成通车** | 云桂沿边铁路核心段 |
| 文靖铁路 | **2030年建成通车** | 西南沿边铁路重要组成 |
| 曲靖至文山铁路 | 争取突破 | — |

3个国家级口岸，融入西部陆海新通道，以大通道带动大物流、促进大产业、推进大发展。

## 四、生态文明建设

| 指标 | 数值 |
|------|------|
| 空气优良率 | **97.5%** |
| 国控省控地表水断面水质优良率 | **100%** |
| 生态文明示范 | 马关县、麻栗坡县：全国生态文明建设示范区 |
| | 西畴县："绿水青山就是金山银山"实践创新基地 |

持续推动石漠化综合治理，以普者黑湖、盘龙河"一湖一河"为重点的河湖保护，推进全域绿化美化。

## 五、民生保障

- "十四五"期间 **80%以上** 财政支出用于民生
- 农村居民人均可支配收入增速高于全省平均水平
- 牢牢守住不发生规模性返贫致贫底线
- 以更大力度"投资于人"

| 指标 | 数据 |

## 七、项目储备

- 建立"谋划一批、储备一批、开工一批、投产一批"滚动推进机制
