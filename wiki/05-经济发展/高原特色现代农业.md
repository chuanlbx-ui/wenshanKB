# 高原特色现代农业

> 文山州依托北回归线气候优势，构建"1+4+N"高原特色现代农业产业体系，持续建设粤港澳大湾区"菜篮子"基地和全国优质绿色农产品生产基地。2026年Q1农林牧渔业总产值56.81亿元，同比+2.5%。

| 指标 | 数值 | 同比/说明 |
|------|------|-----------|
| 2026年Q1农林牧渔业总产值 | **56.81 亿元** | +2.5% |
| 农业产值增速 | — | +5.9% |

## 二、"1+4+N"产业体系

| 层级 | 内容 |
|------|------|
| **1** 条主线 | 高原特色现代农业高质量全产业链 |
| **4** 大方向 | 生态农业、设施农业、高效农业、共享农业 |
| **N** 个特色产业 | 三七、辣椒、蔬菜、水果、茶叶、油茶、肉牛、生猪、家禽、渔业等 |

## 三、产业格局："三带五区多基地"

> 来源：中国农网 2026.06.11

"十五五"期间，文山州按照"稳住粮、优化经、巩固畜、提升林、突破药、保住烟、拓展渔、融合旅"总体思路，加快构建"三带五区多基地"现代高原特色农业产业格局。

### 三大特色产业示范带

| 示范带 | 核心内容 |
|--------|---------|
| 现代绿色农业示范带 | 绿色粮油、有机蔬菜、品牌化运营 |
| 沿边热区特色农业示范带 | 热带水果、香料、沿边特色种养 |
| 林下经济产业示范带 | 林下三七、草果、砂仁等中药材 |

### 五大优势产业区

| 产业区 | 核心方向 |
|--------|---------|
| 优质粮油产业区 | 八宝米、优质稻、冬早玉米 |
| 三七中药材产业区 | GAP标准化种植、种质资源保护 |
| 高端果蔬产业区 | 蓝莓、菜心、上海青等供粤港澳 |
| 高品质肉牛产业区 | 良种繁育、规模化养殖 |
| 农文旅融合产业区 | 普者黑、坝美周边农旅融合 |

## 四、粤港澳大湾区"菜篮子"基地

- 建设全国优质绿色现代农产品生产基地
- 打造绿色食品加工基地
- 重点发展绿色食品加工、冷链物流
- 对接粤港澳大湾区市场需求，推动"文品出山"
- **粤港澳大湾区"菜篮子"生产基地已建成 23 个**
- 省级绿色食品基地 **47 个**
- 2025年已建成绿色农产品基地：583个绿色/有机/地理标志农产品
- 打通"出滇、入湾、出海"通道，电商直销、村播带货助力

## 六、联农带农与农村改革

> 来源：中国农网 2026.06.11

### 6.1 联农带农模式

- 推行"企业+合作社+农户"模式，小农户嵌入大产业链
- 抱团发展、利益共享、风险共担
- 文山三七、八宝米等借助电商直销、村播带货打通"出滇、入湾、出海"通道

### 6.2 农村要素市场化改革

| 改革事项 | 2030 年目标 |
|----------|------------|
| 承包地经营权流转 | **150 万亩**以上 |
| 村级集体经济村均收入 | **25 万元**以上 |
| 承包地经营权/农房抵押贷款 | 扩大试点范围 |

- 深化承包地"三权分置"，引导土地经营权有序流转
- 盘活闲置宅基地和农房发展民宿、文创、康养
- 推动"资源变资产、资金变股金、农民变股东"
