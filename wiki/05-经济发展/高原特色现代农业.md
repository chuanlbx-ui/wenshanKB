# 高原特色现代农业

> 文山州依托北回归线气候优势，构建"1+4+N"高原特色现代农业产业体系，持续建设粤港澳大湾区"菜篮子"基地和全国优质绿色农产品生产基地。2026年Q1农林牧渔业总产值56.81亿元，同比+2.5%。

## 一、核心数据

| 指标 | 数值 | 同比/说明 |
|------|------|-----------|
| 2026年Q1农林牧渔业总产值 | **56.81 亿元** | +2.5% |
| 农业产值增速 | — | +5.9% |
| 林业产值增速 | — | +6.8% |
| 渔业产值增速 | — | +3.3% |
| 牧业产值增速 | — | -1.6% |
| 2026年目标 | 农林牧渔业总产值增长 **3%以上** | 政府工作报告 |

## 二、"1+4+N"产业体系

| 层级 | 内容 |
|------|------|
| **1** 条主线 | 高原特色现代农业高质量全产业链 |
| **4** 大方向 | 生态农业、设施农业、高效农业、共享农业 |
| **N** 个特色产业 | 三七、辣椒、蔬菜、水果、茶叶、油茶、肉牛、生猪、家禽、渔业等 |

## 三、粤港澳大湾区"菜篮子"基地

- 建设全国优质绿色现代农产品生产基地
- 打造绿色食品加工基地
- 重点发展绿色食品加工、冷链物流
- 对接粤港澳大湾区市场需求，推动"文品出山"
- 2025年已建成绿色农产品基地：583个绿色/有机/地理标志农产品

## 四、龙头企业与品牌

| 指标 | 数据 |
|------|------|
| 农业产业化龙头企业 | **552 家** |
| 绿色/有机/地理标志农产品 | **583 个** |
| 高标准农田 | **290.42 万亩** |
| 中药材种植面积 | 245.2 万亩（稳居全省第一） |

## 五、招商引资方向

- 绿色食品加工项目
- 冷链物流体系建设
- 农产品精深加工
- 粤港澳大湾区"菜篮子"基地配套
- 重点依托"四个农业"（生态、设施、高效、共享）推进现代化转型

## 六、"十五五"目标

- 建成全国优质绿色现代农产品生产基地
- 建成绿色食品加工基地
- 建成粤港澳大湾区"菜篮子"基地
- 持续发展壮大生态农业、设施农业、高效农业、共享农业
- 推动农产品从"原料供给"向"精深加工+品牌输出"转型
