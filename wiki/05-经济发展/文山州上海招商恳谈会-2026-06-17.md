# 文山州招商恳谈会在上海举行

> 来源：文山州政务网/文山发布网 | 2026-06-17
> 2026年6月16日下午，文山州招商恳谈会在上海市上海银行举行。

## 概况

| 项目 | 详情 |
|------|------|
| 时间 | 2026年6月16日下午 |
| 地点 | 上海市上海银行 |
| 主办方 | 文山州政府、上海银行 |
| 出席领导 | 州委副书记、州长马忠俊；上海银行党委副书记、行长施红敏 |
| 专题推介 | 副州长郑锦 |
| 参会规模 | 20余家企业负责人 |
| 其他出席 | 张发政 |

## 四大产业推介

马忠俊对文山四大重点产业作了全面介绍：

| 产业 | 定位/目标 |
|------|----------|
| 绿色铝 | 全力打造"绿色铝谷"核心区 |
| 三七中药材 | 打造世界"三七之都" |
| 高原特色现代农业 | 打造粤港澳大湾区"菜篮子" |
| 康养旅游旅居 | 打造世界的"世外桃源" |

马忠俊表示，近年来一批优质上海项目落地文山，为绿色铝、现代农业、新能源等产业延链补链、提质增效提供了强劲支撑。迈上"十五五"新征程，投资文山正当其时、恰逢其势，文山将以最大诚意、最优政策、最好服务、最强保障，当好企业发展的坚强后盾。

## 上海银行角色

施红敏表示，文山"四大重点产业"集群正加快形成，投资文山、兴业文山正当其时。上海银行将以此次招商恳谈会为契机：

- 整合各类优势资源
- 聚焦文山产业发展，提供精准金融服务
- 助力两地企业双向拓展、共赢发展

## 企业家交流方向

与会企业家围绕以下领域开展交流，希望进一步加强协作、谋划具体项目合作：

- 绿色铝
- 低空经济
- 物流运输
- 康养旅游
- 城市建设

## 背景意义

这是文山州"一把手"招商在上海的重要活动。通过与上海银行的金融合作，打通上海—文山产业资源对接通道，拓展"十五五"时期招商引资新格局。低空经济作为新兴方向首次出现在文山招商恳谈议题中，值得关注。
