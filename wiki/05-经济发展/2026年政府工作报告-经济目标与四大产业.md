# 2026年政府工作报告：经济目标与四大产业部署

> 来源：文山州人民政府 2026年政府工作报告（文政发〔2026〕4号），州长马忠俊2026年1月20日作
> 锚点基准：GDP=1632.64亿（2024年）| 绿色铝产值=1000亿（2025年突破）| 三七综合产值=456亿

## 一、2026年主要预期目标

| 指标 | 目标 | 牵头单位 |
|------|------|---------|
| 地区生产总值增长 | **5% 以上** | 州发展改革委 |
| 规模以上工业增加值增长 | **10% 以上** | 州工业和信息化局 |
| 地方一般公共预算收入增长 | **1%** | 州财政局 |
| 固定资产投资 | 与上年持平 | 州发展改革委 |
| 社会消费品零售总额增长 | **4%** | 州商务局 |
| 居民人均可支配收入增长 | 与经济增长同步 | — |
| CPI 涨幅 | **2% 左右** | — |
| 单位GDP二氧化碳排放 | 完成省下达任务 | — |

## 二、"3+3"发展定位

| 层级 | 定位 | 说明 |
|------|------|------|
| 省级 | 民族团结进步示范区 | — |
| 省级 | 石漠化地区生态文明建设排头兵 | — |
| 省级 | 云南面向东南开放先行区 | — |
| 州级 | 新时代兴边富民示范区 | — |
| 州级 | **"绿色铝谷"核心区** | 铝产业引领 |
| 州级 | **三七康养旅居目的地** | 三七+康养+文旅融合 |

## 三、四大重点产业2026年部署

### 3.1 绿色铝（新型工业）

| 维度 | 内容 |
|------|------|
| 产业体系 | 构建完善 **"1+1+3+N"** 铝产业体系 |
| 2026年目标 | 绿色铝产值**增长 20% 以上** |
| 重点项目 | 中铝绿材、信特等项目建成投产 |
| 再生铝 | **50 万吨**再生铝建成投产 |
| 发展模式 | "绿电+先进制造业" |
| 方向 | 电解铝、再生铝和铝加工融合发展 |

### 3.2 三七中药材

| 维度 | 内容 |
|------|------|
| 2026年综合产值目标 | **500 亿元以上** |
| GAP基地 | 新建 **1.5 万亩以上** |
| 重点项目 | 种质种源、特色产业集群、气血康扩产扩能 |
| 招商目标 | 力争华润三九血塞通系列加工基地**落地建设** |
| 平台升级 | 改造提升三七中药材交易中心 |

### 3.3 现代农业

- 农林牧渔业总产值增长 **3% 以上**
- 培育农业龙头企业 **50 户**
- 新增绿色优质农产品 **70 个**
- "1+4+N" 现代农业体系

### 3.4 康养旅游旅居

| 品牌 IP | 定位 |
|------|------|
| 主品牌 | **"有一种叫云南的生活—三七之乡 康养文山"** |
| 旅居品牌 | **"旅居云南·好在文山"** |

重点任务：
- 普者黑景区业态重塑
- 坝美景区生态化提升
- 边境旅游试验区创建
- 培育康养旅居、赛事经济等新业态

## 四、重大基础设施

| 类别 | 2026年目标 |
|------|-----------|
| 高速公路 | 天麻高速、师丘高速建成通车 |
| 铁路 | 文靖铁路年内开工；加快文蒙铁路建设 |
| 水运 | 富宁港水运枢纽建成投运 |
| 机场 | 文山砚山机场改扩建推进 |
| 总体 | 强化"**水陆空铁**"综合交通网络 |

## 五、科技创新与经营主体

| 指标 | 目标 |
|------|------|
| 新增高新技术企业 | **5 户** |
| 新增省级科技型中小企业 | **30 户** |
| 全社会研发经费增长 | **10% 左右** |
| 新增规上工业企业 | **22 户** |
| 新增限上批零住餐企业 | **32 户** |
| 新增资质以上建筑业企业 | **8 户** |

## 六、对外开放

- 外贸进出口总额、跨境电商交易增长 **10% 以上**
- 引进亿元产业项目 **50 个以上**
- 天保口岸进境冰鲜水产品、食用水生动物指定监管场地投用
- 开通田蓬口岸国际道路客货运输线路

## 七、能源保障

- 开工富宁煤电、锦屏西风电三四期
- 加快西畴抽水蓄能项目
- 构建"风光水火储""源网荷储"一体化
- 开工 500kV 南屏变、富宁天然气管道支线
- 建成投运百色至文山天然气管道
