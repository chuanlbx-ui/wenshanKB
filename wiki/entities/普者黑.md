# 普者黑

> 国家 5A 级旅游景区，高原喀斯特山水田园风光代表。312 座孤峰、83 个溶洞、54 个湖泊，4 万亩野生荷花。

## 基本信息

| 项目 | 内容 |
|------|------|
| 等级 | 国家 5A 级旅游景区 |
| 位置 | 丘北县，距昆明 280 公里 |
| 面积 | 388 平方公里（核心区 15 平方公里） |
| 交通 | 距普者黑高铁站 9.8 公里，距文山砚山机场 85.7 公里 |
| 名称含义 | 彝语"盛满鱼虾的湖泊" |

## 六大景观

> 水上田园 · 彝家水乡 · 荷花世界 · 岩溶湿地 · 湖泊峰林 · 鱼鸟天堂

## 核心数字

| 要素 | 数量 |
|------|------|
| 孤峰 | 312 座 |
| 溶洞 | 83 个 |
| 相连湖泊 | 54 个 |
| 野生荷花 | 4 万亩 |
| 高原喀斯特湿地 | 6 万亩 |
| 景点 | 256 个 |

## 权威评价

- 国家湿地公园（2019 年验收）
- 建设部专家评价："世间罕见、中国独一无二的喀斯特山水田园风光"
- 兼具桂林山水之灵秀、江南水乡之古朴、西湖之明丽
