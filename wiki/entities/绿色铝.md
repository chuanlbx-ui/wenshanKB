# 绿色铝产业

> 文山州首个千亿级产业集群，电解铝产能 343 万吨（占全省 52.8%、全国 7.7%），2025 年产值 1016.07 亿元（突破千亿大关），同比增长 22%，合金化率 69.92%，成为全国"北铝南移"的标志性区域。

## 产能规模

| 指标 | 数值 | 备注 |
|------|------|------|
| 电解铝建成产能 | 343 万吨 | 占全省 52.8%、全国 7.7% |
| 铝加工产能 | 418 万吨 | 从 20 万吨飙升至 418 万吨 |
| 铝土矿储量 | 占全省 90% 以上 | — |
| 清洁能源占比 | 99.8% | 绿电使用比例超 70% |
| 2025 年产值 | 1016.07 亿元 | 首个千亿级产业，同比增 22% |
| 合金化率（2025） | 69.92% | 较 2023 年（42.5%）大幅提升 |
| 较 2021 年增长 | 3.18 倍 | — |
| 带动就业 | 1.3 万余人 | 全产业链带动 2.4 万人 |

## 产业链

```
铝土矿 → 氧化铝 → 绿色电解铝 → 铝材精深加工 → 再生铝 → 综合利用
```

> 文山已建成全省唯一全链条绿色铝产业体系。产业体系架构为「**1+1+3+N**」绿色铝产业体系。

## 园区载体

- **砚山产业园区**：全省 7 个千亿级园区之一，2023 年工业总产值 720 亿元
- **文山马塘工业园区**：传统铝工业基地
- **云南马关产业园区、丘北产业园区、广南产业园区**：编制产业地图，推动特色发展
- 全国唯一的铝产业公共实训基地落地文山

## 发展挑战

1. 优惠电价退出推高电解铝成本
2. "双碳"政策约束持续加强
3. 合金化率已提升至 69.92%（2025 年），精深加工链条仍需继续延伸
4. 下游终端消费品环节仍需延伸
