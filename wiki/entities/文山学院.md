# 文山学院

> 文山州唯一本科院校，前身为 1947 年国立西南师范学校，2009 年升格为本科。9 个二级学院，62 个专业，在校生 28,632 人。

## 历史沿革

| 年份 | 事件 |
|------|------|
| 1947 | 国立西南师范学校由昭通南迁文山 |
| 1955 | 成立文山师范学校 |
| 1984 | 升格为文山师范专科学校 |
| 2009 | 升格为本科院校——文山学院 |
| 近年 | 推进迁建新校区 |

## 学科布局

9 个二级学院：

| 学院 | 特色 |
|------|------|
| 教师教育学院 | 师范根基 |
| 三七医药学院 | 全国独树一帜，产教融合 |
| 冶金与材料学院 | 对接绿色铝产业 |
| 人工智能学院 | 新兴工科 |
| 经济管理学院 | — |
| 人文与传媒学院 | — |
| 美术与设计学院 | — |
| 马克思主义学院 | — |
| 高等职业技术学院 | — |

62 个专业（本科 49 个 + 专科 13 个）

## 特色亮点

- 联合发起成立"云南省三七研究院"
- 招收越南、老挝留学生，建设"国门大学"
- 三七医药学院直接服务千亿级三七产业
