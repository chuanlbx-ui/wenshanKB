# 三七

> 文山是三七的原产地和主产区，北回归线横贯境内。2025 年全州三七产业综合产值 **220 亿元**，以三七为主的中药材产业综合产值 **456 亿元**，增长 10.4%。三七精深加工集群获评**国家级特色产业集群**，正向千亿级产业目标迈进。

## 产地地位

- "中国三七之乡"、"世界三七之都"
- 国家地理标志保护产品
- 北回归线横贯境内，道地药材核心产区
- 人工栽培历史 400 年以上
- 三七种植系统认定为**中国重要农业文化遗产**
- 连续获得 **11 个国家级荣誉**

> **数据口径说明**：文山州三七产业数据存在两个口径——①窄口径"三七产业本身"综合产值220亿元（+7%）；②宽口径"以三七为主的中药材产业"综合产值456亿元（+10.4%），差额236亿包含砂仁、草果、石斛等全州其他中药材。引用时务必区分口径。详见 [[../topics/三七产业数据口径区分]]。

## 加工端

- 与 **60 家**知名企业合作（云南白药、华润三九等）
- 近三年签约落地项目 **70 个**，投资 **56 亿元**
- 精深加工集群获评**国家级特色产业集群**
- 三七成功入选"全国名特优新农产品"名录

### 龙头企业

- **云南白药**：投资 3.26 亿元实施产业提升工程，气血康生产线扩产后单家企业产值有望冲刺百亿元
- **昆药集团**（华润三九）：落地文山，打造全国最大血塞通绿色生产基地
- **金七药业** 等龙头企业协同发展

## 产业链

```
标准化 GAP 种植 → 精深加工 → 科技研发 → 大健康产品 → 市场流通 → 联农带农
```

## 制度保障

| 保障 | 说明 |
|------|------|
| 《文山三七发展条例》+ 实施细则 | 全国首个单一中药材品种地方条例 |
| 《文山州关于推动三七产业高质量发展的实施方案》 | "七大体系"（科研、种植、加工、金融等） |
| 文山三七研究院 | 专业研究机构 |
| 三七大健康产业学院 | 推行"课堂+田间+车间"教学模式 |
| 质量标准体系 | 覆盖全产业链的标准体系 |

## "十五五"目标

- 做大做强三七全产业链，**向千亿级产业目标迈进**
- 2028 年州内三七种植实现 GAP 全覆盖
- 持续扩大精深加工规模，提升产品附加值
