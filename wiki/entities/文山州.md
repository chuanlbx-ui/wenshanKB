# 文山州

> 文山壮族苗族自治州，云南省下辖自治州。2025 年 GDP 1,632.64 亿元，增长 5.3%，占全省 GDP 比重 4.98%；2026 年 Q1 增长 5.0%。常住人口 335.4 万人，下辖 1 市 7 县，与越南接壤边境线 438 公里。

## 基本概况

| 项目 | 内容 |
|------|------|
| 全称 | 文山壮族苗族自治州 |
| 行政级别 | 地级自治州 |
| 下辖 | 1 市 7 县（文山市 + 砚山、西畴、麻栗坡、马关、丘北、广南、富宁） |
| 州府 | 文山市 |
| 面积 | 31,456 平方公里 |
| 常住人口 | 335.4 万人（2025 年） |
| 民族 | 11 个世居民族，壮族（28.2%）、苗族（14.5%）为主 |
| 边境线 | 438 公里，与越南接壤 |

## 核心定位

- 云南唯一既紧邻沿海发达地区又接壤越南的州市
- "承接东西、贯通南北、通边达海"
- 全国绿色铝产能第一大州（占全省 52.8%、全国 7.7%）
- 全国三七核心产区
- 云南喀斯特地貌发育最典型区域
- 石漠化治理经验获评**全球最佳生态减贫案例**
