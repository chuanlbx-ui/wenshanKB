# 选题日历：文山年度节庆活动时间线

> 按月份排列，覆盖文山州全年重要节庆、文化活动、产业展会。每个条目标注时间、地点、关键看点、内容选题建议，为内容创作提供节拍器。

---

## 一月 / 二月（农历正月）

### 1. 苗族花山节

| 项目 | 内容 |
|------|------|
| **时间** | 农历正月初二至初八（公历 1 月下旬—2 月中旬），正月初三最隆重 |
| **地点** | 全州各县市均有分会场；主会场轮换，近年多在文山市熊朝忠拳王花山场、马关县城花山场 |
| **级别** | 国家级非物质文化遗产 |
| **关键看点** | 立花杆祭祀仪式；爬花杆竞技（顺爬、倒爬）；斗牛大赛；苗族盛装（蜡染+刺绣，被誉为"穿在身上的史书"）；芦笙舞；对歌相亲 |
| **选题建议** | ① 花山节深度纪实（仪式流程+人物故事）；② 苗族服饰文化解读（图案密码/迁徙记忆）；③ "爬花杆的人"——人物特稿；④ 花山节美食地图（配套年货市集） |

---

## 三月

### 2. 壮族三月三·花街节

| 项目 | 内容 |
|------|------|
| **时间** | 农历三月第一个巳日（公历 3 月下旬—4 月中旬），活动持续 3 天 |
| **地点** | 广南县者兔乡（核心/省级非遗）、丘北县普者黑周边村寨、文山市德厚镇 |
| **关键看点** | 祭竜仪式（竜林祈福，壮族自然崇拜核心仪式）；千人歌圩对唱；碰彩蛋、抛绣球、抢花炮；五色花糯米饭长街宴；坡芽情歌（国家级非遗）展演 |
| **选题建议** | ① "壮族三月三与竜林信仰"——文化深读；② 坡芽歌书：81 个符号唱出的爱情史诗；③ 广南者兔三月三实地纪实（图文/视频向）；④ 花糯米饭的植物染色密码 |

---

## 四月

### 3. 文山州庆

| 项目 | 内容 |
|------|------|
| **时间** | 4 月 1 日（1958 年文山壮族苗族自治州成立），法定假期 |
| **地点** | 全州各地，文山市为主 |
| **关键看点** | 州庆文艺汇演；民族团结进步主题活动；各县市特色文化方阵巡游 |
| **选题建议** | ① "文山建州 XX 年"——年度盘点/成就综述；② "一个外地人眼中的文山州庆"——体验视角；③ 文山 8 县各自最值得骄傲的一件事 |

---

## 五月 / 六月（农历五月）

### 4. 闹兜阳节

| 项目 | 内容 |
|------|------|
| **时间** | 农历五月初五，法定假期（端午节假期基础上顺延 3 天，约公历 5 月底—6 月） |
| **地点** | 马关县小坝子镇老懂寨（发源地+核心区），全州苗族聚居区有分会场 |
| **级别** | 云南省级非物质文化遗产，申报国家级 |
| **关键看点** | 招兜阳仪式（九方招魂棚、芦笙守夜）；"选兜阳"——推举当年好人能人游村；地摊长街宴；斗牛、打陀螺、纺麻刺绣比赛；儿童学芦笙、绘彩蛋 |
| **选题建议** | ① "兜阳是谁"——苗族先贤传说解读；② 闹兜阳 vs 端午节：文山苗族的双重节日；③ 老懂寨实地探访：闹兜阳故乡三日；④ "选兜阳"——一个村寨如何评选"好人能人" |

---

## 六月 — 九月

### 5. 普者黑荷花节 & 花脸节

| 项目 | 内容 |
|------|------|
| **时间** | 荷花节：6 月—9 月（万亩荷花盛花期）；花脸节：7 月中旬—8 月中旬 |
| **地点** | 丘北县普者黑景区，仙人洞村主会场 |
| **关键看点** | 万亩荷花+喀斯特峰丛（世界级景观）；花脸节"抹黑祈福"狂欢（起源：祭祀山神）；柳叶小舟打水仗；彝族弦子舞、篝火晚会；荷花宴（季节限定） |
| **选题建议** | ① 普者黑荷花季全攻略（拍摄点/路线/美食/民宿）；② 花脸节现场：被抹得越黑越幸福；③ 荷花宴食谱揭秘（荷花煎蛋/荷花鱼/炸荷花）；④ "荷塘边的十二时辰"——普者黑一日 Vlog 脚本 |

## 七月 / 八月（农历六月）

### 6. 彝族火把节

| 项目 | 内容 |
|------|------|
| **时间** | 农历六月二十四前后（公历 7 月下旬—8 月上旬） |
| **地点** | 丘北县普者黑景区火把广场（最知名）、文山市七花广场/七都古镇、砚山县民族文化中心、广南县民族体育广场 |
| **级别** | 国家级非物质文化遗产 |
| **关键看点** | 毕摩祭火取火仪式；万人火把巡游；篝火达体舞狂欢；斗牛/斗羊/摔跤；彝族选美；长街宴（坨坨肉、连皮牛肉汤锅） |
| **选题建议** | ① 火把节为什么是彝族"春节"——文化溯源；② 火把节两日摄影攻略；③ "与彝家人一起过火把节"——深度体验游记；④ 火把节美食特辑：坨坨肉/烤乳猪/苦荞粑粑 |

## 九月 / 十月

### 7. 丘北辣椒节

| 项目 | 内容 |
|------|------|
| **时间** | 每年秋季（辣椒收获季，约 9 月—10 月） |
| **地点** | 丘北县 |
| **关键看点** | 辣椒采摘体验；辣椒评选大赛（比辣度/比品相）；辣椒主题美食展；辣椒制品展销（辣椒酱/蘸水/辣椒酥）；晒椒场面（摄影绝佳素材） |
| **选题建议** | ① "中国辣椒之乡"丘北辣椒凭什么香；② 从枝头到餐桌：一根丘北辣椒的旅程；③ 丘北辣椒+普者黑秋色 = 最配的秋游路线；④ 辣椒蘸水文山流派图鉴 |

### 8. 广南句町文化旅游节

| 项目 | 内容 |
|------|------|
| **时间** | 秋季（通常 9 月—10 月） |
| **地点** | 广南县 |
| **关键看点** | 句町古国文化展演；侬氏土司衙署开放活动；坝美"世外桃源"主题体验；壮族铜鼓舞、纸马舞；八宝米丰收季 |
| **选题建议** | ① 句町古国：被遗忘的西南古文明；② 侬氏土司衙署——云南保存最完整的土司建筑；③ 广南秋收：坝美+八宝米+句町文化三日深度 |

## 十月 / 十一月（农历十月）

### 9. 瑶族盘王节

| 项目 | 内容 |
|------|------|
| **时间** | 农历十月十六日（公历 10 月下旬—11 月），活动持续 3 天 |
| **地点** | 全州瑶族聚居区轮办：麻栗坡县猛硐瑶族乡、富宁县洞波瑶族乡、广南县八宝镇、文山市七都古镇等 |
| **级别** | 国家级非物质文化遗产 |
| **关键看点** | 盘王祭祀大典（设坛、师公吟唱《盘王歌》、跳盘王舞）；长鼓舞/铜鼓舞/黄泥鼓舞；瑶族刺绣/蜡染/织锦非遗展；青年对歌交友；"还盘王愿"仪式 |
| **选题建议** | ① 《盘王歌》——瑶族的创世史诗；② 瑶族盘王节 vs 壮族三月三 vs 苗族花山节：文山三大民族节庆对比；③ 走进麻栗坡猛硐瑶族乡——边境线上的盘王节；④ 瑶族服饰里的文化密码 |

## 十一月

### 10. 文山三七节 & 三七产业博览会

| 项目 | 内容 |
|------|------|
| **时间** | 11 月 7 日（法定节日），活动持续约一周 |
| **地点** | 文山市（盘龙体育场、三七国际交易中心等） |
| **关键看点** | 开幕式主题晚会（数字虚拟人"三七姑娘"等科技元素）；三七产业博览会（全产业链展销）；"七王争霸"评选（最大/最优三七）；三七美食节；招商引资签约仪式；康养旅游线路推介 |
| **选题建议** | ① 三七节的"七件事"——逛展/品美食/看晚会/买特产/学辨真伪；② 一颗三七的全球旅程（从文山田间到国际药房）；③ "七王争霸"背后的三七江湖；④ 三七产业年度趋势盘点（数据向） |
