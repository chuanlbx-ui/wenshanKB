# 来源质量分级标准

> 版本：1.0
> 生效日期：2026-05-25
> 适用范围：D:\WenShanKB 知识库所有笔记的数据溯源

---

## 一、分级体系

### S级 — 原始统计公报/政府白皮书/统计局原始页面

| 要素 | 标准 |
|------|------|
| 定义 | 各级政府统计部门、行业主管部门公开发布的原始统计数据、白皮书、公报原文 |
| 典型来源 | 文山州国民经济和社会发展统计公报、云南省统计年鉴、国家统计局原始发布页、政府白皮书、发改局/工信局/商务局正式公文中引用的原始数据 |
| 核验方式 | **自动核验**：直接通过 URL 访问原始页面确认数据一致性 |
| 置信度标记 | **高**（可直接引用） |
| 示例 | `文山州2025年国民经济和社会发展统计公报`（文山州统计局） |

**S级子类细化**：

| 子级 | 说明 | 示例 |
|------|------|------|
| S-Primary | 统计公报/年报原文 | 文山州2025年统计公报 |
| S-Registry | 政府公示/备案原文 | 文旅部5A级景区名录、国家级非遗名录 |
| S-Plan | 政府正式发布的规划文件 | 文山州"十四五"规划纲要、碳达峰实施方案 |

---

### A级 — 政府部门官网新闻/正式发布

| 要素 | 标准 |
|------|------|
| 定义 | 各级政府官网发布的新闻稿、工作动态、政策解读等，经官方编辑审核发布 |
| 典型来源 | 文山州人民政府官网新闻、云南省发改委/工信厅/商务厅网站、国家部委官网新闻稿 |
| 核验方式 | 访问原始URL确认内容一致后标记"已核验" |
| 置信度标记 | **高**（可直接引用） |
| 示例 | `文山州能源局公开数据`（文山州人民政府官网） |

**注意事项**：
- A级来源中的**转载内容**降级为B级处理
- A级来源中的数据若引用自S级来源，需**追溯至S级原始页面**

---

### B级 — 央媒/省媒正式报道

| 要素 | 标准 |
|------|------|
| 定义 | 具有正规新闻采编资质的主流媒体发布的原创报道 |
| 典型来源 | 人民日报、新华社、光明日报、经济日报、云南日报、云南广播电视台 |
| 核验方式 | **需交叉验证**：同一数据至少找到2个独立B级以上来源方可标记"已核验"；仅有单一B级来源标记"待核验" |
| 置信度标记 | **中**（引用时须加"据XX报道"限定语） |
| 示例 | `parkworld.net发表的文章`（仅标记为B级来源时需交叉验证） |

---

### C级 — 其他媒体转载/行业网站/企业官网

| 要素 | 标准 |
|------|------|
| 定义 | 非央媒省媒的媒体转载、行业门户、企业官网等 |
| 典型来源 | 新浪/搜狐/网易等门户转载、行业自媒体平台、企业官网新闻 |
| 核验方式 | **仅作参考**：不纳入核验队列，仅作为数据线索 |
| 置信度标记 | **低**（仅作参考，禁止直接引用） |
| 示例 | `parkworld.net`（如无法确认其采编资质） |

---

### D级 — 无明确来源/推测/自媒体

| 要素 | 标准 |
|------|------|
| 定义 | 无法追溯到具体发布机构的推测性内容、个人自媒体发布、知识库内部循环引用 |
| 典型来源 | 微信公众号个人号、微博、知识库内部合成笔记 |
| 核验方式 | **禁止使用**：标记为"待核验"，不参与任何数据引用 |
| 置信度标记 | **不可用**（除非后续补充S/A级来源） |
| 示例 | 知识库内部合成笔记中标注"—"的URL字段（约45%数据点） |

## 二、来源降级规则

### 2.1 基本降级原则

| 触发条件 | 降级方式 | 示例 |
|----------|---------|------|
| 同一数据存在多源 | 取**最低等级**来源 | 铝土矿90%数据：既有行业报告（C级）也有政府公报未提及→降级为C级 |
| 但S/A级来源已覆盖 | **优先采用S/A级**，忽略低级别来源 | GDP数据以统计公报为准，忽略媒体转载 |
| A级来源为转载非原创 | 标注转载，数据降级为B级 | 省厅官网转载云南日报文章 |
| URL链接失效 | 来源等级不变，标注"链接已失效，待修复" | — |
| 来源为知识库内部合成笔记 | 标记为D级，需追溯至原始来源 | 人物故事集、合成报告中的无URL数据 |

### 2.2 品质加分/扣分规则

| 触发条件 | 效果 |
|----------|------|
| 原始URL可直接访问并验证 | + 保持原等级 |
| 数据附带了完整的统计口径说明 | + 保持原等级 |
| 数据来自行业公认的权威报告（如上市公司年报/行业协会白皮书）| 可评定为A级 |
| 同一数据在3个以上可靠来源中一致 | 可升至B级（从C级） |

## 三、URL存档规范

### 3.1 存档范围

所有 **S级 + A级** 来源的原始URL，必须存档到 `D:\WenShanKB
aw\sources\` 目录。

### 3.2 存档格式

| 字段 | 说明 | 示例 |
|------|------|------|
| 来源ID | 唯一标识 | SRC-001 |
| 原始URL | 完整链接 | https://www.tjj.wenshan.gov.cn/... |
| 存档文件名 | 保存为 .md 文件 | SRC-001_2025统计公报.md |
| 存档日期 | ISO 8601格式 | 2026-05-25 |
| 来源等级 | S/A/B/C/D | S-Primary |
| 引用该来源的数据ID | 逗号分隔 | DS-001, DS-003, DS-004 |

### 3.3 文件模板（`D:\WenShanKB
aw\sources\SRC-XXX_模板.md`）

```markdown
---
source_id: SRC-001
url: [原始URL]
grade: S-Primary
archive_date: 2026-05-25
referenced_by: [DS-001, DS-003]
---

# [来源标题]

**来源机构**：[发布机构全称]
**发布日期**：[发布日期]
**存档日期**：2026-05-25
**原始URL**：[链接]

## 摘录内容

[关键数据摘录或网页全文快照]

## 引用数据点

| 数据ID | 数据内容 | 值 |
|--------|---------|-----|
| DS-001 | GDP | 1632.64亿元 |
```

### 3.4 防失效策略

- 每次使用数据前，优先检查对应存档文件是否存在
- 存档文件应包含**全文快照或关键数据摘录**，防止链接失效后信息丢失
- `raw/sources/` 目录纳入 Git 版本管理（如有），确保可追溯
- 建议使用 Web Archive / archive.is 对 URL 进行互联网存档

## 四、B级以下来源的处理规则

### 4.1 使用限制

| 来源等级 | 在最终产出物中的使用限制 |
|----------|------------------------|
| S / A | **可直接引用**，引用格式：[来源: 文山州2025年统计公报, 文山州统计局] |
| B | **限定引用**，必须附加"据XX报道"或"根据XX发布的数据"，不得作为唯一论据 |
| B（单一来源） | 使用前必须标记"待核验"，等待补充第二来源 |
| C | **禁止在最终产出物中引用**，仅作为数据纠错和线索追踪 |
| D | **完全禁止引用**，所有D级数据必须在数据资产登记表中标注"待核验" |

### 4.2 限定语模板

| 来源等级 | 限定语模板 |
|----------|----------|
| S-Primary | `[来源: {年份}文山州国民经济和社会发展统计公报, 文山州统计局]` |
| S-Registry | `[来源: {名录名称}, {主管部委}]` |
| A | `[来源: {发布机构}, {发布日期}]` |
| B | `据{媒体名}报道, {数据内容}` |
| B（单一源） | `⚠️ 以下数据仅据{来源}单源报道，尚未完成交叉验证` |

| 等级 | 来源类型 | 可核验性 | 置信度 | 引用规则 | 存档要求 |
|------|---------|---------|--------|---------|---------|
