# 文蒙铁路团坡隧道安全贯通

> 来源：文山市融媒/文山发布 | 2026-06-03

2026年5月30日，文山至蒙自铁路**团坡隧道安全贯通**，这是文蒙铁路贯通的**第二条隧道**，标志着西南沿边铁路建设迈出关键一步。

## 一、隧道概况

| 指标 | 数值 |
|------|------|
| 位置 | 文山市薄竹镇附近 |
| 全长 | **443 米** |
| 地质条件 | 岩溶核心区，地层以灰岩为主 |
| 施工方式 | 全工序机械化施工 |

建设团队利用**三臂凿岩台车**等大型自动化智能化工装，在复杂岩溶地质条件下实现安全精准贯通。

## 二、工程进展

### 固定资产投资
截至贯通日，文蒙铁路文山段累计完成固定资产投资 **29.2565 亿元**。

### 站前Ⅰ标

| 指标 | 累计完成 |
|------|---------|
| 开工点位 | **58 个** |
| 隧道暗洞 | **2,117.4 米** |
| 桥梁桩基 | **2,264 根** |
| 承台 | **233 个** |
| 墩台身 | **170.2 个** |

### 站前Ⅱ标（文山段）

| 指标 | 累计完成 |
|------|---------|
| 开工点位 | **15 个** |
| 隧道暗洞 | **4,497.4 米** |
| 桥梁桩基 | **535 根** |

## 三、项目背景

| 指标 | 数值 |
|------|------|
| 线路全长 | 约 **116 公里** |
| 设计时速 | **160 公里** |
| 定位 | 中长期铁路网规划西南沿边铁路重要项目 |
| 站点数量 | 设 **7 座** 车站 |
| 开工时间 | 2024年1月30日 |
| 计划工期 | 4.5 年，**2028 年建成通车** |

全线建成后将完善滇东南路网结构，构建滇中和滇东南城市群之间的便捷运输通道。
