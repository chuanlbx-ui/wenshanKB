# 文靖铁路通过关键评审

> 2026年5月，文山至靖西铁路项目建议书评估暨行业评审会在昆明召开，标志着项目前期工作取得关键突破，为可研批复及开工建设奠定基础。

## 一、项目定位

文靖铁路是国家"十四五"规划重点建设项目，已列入国家《中长期铁路网规划》，是云桂沿边铁路的重要组成部分，也是继南昆铁路、南昆客专之后连接云南与广西的第三条铁路干线。

## 二、工程规模

| 指标 | 数值 |
|------|------|
| 线路全长 | **282 公里**（云南段约202km、广西段约80km） |
| 总投资 | **约 336 亿元** |
| 技术标准 | **国铁 I 级**，客货共线单线电力铁路 |
| 设计时速 | **160 km/h** |
| 计划工期 | **5 年** |

## 三、线路走向

起点广西靖西市既有靖西车站 → 那坡县 → 云南富宁县 → 广南县 → 砚山县 → 文山市，接入在建**文蒙铁路文山站**。

## 四、车站设置

全线共设 **24 座车站**：

| 类型 | 数量 | 说明 |
|------|------|------|
| 新建车站 | 21 座 | — |
| 改建既有站 | 2 座 | 富宁站、靖西站 |
| 利用在建站 | 1 座 | 文蒙铁路文山站 |

## 五、最新进展

| 节点 | 状态 |
|------|------|
| 可研报告 | 已完成鉴修审查 |
| 初步设计 | 通过预审查 |
| 地质勘察 | 定测阶段深孔钻探全面展开 |
| 建设用地预审 | 云南段和广西段已获自然资源部批复 |
| 行业评审会 | **2026年5月**通过（中铁第四勘察设计院联合国家铁路局规划标准院召开） |

## 六、通车时间

按5年工期推算，项目有望在 **2032 年**建成通车。

## 七、战略意义

填补西南沿边地区路网空白，构建云桂沿边铁路战略骨干通道，与文蒙铁路共同构成文山州东西向铁路骨架。
