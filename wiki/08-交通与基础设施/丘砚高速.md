# 丘砚高速（丘北—砚山）

> 丘北至砚山高速公路是文山州连接丘北县与砚山县的重要骨架交通干线，全长63.78公里，概算总投资102.77亿元，采用双向四车道标准，设计时速100公里。计划2027年12月建成通车。

## 一、项目概况

| 指标 | 数值 |
|------|------|
| 线路全长 | **63.78 公里** |
| 概算总投资 | **102.77 亿元** |
| 设计标准 | 双向四车道 |
| 设计时速 | 100 公里/小时 |
| 起点 | 丘北县双龙营镇东侧坝子村（衔接师宗—丘北高速） |
| 终点 | 砚山县城北部（连通蒙自—文山—砚山高速） |
| 计划通车 | **2027 年 12 月** |

## 二、工程进度（截至 2026 年 5 月）

| 工程项 | 完成率 |
|--------|--------|
| 路基工程 | 64.7% |
| 桥梁工程 | 22.91% |
| 隧道工程 | 20.22% |
| 总体形象进度 | **24%** |
| 固贷授信 | 已批复 **58 亿元** |

## 三、重点控制性工程

### 花桂隧道（中铁二十四局承建）

| 指标 | 数值 |
|------|------|
| 隧道左洞 | 长 1.17 公里 |
| 隧道右洞 | 长 1.2 公里 |
| 当前进洞 | 300 米 |
| 日均掘进 | 2—3 米 |
| 预计贯通 | **2027 年 3 月底** |

### 最高八级挖方边坡

| 指标 | 数值 |
|------|------|
| 总开挖高度 | **64 米** |
| 已完成 | 六级边坡开挖 |
| 剩余 | 两级边坡，预计2026年6月底完成开挖 |

## 四、项目进展时间线

| 时间 | 里程碑 |
|------|--------|
| 2025年10月 | 获批固贷授信58亿元，组织复工准备 |
| 2025年底 | 有序组织项目复工 |
| 2026年5月 | 全线大干快上，花桂隧道进洞300米 |
| 2027年3月 | 花桂隧道预计贯通 |
| 2027年12月 | 计划建成通车 |

## 五、战略意义

- 连接丘北县与砚山县的骨架交通干线
- 完善文山州路网布局，畅通内外交通
- 衔接师丘高速（北向）与蒙文砚高速（南向），形成南北纵贯通道
- 服务丘北普者黑旅游经济带
