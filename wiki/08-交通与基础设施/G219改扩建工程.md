# G219 改扩建工程（马关—西畴段）

> 国道G219线云南马关（新桥河）至西畴（撑腰岩）段改扩建工程，全长117.2公里，采用双车道二级公路标准，是完善滇东南沿边公路网的重点基础设施项目。2026年6月进入环评审批公示阶段。

## 一、项目概况

| 指标 | 数值 |
|------|------|
| 项目名称 | 国道G219线云南马关（新桥河）—西畴（撑腰岩）段改扩建 |
| 建设单位 | 云南省普通国道公路建设指挥部 |
| 建设地点 | 马关县古林箐乡、篾厂乡、八寨镇、大栗树乡、坡脚镇、马白镇 |
| 建设性质 | 新建（部分沿既有道路改建） |
| 路线全长 | **117.200 公里** |
| 新建段 | 46.251 公里 |
| 沿既有道路改建 | 69.299 公里 |
| 完全沿用 | 1.65 公里 |

## 二、技术标准

| 指标 | 数值 |
|------|------|
| 公路等级 | 双车道二级公路 |
| 路基宽度 | 8.5m — 12m |
| 设计速度 | 40km/h — 60km/h |
| 路面类型 | 沥青混凝土路面 |

## 三、用地规模

| 类型 | 面积 |
|------|------|
| 总占地 | 362.80 公顷 |
| 永久占地 | 293.56 公顷 |
| 临时占地 | 69.24 公顷 |

## 四、项目进展

| 时间 | 里程碑 |
|------|--------|
| 2026年6月3日—9日 | 环评报告书拟审批公示（5个工作日） |
| 2026年6月（预计） | 环评获批后进入施工准备阶段 |

## 五、战略意义

- 完善滇东南沿边国道网络，G219是连接云南、广西的重要沿边通道
- 改善马关县多个乡镇的交通条件，惠及古林箐、篾厂、八寨、大栗树、坡脚等乡镇
- 支撑口岸经济和边境贸易发展（马关县有都龙口岸）
- 服务兴边富民行动和乡村振兴
