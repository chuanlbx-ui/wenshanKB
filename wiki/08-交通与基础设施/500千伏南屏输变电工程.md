# 500千伏南屏输变电工程

> 500千伏南屏输变电工程已列入国家"十四五"电力发展规划，总投资11.58亿元，站址位于广南县。建成后将满足文山州东部负荷发展需求，优化220千伏电网结构，提升供电可靠性，为绿色铝产业和园区经济提供坚实电力保障。

## 一、项目概况

| 指标 | 数值 |
|------|------|
| 项目名称 | 500千伏南屏输变电工程 |
| 项目代码 | 2506-530000-2026-04-01-127856 |
| 项目法人 | 云南电网有限责任公司 |
| 总投资 | **11.58 亿元** |
| 资本金比例 | 20%（云南电网自筹） |
| 其余资金 | 银行贷款 |
| 建设地点 | 站址：广南县；线路途经广南县、砚山县、富宁县 |
| 批复时间 | 2026年5月21日 |

## 二、建设内容

### 变电站

| 项目 | 规格 |
|------|------|
| 主变压器 | 2 × 750 MVA |
| 500kV出线 | 2回 |
| 220kV出线 | 5回 |
| 低压侧配置 | 每组主变配2×60Mvar电容器 + 1×60Mvar电抗器 |

### 线路工程

| 线路 | 长度 |
|------|------|
| 500kV天星—南屏 | 约 69 公里 |
| 500kV南屏—富宁换流站 | 约 80.5 公里 |

### 扩建工程

- 500kV天星站：扩建1个500kV出线间隔
- 500kV富宁换流站：扩建1个500kV出线间隔
- 配套通信光缆及二次系统

## 三、建设目的

- 满足文山州**东部负荷发展**需求（砚山、广南、富宁工业园区）
- 优化 **220千伏电网** 结构
- 提升**供电可靠性**
- 支撑绿色铝产业、园区经济持续增长的电力需求
- 保障零碳产业园区绿电供应

## 四、列入2026年重大项目

2026年文山州政府工作报告明确"开工建设500千伏南屏变"，列入年度亿元以上重大项目166个之一。
