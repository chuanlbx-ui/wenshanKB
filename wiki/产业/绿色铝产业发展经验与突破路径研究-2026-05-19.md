# 文山州绿色铝产业发展经验成就及千亿元产值之后的突破路径研究

> 来源：中共文山州委政策研究室
> 发布日期：2026-05-19

## 产业现状概览

截至2025年底：

| 指标 | 数值 | 占全省/全国 |
|---|---|---|
| 电解铝产能 | 343万吨 | 占云南省51%，全国近1/10 |
| 工业产值 | 1016.07亿元 | 占全省绿色铝总产值50.7% |
| 经济增长贡献率 | 超24% | — |
| 电解铝产量 | 314.71万吨 | — |
| 铝合金产量 | 220.05万吨 | 占全省63.2% |

文山已成为"南铝北移"标志区、**全国绿色铝产能最大州市**，以及云南打造"中国绿色铝谷"的核心区。

## 一、五大发展经验

### 1. 抢抓政策机遇，紧跟产业转移步伐
- 2006年引进云铝140万吨氧化铝项目
- 2018年河南神火90万吨电解铝一体化项目落地富宁
- 2019年山东魏桥203万吨电解铝产能落地砚山

### 2. 战略引领与政策赋能双轮驱动
- 制定《文山州建设"中国绿色铝谷"战役攻坚行动方案》
- "3+N"发展思路，"一引擎、双基地"布局
- 组建工作专班，设立产业发展基金，实施"一把手招商"

### 3. 链主牵引与集群发展同频共振
- "链长+链主"协同机制（宏泰、神火为链主）
- 累计签约项目53个、建成投产33个
- 铝加工设计产能从474万吨增至690万吨
- 成立全省首家铝业协会

### 4. 要素保障与服务优化
- 最大供电能力775万千瓦，清洁能源装机占比99.8%以上
- 电解铝绿电使用比例超80%
- 动态储备熟地4300亩
- 宏泰项目创造"65天签约到开工、195天开工到投产"的行业纪录

### 5. 科技创新与人才培育
- 三个园区集聚院士工作站6家、省级工程研究中心2家
- 链主企业拥有专利201项
- 宏泰600kA电解铝生产线技术达国际领先水平
- 神火5G智慧工厂获"绽放杯"标杆赛金奖

## 二、五大核心成就

| 成就领域 | 关键数据 |
|---|---|
| 产业规模 | 产值1016.07亿元，增长22%，首个千亿级产业 |
| 产业链延伸 | 铝加工产能407.4万吨，合金化率69.92%，度电产值2.34元 |
| 绿色低碳 | 累计生产绿电铝755万吨，减排CO₂超8000万吨 |
| 基础设施 | 累计投入414.82亿元，91个基础设施项目，标准化厂房220.52万平方米 |
| 就业带动 | 直接就业1.3万余人，间接就业2.4万余人 |

**县市带动效应：**
- 砚山县铝产业产值：561.04亿元
- 富宁县绿色铝产业产值：295.76亿元（占全县规上工业93.1%）
- 文山市绿色铝产业产值：159.27亿元

## 三、面临的五大挑战

| 挑战 | 说明 |
|---|---|
| 贸易摩擦 | 美国铝关税提至50%，欧盟碳关税全面实施 |
| 消费疲软 | 房地产深度调整拖累铝消费 |
| 下游投资压力 | 铝棒加工费转负，企业利润压缩 |
| 产业协同不足 | 汽车、新能源投资集中于沿海，云南市场需求不足 |
| 物流成本高 | 文山交通滞后，铝加工产品运输成本翻倍 |

## 四、五大发展机遇

1. **国家战略叠加**：契合"双碳"、产业转移、区域协调发展政策
2. **绿色低碳市场**：新能源汽车单车用铝200-350kg，需求年均增速超30%；再生铝2027年占比目标30%以上
3. **科技创新**：5G+AI与铝产业深度融合，碳足迹核算技术发展
4. **基础设施完善**：文蒙铁路开工、富宁港与平陆运河同步投运
5. **区位开放**：面向南亚东南亚辐射中心，RCEP红利释放

## 五、"十五五"突破路径

### 总体目标
构建"1+1+3+N"现代化铝产业体系，铝合金转化率达100%，铝材占比70%以上，**2030年综合产值突破1800亿元**。

### 六大重点举措

**1. 扩大产能规模**

| 产能类型 | 2030年目标 |
|---|---|
| 电解铝产能 | 突破400万吨 |
| 再生铝产能 | 突破300万吨 |
| 铝加工产能 | 达到设计标准 |
| 碳素制品产能 | 突破350万吨 |

**2. 精准招商**
- 每年招引全产业链项目10个以上
- 2030年涉铝企业超100家
- 每年新增规上企业8-10家

**3. 打造滇桂铝先进制造业集群**
- 砚山园区：千亿元级高端精深加工
- 富宁园区：跨境合作，产值突破500亿元
- 马塘园区：产城融合，产值突破300亿元
- 集群综合产值突破2500亿元

**4. 科技创新**
- 从"论吨卖"到"按克卖"
- 赤泥综合利用率提升至30%以上
- 建立"文山绿色铝"区域公共品牌

**5. 绿电绿证**
- 绿电消费比例稳定达85%以上
- 2027年底前实现碳标签全覆盖

**6. 要素保障**
- 物流成本较2025年下降10%以上
- 2030年发电装机容量达1000万千瓦以上
- "十五五"培养技能人才1000人以上
