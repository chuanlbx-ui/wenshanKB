# 文山州有色金属产业高质量发展

> 来源：文山州工业和信息化局 | 提案答复（文工信函〔2026〕1号） | 发布日期：2026-05-21
> 关于推动文山州有色金属产业高质量发展的提案（第13050174号）办理答复

## 一、产业总览（2025年）

| 指标 | 数值 |
|------|------|
| 有色金属产业产值 | **1149.6 亿元** |
| 同比增长 | **+21.6%** |
| 占全州规上工业总产值 | **74.7%** |
| 规上有色金属企业 | **42 户**（矿采选业 11 户，冶炼和压延加工业 31 户） |

## 二、绿色铝产业链

| 指标 | 数值 |
|------|------|
| 工业产值 | **1016.07 亿元**（同比 +22%） |
| 氧化铝产量 | 131.61 万吨（同比 -6%） |
| 电解铝产量 | 314.71 万吨（同比 +13.1%） |
| 铝合金产量 | 220.05 万吨（同比 +29.3%） |
| 合金化率 | **69.92%**（提升 8.77 个百分点） |
| 度电产值 | **2.34 元**（同比 +0.19 元） |
| 铝土矿保供州内 | 100.87 万吨 |
| 铝土矿保供州外 | 250.13 万吨 |
| 签约绿色铝项目 | **44 个** |
| 规划铝加工产能 | **418 万吨** |

**2025年新建投产项目**：剑涛、创格、宏砚、智铝、神火新材料等 8 个铝加工项目。
**2025年新开工项目**：信特、宝鼎、中铝（云南）绿色低碳铸轧等 3 个项目。

## 三、锌铟产业

| 指标 | 数值 |
|------|------|
| 锌矿已探明资源量 | **594.47 万吨**（金属吨） |
| 铟矿保有资源量 | **5889.07 吨**（金属吨） |

- 锌铟均与锡铜铅锌多金属矿伴生产出
- 2025年9月成立巩固提升锌铟产业小专班
- 策划推出有色金属招商项目 6 个
- 签约项目：马关万龙山200万吨/年采矿扩建、马关箐脚锌锡矿开发、马关大丫口铅锌多金属矿采选
- 推动云锡文山锌铟冶炼、西畴鑫泰锌业与越南北干省矿业公司接洽，拟从越南进口锌精矿
- "十五五"支撑项目 4 个，规划投资约 30 亿元，建成后增加产值 45 亿元以上

## 四、钨产业

- 全国罕见的高品位钨矿富集区，储量居全省首位
- 骨干企业：紫金集团、海隅钨业、麻栗坡浩宇固体废物处置有限公司
- 产能：年产 60 万吨采矿、年产 5000 吨仲钨酸铵
- 产业闭环：资源开采 → 初级加工 → 冶炼提取 → 固废利用
- 紫金集团：省级企业技术中心，授权专利 74 项
- 海隅钨业：新型环保钨冶炼工艺曾获国家技术发明奖二等奖

## 五、锑产业

| 指标 | 数值 |
|------|------|
| 锑矿资源量（金属量） | **13.65 万吨** |
| 锑锭产能 | **10,000 吨/年** |
| 三氧化二锑产能 | **13,200 吨/年** |
| 乙二醇锑产能 | **5,000 吨/年** |

- 骨干企业：云南木利锑业、云南文冶有色金属
- 完整产业链：锑矿 → 锑锭 → 三氧化二锑（锑白）→ 乙二醇锑

## 六、新兴增长点

**金属镓**：
- 依托全省唯一 140 万吨氧化铝产能，从氧化铝母液提取金属镓
- 投入 5000 万元，建成回收提取生产线 1 条
- 年产金属镓 **50 吨**

**铝基新材料**：
- 云南剑涛：年产 60 万吨绿色低碳高精铝板带 + 10 万吨循环铝合金锭，直供电池铝箔、电池壳坯料
- 云南宏砚新材料：供应电池箔与壳体坯料
- 目标：建设全国重要铝基新材料基地、面向南亚东南亚的铝贸易和制造中心

**钛产业**：已完成广南县、富宁县钛矿资源摸底调研，赴广东惠云钛业实地考察。

## 七、循环经济

| 指标 | 数值 |
|------|------|
| 签约再生铝项目 | **10 个** |
| 规划产能 | **190 万吨** |

## 八、政策与机制

- **矿产资源经济实施方案**：2026年2月12日印发《文山州壮大矿产资源经济实施方案》（2025—2035年），含 22 项工作清单
- **财政奖励**：2025年拨付 775 万元奖励以绿色铝为重点的制造业；推动有色金属（不含铝）纳入产值达标奖励范围
- **资金争取**：为剑涛项目争取超长期国债 1 亿元，为云锡锌铟等 12 个企业项目争取中央省级资金 4564 万元
- **驻企服务**：对 98 户重点企业派驻驻企服务员，建立包保帮扶制度
- **政银企对接**：供应链金融投放 21 笔 6211.7 万元，铝行业转型贷款 1.43 亿元
- **园区用地**：7 个省级开发区，总规划面积 124.74 平方公里，储备用地约 1.1 万亩
