# 民生社会事业

> 文山州"十四五"期间 80% 以上财政支出用于民生，城乡居民收入倍差从 2.5 倍缩小至 2.4 倍。2026 年政府工作报告将民生幸福列为七大重点之一。

## 一、居民收入

| 指标 | 2025 年增速 | 说明 |
|------|------------|------|
| 城镇居民人均可支配收入 | 增长 **3.2%** | — |
| 农村居民人均可支配收入 | 增长 **6.1%** | 增速高于城镇 |
| 城乡居民收入倍差 | **2.4 倍** | 五年间从 2.5 倍缩小至 2.4 倍 |

"十四五"期间，农村居民人均可支配收入年均增速高于全省平均水平。答好群众持续增收"五问"，牢牢守住了不发生规模性返贫底线。

## 二、就业创业

### 2025 年实绩

| 指标 | 数值 |
|------|------|
| 农村劳动力转移就业 | **127.6 万人** |
| 城镇新增就业 | **2.2 万人** |
| 扶持创业 | **2960 人** |

### 2026 年目标

| 指标 | 目标 |
|------|------|
| 职业技能培训 | **3 万人次以上** |
| 创业担保贷款 | **3.4 亿元以上** |
| 就业服务体系 | 完善"就在文山·业兴七乡"公共就业服务体系 |

- 构建企业"点单"、人社部门"派单"、培训机构"接单"的培训体系
- 建立新就业形态劳动争议"一站式"调解中心
- 全面盘活农村资源资产"招小商"

## 三、医疗卫生

### 现状指标

| 指标 | 数值 |
|------|------|
| 全国医改监测评价 | **A+ 等级** |
| 8县（市）消除麻风危害 | 全部达标 |
| 紧密型县域医共体建设 | 8县（市）全部达标 |
| 乡镇卫生院达到国家标准 | 120 所 |
| 每千人口执业（助理）医师 | **2.92 人** |
| 国家卫生县城（城市）达标 | 8县（市）全覆盖 |
| 创卫工作群众满意度 | 连续 2 年全国第一 |

### 县级以上人民医院

全部达到二级以上标准，实现"五大中心"全覆盖。

### 2026 年重点

- 建成全民健康信息"一张网"
- 推进"审方中心+中心药房"建设
- 建立全州会诊转诊中心
- 启动边境医联体建设（医疗惠边工程）
- 促进"中医药+康养"融合发展
- 加快州妇女儿童医院、州中医医院医养中心等项目建设

## 四、社会保障

| 领域 | 数据/措施 |
|------|-----------|
| 参保人数增幅 | 养老、医疗、工伤、失业保险人数比"十三五"末增加 **33.52 万人** |
| 养老服务 | 《文山州养老服务促进条例》启动实施 |
| 长期护理保险 | 推进落实职工长期护理保险制度 |
| 社会救助 | 标准稳步提高，"一老一小"得到有力守护 |
| 生育支持 | 落实生育支持政策，发展普惠托育和托幼一体化服务 |
| 殡葬改革 | 强化公益属性，取消降低收费项目 224 个，试点推行节地生态安葬 |

## 五、文体事业

- 全覆盖建成新时代文明实践中心，县乡村三级文化基础设施覆盖率 100%
- 电影《石漠花开》、民族舞剧《锦绣如歌》荣获国家级/省级多项荣誉
- "坡芽歌书"文化品牌影响力持续提升
- 城乡体育场地设施全覆盖，竞技体育和群众体育全面发展
- 2025 年承办第三届中国民族音乐传承周，文化惠民 1000 余场次，体育赛事 200 余场次
- 2026 年目标：办好"滇超"文山赛事，打造更多品牌赛事

## 六、十五五民生方向

- 深入实施"教育兴州"战略，加快各级教育扩优提质
- 加大就业创业支持力度，实现就业岗位更充足、收入增长更稳健
- 构建整合型医疗卫生服务体系，抓实"三医"协同发展和治理
- 健全覆盖全人群、全生命周期的人口服务体系
- 构建更加公平、更可持续、更有效率的社会保障体系
- 用心用情办好民生实事，让民生幸福更有成色、更有温度、更有质感
