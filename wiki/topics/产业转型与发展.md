# 产业转型与发展

> 文山州坚定不移实施"产业强州"战略，以资源经济、园区经济、口岸经济为路径，发展壮大绿色铝、三七中药材、高原特色现代农业、康养旅游旅居四大重点产业。2025 年 GDP 1,632.64 亿元，增长 5.3%，规上工业增加值增长 12.9%。

## 产业结构演变（2025）

| 产业   | 增加值      | 占 GDP | 增速    | 贡献率       |
| ---- | -------- | ----- | ----- | --------- |
| 第一产业 | 273.23 亿 | 16.7% | +2.6% | 8.2%      |
| 第二产业 | 587.23 亿 | 36.0% | +7.7% | **50.7%** |
| 第三产业 | 772.19 亿 | 47.3% | +4.6% | 41.1%     |

> 第二产业以 7.7% 增速领跑，贡献率过半。规上工业增加值增速 12.9%，高于全省 8.4 个百分点，是经济增长的核心引擎。

## 四大重点产业（"十五五"）

| 产业           | 2025 年规模                                  | "十五五"目标                        |
| ------------ | ----------------------------------------- | ------------------------------ |
| **绿色铝**      | 产值 1016~1021.8 亿元，电解铝产能 343 万吨（占全省 52.8%） | 2030 年综合产值突破 1800 亿，再生铝 150 万吨 |
| **三七中药材**    | 三七综合产值 220 亿，中药材综合产值 456 亿                | 向千亿级产业迈进，2028 年 GAP 全覆盖        |
| **高原特色现代农业** | 综合产值 1759 亿                               | 全国优质绿色农产品基地、粤港澳大湾区"菜篮子"        |
| **康养旅游旅居**   | 游客 5256.7 万人次，花费 425.7 亿                  | "三七之乡 康养文山"品牌                  |

## 三大发展路径

| 路径   | 核心内容                 | 标志                        |
| ---- | -------------------- | ------------------------- |
| 资源经济 | 铝土矿、三七、绿色能源等优势资源就地转化 | 绿色铝千亿产业                   |
| 园区经济 | 砚山千亿级园区为龙头，3 个新设园区协同 | 园区规上工业产值占全省 7.8%          |
| 口岸经济 | 融入西部陆海新通道，富宁港+沿边铁路   | 外贸进出口额 40.2 亿，53 户超 200 万 |

## 转型路径

```
传统农业州 → 资源经济摸底 → 园区经济集聚 → 口岸经济开放 → 绿色工业强州
```

### 关键节点

1. **资源摸底**：8 大类 62 大项两万余项资源资产全面清查
2. **园区集聚**：砚山产业园区跻身全省 7 个千亿级园区
3. **产业链延伸**：绿色铝"铝土矿—氧化铝—电解铝—精深加工—再生铝—综合利用"全链条；三七种植 → 精深加工 → 大健康产品
4. **绿色化**：清洁能源占比 99.8%，万元工业增加值能耗降 42.4%，绿电比例超 70%
5. **交通突破**：富宁港 2026 年建成，文蒙铁路 2028 年通车，文靖铁路 2030 年通车

## 科技创新

- 新增高新技术企业 4 户、科技型中小企业 50 户、专精特新企业 12 户
- 获批首个省级重点实验室和国家级三七工程研究中心
- 30 个院士专家工作站（三七领域）

## 挑战与瓶颈

| 挑战     | 说明                    |
| ------ | --------------------- |
| 合金化率偏低 | 绿色铝精深加工链条尚短           |
| 电价成本   | 优惠电价退出影响电解铝竞争力        |
| 服务业潜力  | 第三产业增速 4.6% 低于 GDP 整体 |
| 城乡差距   | 城乡收入比 2.40:1，仍需缩小     |

## 战略方向

- "3 个百万吨级"绿色铝加工集群（汽车零部件、建筑铝材、终端消费品）
- 三七产业向大健康、生物医药延伸
- 口岸经济从通道经济向落地加工转型
- 文旅产业从观光型向康养度假型升级
- 以旧换新带动消费 18.42 亿元
