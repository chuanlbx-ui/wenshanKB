# 乡村振兴

> 文山州持续巩固拓展脱贫攻坚成果同乡村振兴有效衔接，以产业帮扶、促农增收、人居环境提升为核心抓手，推进乡村全面振兴。2025 年计划投入衔接资金 20 亿元以上，目标农村常住居民人均可支配收入增长 7% 左右。

## 一、脱贫成果巩固

| 指标 | 数值 |
|------|------|
| 2025 年衔接资金投入 | 20 亿元以上 |
| 资金投向 | 产业帮扶 + 小型公益性基础设施建设 |
| 农村人均可支配收入增速（2025 年） | +6.1% |
| 城镇人均可支配收入增速（2025 年） | +3.2% |
| 城乡收入比 | 2.40:1 |

## 二、促农增收行动

- 实施"农村居民和脱贫人口持续增收三年行动"
- 目标：2025 年农村常住居民人均可支配收入增长 **7%** 左右
- 六郎城"五金增收"模式入选全省示范典型
  - 土地流转得租金
  - 务工投劳得薪金
  - 资产入股得股金
  - 产品销售得现金
  - 集体收益得奖金

## 三、产业帮扶

| 产业 | 联农带农成效 |
|------|-------------|
| 三七 | 带动 50 余万农户增收，GAP 基地建设吸纳大量农村劳动力 |
| 绿色铝 | 全产业链带动就业 2.4 万人 |
| 高原特色农业 | 综合产值 1759 亿元，覆盖全州广大农户 |
| 康养旅居 | 普者黑村人均纯收入达 5 万余元，石家湾户均创收 3 万余元 |

## 四、乡村人才培育

| 计划 | 目标 |
|------|------|
| 乡村 CEO 培养 | 2026 年培养 100 名 |
| 乡村运营人才培养 | 2026 年培养 1000 名 |
| 职业教育产教融合 | 建设产教融合园区，服务铝产业和三七产业人才需求 |

## 五、人居环境提升

- 推进农村"厕所革命"、生活垃圾和污水治理
- 建设宜居宜业和美乡村
- 较大人口规模自然村（组）通硬化路
