# 教育发展

> 文山州 2,211 所学校、83.3 万在校生，教育支出 90.77 亿元占一般公共预算 23.4%。九年义务教育巩固率 100.45%，以"一中"为龙头的教育改革成效显著。

## 规模总览

| 指标 | 数值 |
|------|------|
| 各级各类学校 | 2,211 所 |
| 在校生 | 832,682 人 |
| 专任教师 | 51,532 人 |
| 教育支出 | 90.77 亿元（占预算 23.4%） |

## 高等教育

- **文山学院**：唯一本科院校，9 个二级学院，62 个专业
- 三七医药学院全国独树一帜，直接服务千亿级三七产业
- 招收越南、老挝留学生，建设"国门大学"
- 正在推进迁建新校区

## 改革亮点

- "一中"龙头教育改革：600 分以上人数实现翻番
- 产教融合：三七医药学院、冶金与材料学院对接支柱产业
- 特殊教育：州特殊教育学校覆盖学前至职业高中 15 年

## 挑战

- 高等教育资源薄弱，仅 1 所本科院校
- 城乡教育资源不均衡
- 职业教育与产业需求匹配度待提升

## 2026年教育目标

| 目标 | 内容 |
|------|------|
| 新增城区学位 | **8800个** |
| 学前教育 | 全面落实免费学前教育政策 |
| 义务教育 | 加快推进优质均衡发展 |
| 县域普通高中 | 振兴行动 |
| 数字校园 | 整州推进，打造教育数字化转型省级示范 |
| 州职教园区 | 深化办学改革，构建职普融通、产教融合体系 |
| 文山学院 | 支持搬迁建设，三七大健康/绿色铝产业学院建成全省特色学院 |
| 教师队伍 | 建强校长、班主任、骨干教师三支队伍 |
