# 文山"十五五"规划纲要

> 2026年4月10日，文山州人民政府印发《文山壮族苗族自治州国民经济和社会发展第十五个五年规划纲要》。本文基于2026年6月"开局起步'十五五'"新闻发布会内容整理。

## "3+3"发展定位

| 定位 | 类型 |
|------|------|
| 民族团结进步示范区 | 三大定位 |
| 石漠化地区生态文明建设排头兵 | 三大定位 |
| 云南面向东南开放先行区 | 三大定位 |
| 新时代兴边富民示范区 | 三大示范区 |
| "绿色铝谷"核心区 | 三大示范区 |
| 三七康养旅居目的地 | 三大示范区 |

## 四项重点

产业、项目、干部、环境。

## 三大发展路径

- **资源经济**：立足铝土矿、三七等资源优势，推动资源就地转化
- **园区经济**：砚山千亿级园区为龙头，3个新设园区协同发展
- **口岸经济**：融入西部陆海新通道，富宁港+沿边铁路打通出海通道

## 四大重点产业

| 产业 | 2025 年规模 | "十五五"目标 |
|------|------------|-------------|
| 绿色铝 | 产值突破 1000 亿元（1016~1021.8 亿） | 2030 年综合产值突破 1800 亿，再生铝产能 150 万吨 |
| 三七中药材 | 三七综合产值 220 亿，中药材综合产值 456 亿 | 向千亿级产业迈进，2028 年 GAP 全覆盖 |
| 高原特色现代农业 | 综合产值 1759 亿 | 全国优质绿色农产品基地、粤港澳大湾区"菜篮子"基地 |
| 康养旅游旅居 | 接待游客 5256.7 万人次，花费 425.7 亿元 | "有一种叫云南的生活——三七之乡 康养文山" |

## 重大基础设施

| 项目 | 状态/目标 |
|------|----------|
| 富宁港 | 2026 年内建成，与广西平陆运河同步投用 |
| 文蒙铁路 | 2028 年建成通车 |
| 文靖铁路 | 2030 年建成通车 |
| 那兴高速 | 已通车 |

## 经济目标

- 产业投资占比 62%，民间投资占比 37.5%
- 93 个项目纳入省级重大项目清单
- 2845 个储备重点产业项目，总投资近万亿
- 与全国全省同步基本实现社会主义现代化取得决定性进展
