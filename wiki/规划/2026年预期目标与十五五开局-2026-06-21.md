# 文山州2026年预期目标与十五五开局

> 2026年是"十五五"开局之年，也是"3815"战略"八年大发展"承上启下的关键年。

---

## 一、2026年主要预期目标

| 指标 | 2026年目标 | 2025年实际 |
|------|-----------|-----------|
| GDP 增长 | 5% 以上 | +5.3%（1632.64亿） |
| 规模以上工业增加值增长 | 10% 以上 | +12.9% |
| 固定资产投资增速 | 与上年持平 | 537.69亿（连续三年下降） |
| 社会消费品零售总额增长 | 4% | +3.7% |
| 地方一般公共预算收入增长 | 1% | +6.8% |
| 常住人口城镇化率 | 41.8% | 41.19% |
| 外贸进出口总额增长 | 10% | — |
| 全员劳动生产率 | 8.9万元/人 | — |
| 城镇新增就业 | 1.8万人 | 2.20万人 |
| 粮食产量 | 179.5万吨 | 178.98万吨 |

## 二、十五五规划核心指标（2025→2030）

### 经济发展

| 指标 | 2025年 | 2030年目标 |
|------|--------|-----------|
| GDP增长 | 5%左右 | 5%以上（年均） |
| 常住人口城镇化率 | 41% | 45% 左右 |
| 工业增加值占GDP比重 | 25% | 27% 左右 |
| 全员劳动生产率增长 | 6%左右 | 5%左右（年均） |
| 农业加工产值与农业总产值比 | 2.3:1 | 2.8:1 |
| 进出口贸易总额 | 39亿元 | 50亿元 |

### 创新驱动

| 指标 | 2025年 | 2030年目标 |
|------|--------|-----------|
| 研发经费投入增长 | 5% | 9%（年均） |
| 每万人口高价值发明专利 | 0.34件 | 0.5件 |
| 数字经济核心产业营收 | 50亿元 | 67亿元 |

### 绿色低碳

| 指标 | 2025年 | 2030年目标 |
|------|--------|-----------|
| 非化石能源占能源消费比重 | 71.3% | 完成省下达目标 |
| 森林覆盖率 | 40% | 45% |
| PM2.5浓度 | 23μg/m³ | 完成省下达目标 |
| 优良水体比例 | 100% | 完成省下达目标 |
