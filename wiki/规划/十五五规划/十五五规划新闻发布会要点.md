## 核心经济目标

### 绿色铝产业

| 指标 | 数值 |
|------|------|
| 2025年电解铝产能 | 占全省 52.8%、全国 7.7% |
| **2025年绿色铝产值** | **突破 1000 亿元**（锚点值），首个千亿级产业 |
| **2030年综合产值目标** | **1800 亿元** |
| 2030年再生铝产能目标 | **150 万吨** |
| 产业体系 | "1+1+3+N"：电解铝 + 再生铝 + 3个铝产业园区 + N个企业项目 |

重点谋划项目：中铝 20 万吨绿色低碳铸轧、砚山智铝二期、云南神火超薄铝箔等。

### 三七产业

| 指标 | 数值 |
|------|------|
| **2025年综合产值** | **456 亿元**（锚点值） |
| **远景目标** | **向千亿级迈进** |
| 2025年在地面积 | 5.3 万亩（GAP 基地 7000 亩、林下三七 6000 亩） |
| 2026年 GAP 基地目标 | 扩大至 1.5 万亩 |
| 2028年目标 | 州内三七种植 GAP 全覆盖 |

- 七统一体系：种子种苗、规范种植、公开购销、产地加工、仓储物流、地标赋能
- 已从 80% 原料外销转变为 80% 在产地就地加工后供应市场
- 与 60 家企业合作（云南白药、华润三九等），30 个院士专家工作站
- 上市大健康产品 95 种，产值超 10 亿元

### 康养旅居

- 打造"有一种叫云南的生活——三七之乡 康养文山"品牌
- "旅居云南·好在文山"品牌 IP
- 推动"中医药+文旅"融合

## 交通与开放

- **富宁港年内建成**，与广西平陆运河同步投用
- 加快文蒙铁路、文靖铁路建设
- 争取曲靖至文山铁路突破
- 依托 3 个国家级口岸推进口岸经济
- 发展"边民互市 + 落地加工"
