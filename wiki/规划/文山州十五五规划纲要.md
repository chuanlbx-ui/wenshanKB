# 文山州"十五五"规划纲要

> 2026年4月10日，文山州人民政府正式印发《文山壮族苗族自治州国民经济和社会发展第十五个五年规划纲要》。
> 补充来源：2026年6月3日云南省"开局起步'十五五'"文山专场新闻发布会；2026年6月16日州政府办"决胜十四五 启航十五五"综合稿。

---

## 一、"十四五"收官成就

| 指标 | 成就 |
|------|------|
| GDP | 突破 1600 亿元（2025年 1632.64 亿） |
| 绿色铝 | 首个千亿级产业 |
| 三七 | 入选全国优势特色产业集群和中小企业特色产业集群 |
| 民族团结 | 全国民族团结进步示范州 |
| 双拥模范 | 全国双拥模范城 |
| 石漠化治理 | 第五届全球最佳生态减贫案例 |

## 三、四大重点产业目标

| 产业 | 2025年基准 | "十五五"末目标 |
|------|-----------|---------------|
| **绿色铝** | 产值突破 1000 亿，电解铝产能占全省 52.8%/全国 7.7% | 综合产值 **1800 亿元** |
| **三七中药材** | 综合产值 456 亿（+10.4%） | 向 **千亿级** 产业迈进 |
| **高原特色现代农业** | 农林牧渔业总产值 441.61 亿 | 总产值超 **550 亿元** |
| **康养旅游旅居** | 游客 5256.7 万人次，花费 425.7 亿 | 旅游总人数/花费年均各增 10%+ |

---

## 四、铝产业"1+1+3+N"体系

| 层级 | 内容 |
|------|------|
| 第1个"1" | 电解铝（343万吨产能，占全国7.7%） |
| 第2个"1" | 再生铝（获批产能 90 万吨） |
| "3" | 砚山、富宁、马关 3 个铝产业园区 |
| "N" | 中铝20万吨铸轧、砚山智铝二期、神火超薄铝箔、华疆再生铝等 N 个项目 |

> 路线：氧化铝全省唯一 → 电解铝延链补链 → 再生铝异军突起

## 六、重点战略方向

1. **产业强州**：资源经济 + 园区经济 + 口岸经济
2. **对外开放**：打造云南出海最近"黄金港口"（富宁港），融入西部陆海新通道
3. **生态文明**：石漠化综合治理，普者黑湖/盘龙河"一湖一河"保护
4. **兴边富民**：新时代兴边富民行动，边境幸福村升级版
