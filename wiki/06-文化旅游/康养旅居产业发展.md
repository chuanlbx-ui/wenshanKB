# 康养旅居产业发展

> 文山州紧扣"有一种叫云南的生活"定位，全力打造"旅居云南·好在文山"品牌，推动从"旅游过境地"向"旅居目的地"转型升级。"十五五"期间以五大主题为抓手，融合三七文化与民族非遗资源，培育千亿级康养旅居产业。

## 一、核心数据

### 2025年旅居数据

| 指标 | 数值 | 同比 |
|------|------|------|
| 旅居人数 | **16.63 万人** | **+49.82%** |

| 接待游客 | 5256.7 万人次 | +5.1% |
| 旅游花费 | 425.7 亿元 | +8.4% |

| 指标 | 数值 | 同比 |
|------|------|------|
| 接待游客 | 1643.45 万人次 | +5.72% |

| 旅居人数 | 8.77 万人 | +15.09% |

## 二、"旅居云南·好在文山"品牌

### 品牌矩阵

"有一种叫云南的生活——三七之乡 康养文山"为核心总品牌，"旅居云南·好在文山"为旅居IP，细分为五大子品牌：

| 子品牌 | 核心价值 | 代表载体 |
|--------|---------|---------|
| **好在文山** | 北回归线穿境而过的温润气候、宜居宜业的生活品质 | 全州气候资源 |
| **游在文山** | 普者黑和坝美的一汪碧水与千峰叠翠、"水陆空铁"立体交通网 | 普者黑、坝美 |
| **居在文山** | 田园旅居村、艺术家第二居所、康养社区 | 白脸山村、马洒村、龙华村等 |
| **养在文山** | 三七中医康养、温泉疗养、气候避暑、"中医药+文旅"融合 | 文山市、丘北 |
| **乐在文山** | 赛事经济、民族节庆、边境风情体验 | 老山国际春茶节、中越边民大联欢 |

### 五大核心业态

| 业态 | 说明 |
|------|------|
| **山水游** | 喀斯特地貌奇观，普者黑湖泊峰林、坝美世外桃源 |
| **红色游** | 老山精神、西畴精神现场教育基地，理想信念教育 |
| **边境游** | 438公里边境线风情，跨境旅游线路，"一日游两国" |
| **文化游** | 11个世居民族非遗，壮族三月三、苗族花山节等 |
| **赛事游** | 全省大学生军事技能大赛、老山国际春茶节等活动引流 |

### "旅游黄金三角"

丘北—广南—麻栗坡三县联动，形成"**丘北（普者黑山水）→ 广南（坝美田园）→ 麻栗坡（老山红色+边境）**"黄金旅游线路。

### "中医药+文旅"融合

从 **医、康、养、健、食、药、游、赏、学、购** 十个维度构建三七康养体验体系，让游客在每一个维度找到康养的兴奋点。

### 五大主题

| 主题 | 核心内容 | 代表区域 |
|------|---------|---------|
| **文旅体验** | 喀斯特山水、民族风情深度体验 | 普者黑、坝美 |
| **康养度假** | 三七中医康养、温泉疗养、气候避暑 | 文山市、丘北 |
| **边境风情** | 中越边境特色文化、跨境旅游 | 麻栗坡、马关、富宁 |
| **田园旅居** | 乡村田园生活、农事体验 | 八县市乡村 |
| **红色研学** | 老山精神、红色教育基地 | 麻栗坡老山 |

## 三、重点景区升级

| 景区 | 等级 | 方向 |
|------|------|------|

| **坝美** | 4A级 | 生态化提升，世外桃源品牌深化 |

## 四、三七康养融合

- 将**三七文化**融入康养旅居产品开发
- 发展中医药健康旅游、三七膳食体验
- 打造"三七之乡 康养文山"核心IP

## 五、民族非遗融合

- 11个世居民族非遗资源深度开发
- 壮族"三月三"、苗族花山节等节庆IP化
- 民族手工艺体验、非遗研学旅行

## 六、旅居8大业态

> 来源：文山州发展改革委《聚力旅居项目建设赋能文旅高质量发展》，文山发布网 2026-06-11

"十五五"期间，文山州明确打造8大旅居业态：

| 业态 | 方向 |
|------|------|
| **乡村旅居** | 田园旅居村、农事体验、联农带农 |
| **城镇旅居** | 城市康养社区、一刻钟便民生活圈 |
| **医养融合旅居** | 三七中医康养、"1+1+10"中医药康旅模式 |
| **度假旅居** | 普者黑、坝美等景区深度度假 |
| **文艺旅居** | 艺术家第二居所、民族非遗创作基地 |
| **边境旅居** | 中越边境风情、跨境文化体验 |
| **体育旅居** | 赛事经济、户外运动、滇超联赛 |
| **养老旅居** | 气候避寒、银发消费、康养社区 |

空间格局："一核引领（文山市）、三区联动（丘北-广南-麻栗坡）、两带串联（边境风情带、喀斯特山水带）"。

项目储备：策划旅居项目 **93 个**，总投资 **285.96 亿元**，涵盖景区提档升级、高品质酒店、智慧景区、旅居村建设等。

资金投入：累计争取中央预算内资金、地方政府专项债券资金 **14.35 亿余元**，带动行业投资 **46.33 亿余元**；全面推进砚山维摩石漠公园、西畴香坪山森林养生谷、麻栗坡国门水境旅游度假区、广南游客咨询服务中心、丘北猫猫冲生态湿地保护等旅居设施建设。

"十五五"发展目标：到 2030 年，旅游总人数、旅游总花费分别增长 **10% 以上**。

五项重点工程：康养旅居示范样板、创建一批旅游品牌、丰富研学旅游业态、大力发展边境旅游、培育数字化旅游业态、推动精品业态提质升级。

## 七、转型目标

| 维度 | 从 | 到 |
|------|-----|-----|
| 旅游定位 | 旅游过境地 | **旅居目的地** |
| 消费模式 | 观光打卡 | 深度停留、品质消费 |
| 产业形态 | 单一景区经济 | 康养旅居全产业链 |
| 品牌层级 | 区域品牌 | "旅居云南"核心支撑 |
