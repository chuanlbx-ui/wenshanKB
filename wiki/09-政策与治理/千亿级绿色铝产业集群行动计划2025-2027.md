# 千亿级绿色铝产业集群行动计划（2025—2027）

> 文山州已初步形成《打造千亿级绿色铝产业集群行动计划（2025—2027）》，明确未来三年在招商引资、科技创新及物流体系建设方面的具体扶持政策，旨在构建"1+1+3+N"铝产业体系。

## 一、产业发展现状（截至 2025 年）

| 指标 | 数值 |
|------|------|
| 电解铝建成产能 | 343 万吨（占全省 52.8%、全国 7.7%） |
| 铝加工产能 | 418 万吨 |
| 2025 年产值 | 1016.07 亿元（突破千亿大关） |
| 合金化率 | 69.92% |
| 就地转化率 | 70% 以上 |
| 龙头企业 | 魏桥集团、中铝集团、云南神火（均世界/中国 500 强） |

## 二、2025—2027 年核心目标

| 目标 | 数值 | 时间节点 |
|------|------|----------|
| 铝产业产值 | 突破 800 亿元（已达成）→ 持续增长 | 2025—2027 |
| 再生铝产能 | 100 万吨（已有）→ 150 万吨 | 2027 年 |
| 合金化率 | 69.92% → 持续提升 | 2027 年 |
| 就地转化率 | 70% → 80% 以上 | 2027 年 |
| 产业集群 | 新能源汽车零部件、建筑铝材、铝制消费品三大百万吨集群 | 2027 年 |

## 三、重点举措

### 招商引资
- 精准链式招商，围绕四大主导产业绘制招商图谱
- 重点引进铝箔、铝模板、汽车轻量化零部件、铝制消费品企业
- 砚山、富宁等划定专门产业承接园区，配套优惠用地和税收政策

### 科技创新
- 推进制造业数字化转型
- 文山铝业获评制造业数字化转型标杆
- 云南神火入选工业互联网试点示范
- 建设铝产业公共实训基地（全国唯一）

### 物流体系
- 富宁港 + 平陆运河：打通经珠江水系入海通道，绿色铝低碳水运出海
- 文蒙铁路（2028 年通车）+ 文靖铁路：东西向铁路动脉
- 构建"绿色铝—航运物流—内外市场"一体化发展格局

## 四、重大项目

| 项目 | 说明 |
|------|------|
| 云南宏泰（二期） | 持续推进产能释放和技术改造 |
| 中铝 20 万吨绿色低碳铸轧 | 精深加工标志性项目 |
| 云南神火超薄铝箔 | 高端铝箔材料 |
| 文山华疆再生铝循环利用 | 再生铝产业链闭环 |
| 云南文都 50 万吨新型预焙阳极 | 补链项目 |
| 信特等重点项目 | 50 万吨再生铝建成投产 |

## 五、营商环境

- 推行"局长坐诊接诉"机制：针对企业诉求现场办公
- 深化"小管委会+大公司"运营改革
- 全面推行工业项目"标准地"出让，企业拿地即开工

## 六、"十五五"远景

- 2030 年铝产业综合产值突破 **1800 亿元**
- 再生铝产能达 **150 万吨**
- 高端化、终端化、集群化、绿色化发展
- 打造面向东南亚的绿色铝产业集散地
