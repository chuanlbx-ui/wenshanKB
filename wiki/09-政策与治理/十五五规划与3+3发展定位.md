# 十五五规划与3+3发展定位

> "十五五"时期（2026—2030年），文山州确立"3+3"发展定位，突出"产业、项目、干部、环境"四项重点，以"产业强州"战略为牵引，全力构建现代化产业体系。

## 一、"3+3"发展定位

| 类型 | 定位 | 核心内涵 |
|------|------|---------|
| 三大定位 | **民族团结进步示范区** | 11个世居民族和谐共处，全国民族团结进步示范州 |
| 三大定位 | **石漠化地区生态文明建设排头兵** | 石漠化治理全球最佳生态减贫案例，喀斯特生态修复标杆 |
| 三大定位 | **云南面向东南开放先行区** | 438公里边境线接壤越南，融入西部陆海新通道 |
| 三大示范区/目的地 | **新时代兴边富民示范区** | 边境县市发展振兴，强边固防与民生改善并重 |
| 三大示范区/目的地 | **绿色铝谷核心区** | 全国绿色铝产能第一大州（占全省52.8%、全国7.7%） |
| 三大示范区/目的地 | **三七康养旅居目的地** | "三七之乡 康养文山"，从观光旅游向旅居目的地转型 |

## 二、四项重点

十五五期间突出抓好四项重点：

| 重点 | 说明 |
|------|------|
| **产业** | 以资源经济、园区经济、口岸经济为路径，壮大四大重点产业 |
| **项目** | 全州谋划储备重点产业类项目2845个，总投资接近万亿 |
| **干部** | 以干部队伍能力作风保障规划落地 |
| **环境** | 优化营商环境，激发市场活力 |

## 三、产业强州战略

### 3.1 三大路径

- **资源经济**：依托铝土矿、三七、高原特色农业等资源优势，推动就地转化增值
- **园区经济**：7个省级开发区覆盖8县市，营收、产值"两个破千亿"
- **口岸经济**：天保、都龙、田蓬三大国家级口岸驱动边境贸易升级

### 3.2 四大重点产业

| 产业 | 十五五目标 | 核心方向 |
|------|-----------|---------|
| **绿色铝** | 产值突破**1800亿元** | 铝精深加工，延伸产业链，打造绿色铝谷 |
| **三七中药材** | 综合产值达**千亿级** | 种植标准化、精深加工、品牌建设、康养融合 |
| **高原特色现代农业** | 总产值突破**550亿元** | 粤港澳大湾区"菜篮子"、绿色食品加工 |
| **康养旅游旅居** | 旅居目的地转型 | 五大主题、三七康养、民族非遗融合 |

### 3.3 项目储备

> 注：存在两个统计口径，均来自官方发布。

| 口径 | 项目数 | 总投资 | 来源 |
|------|--------|--------|------|
| 新闻发布会口径 | **2845 个** | 接近**万亿**元 | 2026-06-04 文山发布网 |
| 产业类项目口径 | **615 个** | **3804 亿元** | 2026-06-08 人民网 |

两个口径差异源于统计范围不同：2845个覆盖基础设施等全口径储备项目，615个聚焦产业类已通过论证的成熟项目。涵盖绿色铝、三七、现代农业、康养旅游、基础设施等领域。

## 四、"十四五"关键成就

| 领域 | 成就 |
|------|------|
| GDP | 突破1600亿元（2025年1632.64亿），占全省比重4.98% |
| 首个千亿产业 | 绿色铝产值1016亿元 |
| 工业 | 规上工业总产值从545.8亿元增至1539亿元 |
| 交通 | 高速公路通车里程达650公里 |
| 园区 | 7个省级开发区覆盖8县市 |
| 三七 | 入选全国优势特色产业集群和中小企业特色产业集群 |
| 用电 | 全社会用电量540.66亿千瓦时，稳居全省第一 |
| 民生投入 | 财政支出**80%以上**用于民生 |
| 农民增收 | 农村居民人均可支配收入增速**高于全省平均水平** |
| 脱贫底线 | 牢牢守住不发生规模性返贫致贫底线 |
| 民族团结 | 创成全国民族团结进步示范州、全国双拥模范城 |
| 全球荣誉 | 石漠化治理经验获评第五届全球最佳生态减贫案例 |
