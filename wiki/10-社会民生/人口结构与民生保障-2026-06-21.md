# 文山州人口结构与民生保障（2025年）

> 来源：文山州2025年统计公报、2026年政府工作报告

---

## 一、人口总量与结构

| 指标 | 数值 | 说明 |
|------|------|------|
| 常住人口 | 335.4 万人 | — |
| 城镇人口 | 138.1 万人 | 城镇化率 41.19% |
| 乡村人口 | 197.3 万人 | 占 58.81% |
| 男性 | 172.2 万人 | 占 51.34% |
| 女性 | 163.2 万人 | 占 48.66% |

## 二、年龄结构

| 年龄段 | 人口（万人） | 占比 |
|--------|-------------|------|
| 0-15岁 | 80.8 | 24.09% |
| 16-59岁 | 193.0 | 57.54% |
| 60岁及以上 | 61.6 | 18.37% |
| 其中65岁及以上 | 39.3 | 11.72% |

## 三、人口变动

| 指标 | 数值 |
|------|------|
| 出生人口 | 2.85 万人 |
| 出生率 | 8.47‰ |
| 死亡人口 | 3.35 万人 |
| 死亡率 | 9.96‰ |
| 自然增长率 | -1.49‰ |

人口自然增长率为负，老龄化率（60岁以上）达18.37%，已进入中度老龄化阶段。
