# 脱贫人口就业增收

> 文山州以政策落实、技能培训、劳务协作、特色产业为抓手，全方位推动脱贫人口就业增收。截至目前，全州脱贫劳动力外出就业 21.69 万人，预计全年实现务工收入 120 亿元。

## 一、核心数据速览

| 指标 | 数值 |
|------|------|
| 脱贫劳动力外出就业 | **21.69 万人** |
| 预计全年务工收入 | **120 亿元** |
| 公益性岗位托底安置 | **4.12 万人** |
| 就业帮扶车间 | **249 个** |
| 帮扶车间吸纳脱贫劳动力 | **0.44 万人** |
| 政府补贴性技能培训 | **1.86 万人次** |
| 培训后实现就业 | **0.79 万人**（含脱贫劳动力 0.27 万人） |
| 驻外劳务工作站 | **14 个** |
| 线上线下招聘会 | **627 场次** |
| 签订劳务协作协议 | **20 余份** |

## 二、四大举措

### 2.1 政策落实 — 夯实就业增收底盘

- 出台《文山州促进群众就业创业十件实事》
- 推出"就在文山·业兴七乡"二维码、小程序，集成创业贷款、惠企政策、求职招聘等 10 类服务事项
- 实现政策查询、岗位推荐"一键直达、一网通办"
- 统筹开发保洁员、护林员、护路员、护河员等农村基层公益性岗位，确保有就业意愿、有劳动能力的就业困难家庭至少 1 人实现就业
- 实施"归雁工程"，引进企业进乡入村办厂，通过"分散加工、家庭作坊"等形式建设就业帮扶车间 249 个
- 推动帮扶车间从"有"向"优"升级，配套建设产业链和技能提升体系

### 2.2 技能培训 — 提升就业核心能力

- 推行"企业订单、人社派单、机构接单、学员点单"培训模式
- 实施订单式、定向式、定岗式技能培训
- 紧扣四大重点产业（绿色铝、三七中医药、高原特色农业、康养旅游）需求

**典型案例 — 广南县**：构建"培训—考证—就业—创业扶持"全链条服务体系，开展人工智能训练师、焊接技术、电商运营等多元化培训，课堂办到产业园区、社区村组、群众家门口。已开展职业技能培训 63 期 3010 人次。

### 2.3 劳务协作 — 拓宽外出就业空间

- 组织开展"春风行动"系列就业服务活动
- 邀请南京、漳州、静安等地人社部门及优质用工企业赴文山考察洽谈
- 在上海、广东、福建等劳务输入地设立 14 个驻外劳务工作站
- 帮助脱贫人口稳在当地、稳在企业、稳在岗位

**典型案例 — 丘北县**：驻嘉兴市劳务工作站成立"企业联络队"，深入浙江杭州、宁波、嘉兴等 45 家重点企业"车间问需"，举办"丘北—浙江"线上线下专场招聘会 3 场，帮助 35 家浙江企业完成招工，对接落实普工、技工、后勤等岗位 1100 余个。

### 2.4 特色产业 — 扩容就近就业载体

围绕四大重点产业延链补链强链，发展乡村种植养殖、农产品精深加工、乡村旅游等业态，深度开发采摘分拣、生产加工、包装物流等就业岗位。

**典型案例 — 马关县**：以中药材产业为突破口，种植砂仁、草果、三七、黄精、重楼等 87 种中药材 42.56 万亩，联动 78 家中药材产业企业（含合作社），构建标准化种植、产地初加工、仓储物流、品牌营销一体化体系，带动就业 1 万余人（其中脱贫劳动力 0.21 万人）。
