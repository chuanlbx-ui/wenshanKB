# 文山州劳务品牌体系建设

> 来源：文山州人社局 | 2026-05-29
> 文山州推动劳务品牌与优势农业、绿色工业、非遗文化、特色餐饮融合发展，构建独具边疆特色的劳务品牌体系。

## 一、总体规模

| 指标 | 数值 |
|------|------|
| 培育特色劳务品牌 | **36 个** |
| 累计带动就业 | **超 80 万人** |
| 年务工总收入 | **151.56 亿元** |
| 劳务品牌经济规模 | **突破 1500 亿元** |

## 二、四大劳务品牌矩阵

### 2.1 "文山三七产业工"——优势农业

| 指标 | 数值 |
|------|------|
| 直接从业人员 | **1.46 万人** |
| 带动就业 | **4.38 万人** |
| 合作科研机构及院校 | **30 余家** |
| 院士（专家）工作站 | **32 个** |

- 组建三七大健康产业学院，推行"课堂+田间+车间"教学模式
- 推进三七产业直播电商人才培养
- 构建"技能培育—就业保障—产业联动"全链条一体化发展机制

### 2.2 "文山州绿色铝产业工"——绿色工业

| 指标 | 数值 |
|------|------|
| 已建/在建电解铝产能 | **343 万吨** |
| 全产业链直接就业 | **1.2 万人** |
| 带动就业 | **2.4 万人** |
| 年务工收入 | **13.18 亿元** |

- 成立**文山州绿色铝产业学院**，深化与昆明理工大学、昆明冶金高等专科学校合作
- 建成**全国唯一**铝产业公共实训基地
- 助力全州铝产业企业 **1100 名**员工实现学历、技能"双提升"
- 引进魏桥、神火、中铝等世界500强/中国500强企业

### 2.3 "砚山彝绣工匠"——非遗文化

| 指标 | 数值 |
|------|------|
| 从业人员 | **1.05 万人** |
| 带动绣娘居家就业 | **4600 余人** |
| 绣娘人均年收入 | **2.3 万元** |

- 推行"党建+绣娘+非遗文旅融合+文创品牌+数字电商"模式
- 砚山青禾文化：抖音月均销售额增长 40%，累计培训绣娘超千人，与 86 名绣娘稳定合作
- 已吸引 300 余名大学生参与研学

### 2.4 "文山壮乡苗岭米线师"——特色餐饮

| 指标 | 数值 |
|------|------|
| 全州米线小吃店面 | **8000 余家** |
| 加工企业 | **300 余家** |
| 直接从业人员 | **2.32 万人** |
| 带动就业 | **2.16 万人** |

- 2026 年文山早点米线文化周：客流量 **60 万余人次**，总营业额突破 **3000 万元**
- 马塘蔡家酒店文化周期间总营收约 15 万元，售出鳝鱼米线近 2000 碗
- 带动食材种植、加工生产、配套服务等上下游协同发展

## 三、产业融合模式

| 融合方向 | 劳务品牌 | 核心路径 |
|----------|---------|---------|
| 优势农业 | 三七产业工 | 产学研协同 + 直播电商 + 全链条技能 |
| 绿色工业 | 绿色铝产业工 | 产业学院 + 公共实训基地 + 学历技能双提升 |
| 非遗文化 | 彝绣工匠 | 电商平台 + 非遗文旅 + 居家灵活就业 |
| 特色餐饮 | 米线师 | 文化周引流 + 上下游产业链 + 品牌化经营 |
