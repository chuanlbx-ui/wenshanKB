# 文山州2025年人口数据

> 来源：文山州2025年国民经济和社会发展统计公报（2026年5月发布）、2026年政府工作报告（文政发〔2026〕4号）

---

## 一、人口总量与分布

| 指标 | 数值 |
|------|------|
| 年末常住人口 | **335.4 万人** |
| 城镇常住人口 | 138.1 万人 |
| 乡村常住人口 | 197.3 万人 |
| 城镇化率 | **41.19%**（比上年提高 0.62 个百分点） |

---

## 二、人口性别与年龄结构

| 指标 | 年末数（万人） | 比重（%） |
|------|:----------:|:------:|
| 总人口 | 335.4 | 100.0 |
| #男性 | 172.2 | 51.34 |
| #女性 | 163.2 | 48.66 |
| #0—15 岁（含不满 16 周岁） | 80.8 | 24.09 |
| #16—59 岁（含不满 60 周岁） | 193.0 | 57.54 |
| #60 周岁及以上 | 61.6 | 18.37 |
| #65 周岁及以上 | 39.3 | 11.72 |

---

## 三、人口自然变动

| 指标 | 数值 |
|------|------|

| 死亡人口 | 3.35 万人 |
| 人口死亡率 | 9.96‰ |
| 自然增长率 | **-1.49‰** |

## 四、就业

| 指标 | 数值 |
|------|------|
| 城镇新增就业 | 2.20 万人（较上年增加 0.10 万人） |

| 新增发放创业担保贷款 | 4.36 亿元（扶持 649 人/户，带动吸纳 1,414 人就业） |

## 五、居民收入

| 指标 | 数值 | 同比 |
|------|------|------|
| 居民人均可支配收入 | 26,580 元 | +4.4% |

| 农村常住居民人均可支配收入 | 17,501 元 | +6.2% |
| 城乡居民收入比 | 2.40 | 缩小 0.07 |

---

## 六、教育普及水平

| 指标 | 2025年 |
|------|--------|
| 学前教育毛入园率 | 95% |
| 九年义务教育巩固率 | 99.6% |
| 高中阶段毛入学率 | 91% |
| 义务教育优质均衡综合达标率 | 75.7% |
| 高考特控上线率 | 26.2% |
| 每千人口执业（助理）医师 | 2.92 人 |

> 来源说明：教育数据来自2026年政府工作报告"十四五"成就总结。统计公报口径为：学前三年毛入园率94.89%、九年义务教育巩固率100.22%、高中阶段毛入学率91.18%，与政府工作报告口径略有差异（系统计时点与统计口径微调所致），本笔记以政府工作报告为准。
