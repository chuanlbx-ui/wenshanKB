# 兴边富民与边境幸福村

> 文山州拥有438公里边境线，马关、麻栗坡、富宁三个边境县，少数民族人口占全州总人口的57.5%，维护边疆稳定安宁责任重大。十四五以来，以现代化边境幸福村建设为抓手，全面推进兴边富民行动；十五五期间创新实施"两增长三提升"三年行动，构建"1+4+N"沿边产业体系，全力打造边境幸福村升级版。

## 一、十四五成就：边境幸福村建设

| 指标 | 数值 |
|------|------|
| 总投入 | **15.63 亿元** |
| 覆盖沿边行政村 | **42 个** |
| 受益人口 | **13 万余人** |
| 建设周期 | 十四五期间 |

## 二、十五五新部署："两增长三提升"三年行动

十五五期间创新实施"两增长三提升"三年行动：

- **两增长**：沿边群众收入增长、村集体经济收入增长
- **三提升**：基础设施提升、产业发展提升、公共服务提升

## 四、"招小商、引厂到边"模式

- 创新就业帮扶模式，将小型加工厂引入边境村寨
- 带动**1.49万**边民实现就近就业
- 实现"家门口务工、务农两不误"

## 五、"1+4+N"沿边产业体系

| 层级 | 内容 |
|------|------|
| **1** | 热区特色种植（中药材、茶叶、水果等） |
| **4** | 四个核心加工方向：边境加工贸易、农特产品精深加工、中药材初加工、民族手工艺品 |
| **N** | 多个配套业态：电商物流、边民互市、乡村旅游等 |

## 六、民族团结与边疆治理

### 6.1 "十进"创建

文山州深入开展民族团结进步全域创建"**十进**"工作——进机关、进企业、进社区、进乡镇、进学校、进宗教活动场所、进军（警）营、进医院、进景区、进家庭，推动民族团结进步创建向更广领域延伸。

### 6.2 平远片区"三区"建设

持续推进平远片区"**三区**"建设（民族团结进步示范区、宗教和顺模范区、社会治理创新区），打造文山民族团结进步的标杆样板。
