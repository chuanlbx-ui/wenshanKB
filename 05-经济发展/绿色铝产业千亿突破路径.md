# 绿色铝产业：千亿产值突破路径

## 发展成就（截至 2025 年底）

| 指标 | 数据 |
|------|------|
| 电解铝产能 | **343 万吨**（占云南 51%、全国近 1/10） |
| 工业产值 | **1,016.07 亿元**（+22%） |
| 占全省绿色铝产值 | **50.7%** |
| 对全州经济增长贡献率 | 超 **24%** |
| 铝加工产能 | **407.4 万吨** |
| 合金化率 | **69.92%** |
| 累计绿电铝产量 | 755 万吨 |
| 累计减排 CO₂ | 超 8,000 万吨 |
| 直接/间接带动就业 | 1.3 万 / 2.4 万余人 |

## 五大发展经验

1. **抢抓政策机遇**：引进中铝、河南神火、山东魏桥等龙头
2. **战略引领与政策赋能**：锚定"中国绿色铝谷"核心区定位
3. **链主牵引与集群发展**："链长+链主"机制，累计签约 53 个项目
4. **要素保障精准发力**：清洁能源装机近 700 万千瓦（占比 99.8%+），电解铝绿电比例超 80%
5. **科技创新**：链主企业专利 201 项，宏泰 600kA 生产线达国际领先

## 挑战

- 美国铝制品关税提至 50%，欧盟碳关税全面实施
- 房地产深度调整拖累铝消费需求
- 产业链下游投资压力大
- 物流成本居高不下

## 机遇

- 双碳、产业转移等国家战略叠加
- 2026 年全球新能源汽车销量预计突破 1.5 亿辆
- 文蒙铁路开工、文靖铁路列入规划
- RCEP 协议生效，面向南亚东南亚

## "十五五"突破路径

构建**"1+1+3+N"**现代化铝产业体系：

- 电解铝产能突破 **400 万吨**
- 铝企超 **100 家**
- 力争 2030 年铝产业综合产值突破 **1,800 亿元**
- 打造滇桂铝产业先进制造业集群
- 推广铝家居应用
