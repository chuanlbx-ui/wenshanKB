# 文山：从"产量大州"到"产业强州"

> **来源**：农民日报客户端 | **发布时间**：2026-06-11
> **核心**：文山州依托高原特色、立足区位优势，加快构建"三带五区多基地"现代农业产业格局，从靠天吃饭的"产量大州"向价值驱动的"产业强州"跨越。

## 一、核心指标一览

### 2026年Q1关键数据

| 指标 | 数值 | 同比 |
|------|------|------|
| 农林牧渔业总产值 | **56.81 亿元** | +2.5% |
| 农村常住居民人均可支配收入 | **4,225 元** | **+6.1%** |

### 中药材在地面积

| 指标 | 数值 | 排名 |
|------|------|------|
| 以三七为主的中药材在地面积 | **257 万亩** | 全省第一 |

## 二、"十五五"农业发展思路

**总体方针**：稳住粮、优化经、巩固畜、提升林、突破药、保住烟、拓展渔、融合旅

**产业空间布局**："三带五区多基地"

| 维度 | 内容 |
|------|------|
| 三大特色产业示范带 | 现代绿色农业、沿边热区特色农业、林下经济产业 |
| 五大优势产业区 | 优质粮油、三七中药材、高端果蔬、高品质肉牛、农文旅融合 |
| 补齐短板 | 清洗分级、冷链仓储、精深加工 |

## 三、十五五量化目标

| 指标 | 目标值 | 时间节点 |

| 农产品加工转化率 | **75% 以上** | 2030年 |

| 肉牛存栏 | **120 万头** | — |
| 农产品加工产值与农业总产值比 | **2.8:1 以上** | — |

## 四、联农带农模式

探索 **"企业 + 合作社 + 农户"** 模式：
- 小农户嵌入大产业链，抱团发展
- 利益共享、风险共担
- 推动产业做强同时确保小农户不被甩在身后

## 六、农文旅融合

依托普者黑、坝美等核心资源打造康养旅居目的地，为农业拓展文旅融合新赛道。

> 目标：2030 年培育旅居村 **80 个以上**

## 七、农村要素市场化改革

| 领域 | 目标 / 模式 |
|------|-------------|
| 承包地"三权分置" | 引导土地经营权有序流转 |
| 土地流转规模 | 2030 年 **150 万亩以上** |
| 闲置宅基地和农房 | 盘活发展民宿、文创、康养 |
| "三变"改革 | 资源变资产、资金变股金、农民变股东 |
| 村级集体经济 | 村均收入力争 **25 万元以上** |
| 农村金融 | 扩大承包地经营权、农房抵押贷款范围 |

## 八、城乡融合与乡村振兴

- **冷链物流体系**：完善农产品进城通道
- **"家门口的务工车间"**：农产品加工、劳动密集型产业向县域乡镇布局 → 挣钱、顾家、守边三不误
- **"千万工程"落地**：

| 绿美乡村建设 | 数量 |
|-------------|------|
| 省级绿美乡镇 | **52 个** |
| 省级绿美村庄 | **105 个** |
| 州级绿美村庄 | **5,015 个** |

## 九、人才振兴

| 方向 | 措施 |
|------|------|
| 精准培训 | 三七种植、肉牛养殖、电商直播等 |
| 返乡创业 | 优化环境、搭建对接平台 → "回得来、留得住、干得好" |
| 高端引才 | 种业创新、数字农业、精深加工 → 专家工作站、良种繁育基地 |
| 良种目标（2030） | 主要农作物良种覆盖率 **96% 以上**；畜禽良种覆盖率 **92% 以上** |

## 十、底线与驱动

- **三条底线**：粮食安全、耕地保护、不发生规模性返贫
- **四轮驱动**：产业筑基 + 改革赋能 + 城乡互通 + 人才赋能
