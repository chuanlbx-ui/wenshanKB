# RCEP框架下文山口岸机遇

> RCEP（区域全面经济伙伴关系协定）自 2022 年 1 月生效，2025-2026 年进入**关税减让深化期**与**原产地规则红利爆发期**。文山州天保、都龙、田蓬三大口岸作为中国西南连接东盟的前沿节点，面临前所未有的制度性开放红利。

## 三、原产地累积规则

### 3.1 规则核心机制

```
传统双边FTA（中越）：
  中国原料 → 中国加工 → 出口越南 → 原产地：中国 ✓
  越南原料 → 中国加工 → 出口越南 → 原产地：中国 ✗（不含越南原料）

RCEP累积规则：
  中国原料 + 越南原料 → 中国加工 → 出口日本 → 原产地：RCEP ✓
  （中越两国原料累积计算，突破双边限制）
```

### 3.2 文山受益场景

| 模式 | 具体场景 | RCEP优势 |
|------|----------|----------|
| **中国原料+越南加工** | 文山铝锭 → 越南型材加工 → 出口日本/韩国 | 利用越南低廉劳动力 + 中日韩零关税 |
| **越南原料+中国加工** | 越南铜精矿 → 文山冶炼 → 出口东盟/日本 | 原产地累积，享RCEP优惠税率 |
