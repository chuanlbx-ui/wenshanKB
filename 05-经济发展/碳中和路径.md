# 碳中和路径

## 碳排放现状

| 指标 | 数据 |
|------|------|
| 可再生能源发电装机占比 | 约 **99.8%** |
| 非化石能源消费占比 | 超 **60%**（全省前列） |
| 电力总装机（2025） | 约 **466 万 kW** |
| 电解铝产能 | **343 万吨**（占全国近 1/10） |
| 累计生产绿电铝 | 约 **755 万吨** |
| 累计减排 CO₂ | 超 **8,000 万吨** |

**减排思路**：以 **"控制二氧化碳排放强度为主、总量控制为辅"** 推进碳达峰。

## 绿色铝产业减排

每吨铝比火电炼铝减少 **8-10 吨** CO₂ 排放。2025 年绿色铝产值突破 **1000 亿元**。

循环产业链：
```
铝土矿 → 氧化铝 → 电解铝 → 合金铝 → 再生铝 → 铝灰渣综合利用
```
铝水直供减少重熔能耗，绿电直供节约年成本约 **6 亿元**、节电约 **7,600 万 kWh**。

## 能源侧绿色转型路线

### 新能源装机路线图

| 时间节点 | 目标 |
|----------|------|
| 2029 年 | 电力总装机约 **900 万 kW**（新增新能源 500 万 kW） |
| 2030 年 | 新能源装机达 **820 万 kW** |
| 2035 年 | 新能源装机达 **1,020 万 kW** |

### 配套措施
- **储能**：2029 年建成新型储能约 60 万 kW
- **电网**："十四五"投入 85.7 亿元电网建设
- **绿电直供**：探索"绿电直供"模式，提升消纳与稳定性
- **天然气**：建设百色至文山天然气管道
