# 中越企业对接会（2026年4月）

## 基本信息

| 项目 | 详情 |
|------|------|
| **时间** | 2026年4月 |
| **主办方** | 文山州商务局 + 越南工贸部贸易促进局贸易投资促进中心 |
| **参会规模** | **61家**中越企业 |
| **报道时间** | 2026年4月22日 |
| **来源** | 云南日报 |

## 参会领域

| 领域 | 说明 |
|------|------|
| 教育科技 | 技术合作、人才培训 |
| 农业物资 | 农资贸易、技术交流 |
| 医疗器械与康养 | 医疗设备、健康产业 |
| 中药材及保健食品 | 三七等道地药材贸易 |

## 对接会内容

### 中方推介
文山州商务局重点推介：
- 文山作为 **面向南亚东南亚辐射中心重要节点** 的区位优势
- **天保口岸** 等对越通道建设成果

### 越方推介
越南工贸部贸易促进局专题介绍：
- 越南投资环境与劳动力资源优势
- **RCEP 框架下**关税减免、原产地累积等政策红利
- 热情邀请文山企业赴越投资兴业

### 企业推介
6家中越企业代表依次推介。

## 越方实地考察

越方代表团实地考察三家文山标杆企业：

| 企业 | 考察重点 |
|------|---------|
| **云南七丹药业股份有限公司** | 三七全产业链发展 |
| **文山三七集团有限公司** | 三七产业价值转型 |
| **云南文品出山供应链管理有限公司** | 农产品供应链运营模式 |

## 战略意义

- 对接会是文山州落实 **"口岸经济"路径**、深化对越经贸合作的重要举措
- 覆盖教育科技、农业物资、医疗器械、中药材四大领域，为中越企业搭建精准对接平台
- 越方实地考察三七核心企业，为三七产业国际化拓展越南市场打开窗口
- 与天保口岸协同共治机制（2026年4月23日启动）形成"口岸+产业"双轮驱动格局
