# 文山州2026年经济社会发展目标

> 2026年是"十五五"开局之年。政府工作报告明确了全年主要预期目标，发改委预算公开进一步细化。

---

## 一、主要预期目标

| 指标 | 2026年目标 | 来源 |
|------|-----------|------|
| 地区生产总值（GDP）增长 | **5%以上** | 政府工作报告 |
| 规模以上工业增加值增长 | **10%以上** | 政府工作报告 |
| 地方一般公共预算收入增长 | **1%** | 政府工作报告 |
| 固定资产投资 | 与上年**持平** | 政府工作报告 |
| 社会消费品零售总额增长 | **4%** | 政府工作报告 |
| 居民人均可支配收入增长 | 与经济增长**同步** | 政府工作报告 |
| 居民消费价格指数（CPI）涨幅 | **控制在2%左右** | 政府工作报告 |
| 单位GDP二氧化碳排放 | 完成省下达任务 | 政府工作报告 |

## 二、发改委细化目标

| 指标 | 2026年目标 |
|------|-----------|
| 全员劳动生产率 | 达**8.9万元/人** |
| 常住人口城镇化率 | 达**41.8%**（2026年目标） |
| 外贸进出口总额增长 | **10%** |
| 实际使用外资额增速 | 完成省下达任务 |
| 全社会研发经费投入增长 | **10%左右** |
| 城镇新增就业人数 | **1.8万人** |
| 粮食产量 | **179.5万吨** |
| 耕地保有量 | 稳定在**921.6万亩** |
| 生产安全事故起数 | 下降**5%** |

> **注**：2025年文山州常住人口城镇化率实际值为**41.19%**（统计公报口径），十五五规划纲要2025年预计值约41%。

## 三、重点产业发展目标

### 绿色铝产业

- 绿色铝产值增长**20%以上**
- 推动中铝绿材、信特等重点项目建成投产
- 推动**50万吨**再生铝项目建成投产
- 加快剩余电解铝产能转移落地
- 构建完善"1+1+3+N"铝产业体系

### 三七中药材产业

- 三七中药材产业综合产值**500亿元以上**（2026年目标，2025年实际值为**456亿元**，增长10.4%）
- 新建三七GAP基地**1.5万亩**以上
- 推动种质种源、特色产业集群、气血康扩产扩能等重点项目建成投产
- 力争华润三九血塞通系列加工基地落地建设
- 改造提升三七中药材交易中心

### 高原特色现代农业

- 农林牧渔业总产值增长**3%以上**
- 培育农业龙头企业**50户**
- 培育绿色优质农产品**70个**
- 构建"1+4+N"高原特色现代农业产业体系

### 旅游旅居

- 打造"有一种叫云南的生活—三七之乡 康养文山""旅居云南·好在文山"品牌IP
- 推动普者黑景区业态重塑
- 坝美景区生态化提升
- 边境旅游试验区创建

## 五、基础设施建设

- 实施亿元以上重大项目**166个**
- 开工建设：文靖铁路、清水河水利枢纽工程、500千伏南屏变及富宁天然气管道支线
- 加快进度：文蒙铁路、国道219
- 确保建成：天麻高速、师丘高速、富宁港水运枢纽、百色至文山天然气管道
