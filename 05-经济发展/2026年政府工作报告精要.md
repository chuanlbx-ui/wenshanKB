# 文山州2026年政府工作报告精要

> **来源**：文山州人民政府公报 | **发布时间**：2026-02-28

### 二、产业发展

| 领域 | 关键成果 |
|------|---------|
| 园区建设 | 马关、丘北、广南 3 个园区获批省级产业园区；全州园区营业收入、规上工业总产值增速保持全省第一 |
| 绿色铝 | 产值突破 **1000 亿元**，占全省半壁江山；新转移电解铝产能 31 万吨，新建成再生铝产能 70 万吨，新投产涉铝项目 11 个；云南宏泰上榜"中国制造业民营企业 500 强" |
| 有色金属 | 锌、铟、钨、锑、硅等产值达 **140.62 亿元** |
| 中药材 | 入选全国名特优新农产品和品牌精品培育计划；综合产值 **456 亿元**，增长 10.4% |
| 高原特色农业 | 新增省级龙头企业 12 户；农林牧渔业总产值增长 **2.6%** |
| 文旅 | 新增 A 级景区 7 个；仙人洞、六郎城入选首批"旅居云南"高质量发展试点；接待游客 **5,256.7 万人次**，旅游花费 **425.7 亿元**，分别增长 5.1%、8.4% |

### 三、改革开放

- 推动国家、省 50 个重点事项和 15 个文山特色"一件事"落地实施
- 政务服务网上可办率 **100%**，全程网办率 **91.4%**
- 招引省外产业项目 **140 个**、到位资金 **158.7 亿元**
- 外贸进出口总额 **39 亿元**、增长 38.8%
- 口岸进出口货值 **69.5 亿元**，同比增长 **136.7%**（增速居全省第2位）

### 四、城乡发展

- 改造老旧小区 **18 个**，新建改造城市地下管网 **200 公里**
- 交付保交楼、保交房 **10,634 套**
- **文山市创成全国文明城市**
- 那兴高速建成通车，文蒙铁路建设全线铺开
- 新增新能源装机 **129.05 万千瓦**，建成新型储能装机 **49.1 万千瓦**

## "十五五"时期总体目标

加快建设"平安、康养、文明、美丽、幸福、清廉"文山，定位为六大方向：
1. 民族团结进步示范区
2. 石漠化地区生态文明建设排头兵
3. 云南面向东南开放先行区
4. 新时代兴边富民示范区
5. "绿色铝谷"核心区
6. 三七康养旅居目的地

## 2026年重点工作

### 主要预期目标
| 指标 | 目标 |
|------|------|
| GDP增长 | 5%以上 |
| 规上工业增加值增长 | 10%以上 |
| 地方一般公共预算收入增长 | 1% |
| 固定资产投资 | 与上年持平 |
| 社会消费品零售总额增长 | 4% |
| 居民人均可支配收入 | 与经济增长基本同步 |
| CPI涨幅 | 控制在2%左右 |
| 绿色铝产值增长 | **20%以上** |
| 三七中药材综合产值 | **500亿元以上**（2026年目标） |
| 重大项目 | **166个**（亿元以上） |

### 2026年重大项目
- **建成投运**：天麻高速、师丘高速、富宁港水运枢纽、百色至文山天然气管道
- **开工建设**：文靖铁路、清水河水利枢纽工程、500千伏南屏变、富宁天然气管道支线
- **加快建设**：文蒙铁路、国道219等

### 八个方面重点工作
1. **挖掘有效需求**：提振消费，扩大有效投资，壮大经营主体
2. **实施产业强州**：做大做强绿色铝、三七等主导产业，培育新兴产业
3. **推进科教兴州**：加快科技创新，坚持教育优先，汇聚人才
4. **深化改革开放**：深化重点领域改革，优化营商环境，扩大对越开放
5. **促进城乡融合**：提升城镇品质，推进乡村全面振兴
6. **建设美丽文山**：加强生态环境保护，推动绿色低碳发展，推进"无废城市"建设
7. **改善社会民生**：促进高质量就业创业，推进"健康文山"建设，健全社保体系
8. **加强边疆治理**：推进民族团结进步，筑牢边境安全屏障
