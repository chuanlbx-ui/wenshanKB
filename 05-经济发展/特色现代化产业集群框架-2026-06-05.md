# 文山州特色现代化产业集群框架

> **来源**：经济日报 2026-06-05 | **记者**：曹松
> **核心**：文山州以资源经济、园区经济、口岸经济为路径，构建四大重点产业为核心的特色现代化产业集群。

## 一、总体战略

- **战略**："产业强州"，四大重点"产业、项目、干部、环境"
- **路径**：资源经济 + 园区经济 + 口岸经济
- **定位**："3+3"发展定位（民族团结进步示范区、石漠化地区生态文明建设排头兵、云南面向东南开放先行区 + 新时代兴边富民示范区、"绿色铝谷"核心区、三七康养旅居目的地）

## 二、绿色铝产业

| 指标 | 数据 |
|------|------|
| 2025年产值 | **1016亿元**（首次突破千亿，锚点值1000亿） |
| 战略定位 | 发挥"绿电+先进制造业"新优势 |
| 2030年目标 | 铝产业综合产值突破**1800亿元** |

## 三、三七中药材产业

| 指标 | 数据 |
|------|------|
| 2025年综合产值（核心口径） | **220亿元**（同比+7%） |
| 种植方向 | 绿色化、标准化种植 |
| GAP目标 | 2028年州内三七种植实现GAP全覆盖 |
| 合作企业 | 云南白药、华润三九等 |
| 质控体系 | "绿色种植+趁鲜加工+产地仓+地标赋能"全链条 |

## 四、高原特色现代农业

| 维度 | 内容 |
|------|------|
| 四大农业 | 生态农业、设施农业、高效农业、共享农业 |
| 融合方向 | 一二三产深度融合 |
| 目标市场 | 全国优质绿色现代农产品生产基地、粤港澳大湾区"菜篮子"基地 |

## 五、康养旅游旅居

| 品牌 | 定位 |
|------|------|
| "有一种叫云南的生活——三七之乡 康养文山" | 全省旅游品牌 |
| "旅居云南·好在文山" | 旅居品牌 |
| 方向 | 旅游产品、模式、业态创新和服务创优 |

## 六、新兴产业布局

同步布局**新能源、新材料、数字经济**等新兴产业，集中推进一批投资大、链条长、带动强的标志性项目。
