# 2026年文山州四大重点产业最新数据

> **数据截止**：2026年6月 | **来源**：政府工作报告 + 经济日报 + 文山发布网 + 央视网
> "产业强州"战略：以资源经济、园区经济、口岸经济为路径，持续壮大四大重点产业。

---

## 一、绿色铝产业 — 首个千亿级产业

### 核心数据

| 指标 | 数值 | 说明 |
|------|------|------|
| 2025年产值 | **突破1016亿（精确值1021.8亿元）** | 突破千亿，同比+22%，占全省半壁江山 |
| 电解铝建成产能 | **343 万吨** | 占全省 52.8%、全国 7.7% |
| 转移产能 | 319.8 万吨 | 新增转移 31 万吨，满负荷运行 |
| 铝加工产能 | **418 万吨** | 较2021年增长21倍 |
| 再生铝产能 | **90 万吨** | 实现从零到一突破 |
| 合金化率 | 69.92% | 铝合金转化率接近70% |
| 直接带动就业 | 1.3 万余人 | — |
| 累计减排 | 超 7,000 万吨 CO₂ | 三家电解铝企业绿电使用比例均超70% |

### "十五五"目标

- **2030年综合产值突破 1,800 亿元**
- 推动"链主+链创"深度融合，向新材料、终端制造等价值链高端延伸
- 构建"电解铝+再生铝+3园区+N项目"产业体系

### 重点项目

| 项目 | 说明 |
|------|------|
| 中铝 20 万吨绿色低碳铸轧 | 补齐精深加工短板 |
| 砚山智铝二期 | 延伸产业链 |
| 云南神火超薄铝箔（0.0045mm） | 高附加值终端产品 |
| 文山华疆再生铝循环利用 | 循环经济 |
| 信特项目 | 新投产 |
| 50 万吨再生铝投产 | 2026年目标 |

### 产业体系："1+1+3+N"

| 层级 | 内容 |
|------|------|
| 1 个核心 | 绿色铝冶炼（电解铝） |
| 1 个基础 | 铝土矿—氧化铝 |
| 3 大精深加工方向 | 铝合金材料、铝制零部件、终端铝制品 |
| N 个细分领域 | 汽车轻量化、建筑型材、电子箔/超薄铝箔等 |

### 零碳园区

- **砚山零碳产业园区**：全省 7 个千亿级产业园区之一
- **富宁零碳产业园区**：绿色铝冶炼与再生铝循环利用基地
- 清洁能源占比 **99.8%**
- 全省唯一全链条绿色铝产业体系

## 二、三七及中药材产业 — 冲刺千亿级

### 核心数据

| 指标 | 数值 | 说明 |
|------|------|------|
| 三七单品种产值（2025） | **220 亿元** | 增长 7%（单一品种，非中药材全口径） |
| 全部中药材综合产值（2025） | **456 亿元** | 增长 10.4% |
| 三七加工业产值（2025） | **118.57 亿元** | 较2020年增长 227.54% |
| 中药材种植面积 | **245.2 万亩** | 全省第一 |
| 三七 GAP 标准化基地 | **1.5 万亩**（2026年已建成） | 2028年全覆盖 |
| 林下三七 | 8,000 亩 | — |
| 带动群众增收 | 2 万余户 | 户均年增收约 3 万元 |
| 年提供务工岗位 | 20 万人次 | — |

### "十五五"目标

- **千亿级产业**
- 2028年实现州内三七种植 GAP 全覆盖
- "七统一"全链条质控：种子种苗、订单种植、规范种植、公开购销、产地加工、仓储物流、地标赋能
- 2026年目标：三七中药材综合产值 **500 亿元以上**
- 新建三七 GAP 基地 **1.5 万亩以上**

### 龙头企业与项目

| 企业/项目 | 投资/说明 |
|------|------|
| 云南白药 | 投资 3.26 亿元产业提升工程，气血康扩产 |
| 昆药集团 | 全国最大血塞通绿色生产基地 |
| 华润三九 | 力争血塞通系列加工基地落地 |
| 金七药业 | 精深加工 |
| 三七精深加工集群 | 国家级中小企业特色产业集群，中央财政千万级奖补 |

### 科技突破

- 轮作周期从 20 年缩短至 **3—5 年**（联合院士团队攻关）
- 道地药材防控技术入选国家级主推技术
- 张伯礼院士点赞品质升级成效
- 推进三七药食同源申报
