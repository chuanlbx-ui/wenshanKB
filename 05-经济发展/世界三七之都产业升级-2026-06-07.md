# 世界三七之都建设与产业升级

> **来源**：科技日报 2026-06-07 | **记者**：赵汉斌
> **核心信息**：文山锚定千亿级产业目标，以全产业链升级、科技创新赋能、品牌品质筑基为抓手，建设世界三七之都。

## 一、2025年产业数据

> **注意**：三七产值存在多口径统计。副州长李松涛口径为三七产业核心产值220亿元（+7%），政府工作报告锚点值456亿元为含相关产业的广义口径。两者不矛盾，仅口径不同。

## 二、种植端升级

| 举措 | 现状 | 目标 |
|------|------|------|
| GAP标准化基地 | 2025年7000亩 | **2026年扩至1.5万亩** |
| GAP全覆盖 | — | **2028年全州覆盖** |
| 种植面积 | 5.3万亩 | 2028年突破8万亩，2030年稳定11万亩 |
| 良种覆盖 | 13个新品种 | **2030年三七良种全覆盖** |
| 种源平台 | "云药种谷" + 新质生产力孵化园 | 与云南白药等共建 |

## 三、精深加工升级

- **合作模式**："企业+"与"+企业"双向合作
- **合作企业**：云南白药、华润三九等**60余家**知名企业
- **近三年落地**：**70个**产业项目，总投资**56亿元**
- **加工转化**：从"80%原料外销"逆转为**80%实现就地精深加工**
- **加工能力**：2条趁鲜加工示范线、5个标准化三七产地仓，鲜七年加工能力**4万吨**

## 四、质控体系

| 环节 | 措施 |
|------|------|
| 标准体系 | 种子种苗、种植、加工、购销等"七统一"管理 |
| 全链条 | "绿色种植+趁鲜加工+产地仓+地标赋能" |
| 产地仓 | 5个标准化三七产地仓 |

## 五、千亿级目标路径

| 维度 | 路径 |
|------|------|
| 标志性项目 | 三七健康产业园、三七日化品加工 |
| 招商方向 | 三七提取物、中药配方颗粒、健康食品 |
| 功效拓展 | 从"生打熟补"延伸至慢病调理、全周期养生 |
| 2030年目标 | 综合产值突破**千亿元**，种植面积稳定11万亩、年产量1.5万吨以上 |
