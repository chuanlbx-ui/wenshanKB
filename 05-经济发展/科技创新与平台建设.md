# 科技创新与平台建设

> 数据来源：2026年政府工作报告 + 文山州发改委巡察整改通报
> 统计口径：2025年全年

## 一、省级重点实验室"零"突破

2025年，文山州牵头联动南方电网储能股份有限公司、高等院校等优质力量，获批建设 **云南省电力储能重点实验室**，实现文山州省级重大科技创新平台"零"的突破。

| 平台 | 级别 | 领域 | 意义 |
|------|------|------|------|
| 云南省电力储能重点实验室 | 省级 | 电力储能 | 全州首个省级重点实验室 |

### 核心技术突破

实验室集中科研力量开展技术攻坚，成功攻克多项关键核心技术：

| 核心技术 | 说明 |
|----------|------|
| 锂钠混合储能系统集成 | 多类型储能协同控制与系统集成 |
| 构网型储能精准控制 | 提升电网惯量与频率支撑能力 |
| 高压直挂式储能拓扑 | 提升单机容量与效率 |

### 落地应用：丘北宝池储能站

系列创新技术已落地应用于 **云南丘北200MW/400MWh锂钠混合储能宝池储能站**，关键指标：

| 指标 | 数据 |
|------|------|
| 装机规模 | 200MW / 400MWh |
| 年可调节电量 | 5.8 亿千瓦时 |
| 累计充放电量 | 突破 3 亿千瓦时 |

### 世界纪录与荣誉

| 荣誉 | 说明 |
|------|------|
| 全球首套构网型钠离子储能系统 | 国际首创 |
| 全球最大单机高压直挂式构网型储能系统 | 国际领先 |
| 2026储能应用创新典范TOP10 | 国际储能创新大赛入选 |

此次多项核心技术的突破性成果，为文山州新型电力系统建设、绿色能源产业链提质增效注入强劲动能，也为全省储能产业技术革新、能源绿色低碳转型提供了可复制、可推广的文山经验。

## 二、国家级三七工程研究中心

获批建设 **国家级三七工程研究中心**，是文山州首个国家级产业技术研发平台。

围绕三七种质资源保护、连作障碍突破、精深加工、质量标准等全产业链环节开展工程技术研发与成果转化。
