# 重点产业项目与营商环境优化

> **来源**：云南网 | **记者**：普建彬 | **发布时间**：2026-06-04

## 一、总体框架

2026年"十五五"开局之年，文山州以"项目为王、产业为基"为主线，通过万亿级项目储备与"文山服务"营商环境优化双轮驱动。

| 维度 | 数值 |
|------|------|
| 重点产业项目储备 | **2845 个** |
| 总投资规模 | **近万亿** |

## 二、四大产业重点项目

### 2.1 绿色铝

| 项目 | 说明 |
|------|------|
| **中铝20万吨绿色低碳铸轧** | 精深加工短板填补 |
| **云南神火超薄铝箔** | 0.0045mm，打破国外垄断 |

目标："十五五"末综合产值突破 **1800亿元**。

### 2.2 三七中药材

| 项目 | 说明 |
|------|------|
| **华润三九血塞通基地** | 全产业链升级龙头 |
| **云南白药气血康口服液二期** | 扩大产能，提升附加值 |

目标：向**千亿级产业**迈进，从"原料基地"向"精深加工高地"转型。

### 2.3 高原特色现代农业

| 项目 | 说明 |
|------|------|
| **砚山蔬菜供应基地** | 粤港澳大湾区"菜篮子" |
| **广南铁皮石斛种植加工** | 特色中药材种植+加工 |

目标：现代农业产值突破 **550亿元**。

### 2.4 康养旅游

- 融合**三七文化**与**民族非遗**资源
- 统筹山水田园、边境风情、红色研学等板块
- 绘就全域旅游新图景

## 三、营商环境：28条惠企措施

文山州将优化营商环境作为"一号工程"，出台 **28条惠企措施**，推动招标投标、便利纳税等指标位居全省前列。

### 3.1 服务提速

| 举措 | 内容 |
|------|------|
| "高效办成一件事"改革 | 深化政务服务提效 |
| 促进民营经济发展壮大十条措施 | 提升政务效能 |
| "园区事园区办" | 组建专班跟进用地、融资、人才需求 |

### 3.2 亲商机制

| 制度 | 内容 |
|------|------|
| **"第一服务员"** | 主要领导带头对接企业 |
| **"驻企服务员"** | 专人驻企全程服务 |
| **"周末亲清约·政企面对面"** | 常态化政企沟通平台 |
| **"红色法务团"** | 实施柔性监管，提供法律服务 |
| **"企业安静期"** | 减少对企业正常经营的干扰 |

### 3.3 要素保障

- 组建专班跟进用地、融资、人才等需求
- 推行"园区事园区办"，破解企业落地难题
- 鼓励民间资本参与新质生产力、新兴服务业等领域投资
- 深入贯彻《民营经济促进法》

> 核心理念："换位思考"——以"有一种叫文山的服务"让企业安心发展。
