# 砚山绿色铝产业园区：千亿传奇

全州绿色铝产值已于2025年突破1000亿元大关。

> **来源**：云南网 | **发布时间**：2026-01-05 | **作者**：韩艳慧
> **核心**：砚山产业园区（云南铝业创新产业园）集聚28家企业，以铝水直供模式和零碳园区建设书写千亿级产业新城的崛起之路。项目累计引入投资额约286亿元。

---

## 一、园区概况

砚山产业园区是文山州打造"中国绿色铝谷"核心区的主阵地，2019年魏桥集团落户以来，依托"规划引领"完成万亩土地盘活，实现近中远三期发展空间有序衔接。

| 指标 | 数据 |
|------|------|
| 入驻企业 | **28家** |
| 项目累计投资（非产值） | **约286亿元** |
| 2024年铝产业产值 | **476.7亿元**（占全县规上工业 92.6%） |
| 2025年预计产值 | 突破 **550亿元** |
| 全州绿色铝产值 | 突破 **1000亿元**（2025年 首个千亿级产业，实际值1021.8亿） |
| 涉铝项目 | 引进 24个，投产 19个 |

---

## 二、全产业链布局

以魏桥集团云南宏泰公司 **203万吨绿色铝项目** 为依托，构建完整产业链：

> **碳素 → 电解铝 → 初加工 → 精深加工 → 铝应用 → 再生铝**

产品涵盖铝合金棒、高端铝箔、建筑型材等多个领域。

### 铝水直供模式

| 维度 | 数据 |
|------|------|
| 铝水就地转化率 | **100%** |
| 铝材精深加工目标 | **100%** |
| 年节约成本 | **约6亿元** |
| 年节电 | **7600万千瓦时** |
| 度电产值（2023→2026E） | 2.04元 → **3.5元** |

---

## 三、零碳园区建设

| 指标 | 数据 |
|------|------|
| 绿电使用比例 | **>80%** |
| 电解槽技术 | 600kA 超大容量（全球领先） |
| 年减少CO₂排放 | **2600万吨**（相当于植树 14.3亿棵） |
| 阳极残极回收率 | **100%** |
| 烟气净化效率 | **>99.8%** |
| 认证体系 | ASI 体系认证 / 碳足迹认证 / 绿电铝认证 |

---

## 四、创新机制

### "砚山魏桥速度"

- 2020年云南宏泰从开工到投产仅用 **195天**
- 机制：「县委书记、县长担任施工队长 + 园区吹哨、部门报到」
- 30多个部门协同作战，每个项目配备专属服务班子、"保姆式"代办

### 未来展望

| 目标 | 时间 |
|------|------|
| 企业营业收入突破 **1000亿元** | **2026年** |
| 规上工业总产值突破 **1000亿元** | **2028年** |
| "十五五"末建成最具核心竞争力绿色铝谷核心区 | **2030年** |

---

## 五、产业升级路径

重点引进汽车轻量化用铝、航空航天用铝、高纯铝等精深加工项目，推动原铝产品和精深加工两个 100% 就地转化，通过云南辐射中心枢纽优势走向南亚东南亚及全球市场。
