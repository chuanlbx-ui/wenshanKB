# 文山"十五五"四大重点产业新闻发布会

> 发布时间：2026年6月3日 | 来源：县域经济网

## 一、总体定位："3+3"发展格局

文山州在"十五五"期间确立 **"3+3"发展定位**，以四大重点产业为引擎，推动全州经济社会跨越发展。

**"3+3"定位**：
1. 民族团结进步示范区
2. 石漠化地区生态文明建设排头兵
3. 云南面向东南开放先行区
4. 新时代兴边富民示范区
5. "绿色铝谷"核心区
6. 三七康养旅居目的地

## 二、四大重点产业

### 1. 绿色铝产业

| 指标 | 数值 |
|------|------|
| 2025年产值 | 突破 **1000 亿元**（2025年实际1016亿元） |
| 2030年目标 | **1800 亿元** |
| 再生铝产能规划 | **150 万吨** |

- 巩固"中国绿色铝谷"核心地位
- 构建"1+1+3+N"铝产业体系
- 促进电解铝、再生铝和铝加工融合发展
- 大力发展"绿电+先进制造业"

### 2. 三七及中药材产业

| 指标 | 数值 |
|------|------|
| 三七综合产值（2025年） | **220 亿元**（仅三七品种） |
| 中药材综合产值（2025年） | **456 亿元** |
| 2030年目标 | 打造**千亿级**产业集群 |
| 三七 GAP 基地 | 扩大至 **1.5 万亩** |

- 推进种质资源保护、标准化种植、精深加工
- 建设智能化产地仓和质控体系
- 文山三七入选全国名特优新农产品和品牌精品培育计划
- 30 个院士工作站，7 个种质资源圃，13 个新品种

### 3. 口岸经济

- **富宁港**：年内建成投用
- 拥有 **3 个国家级口岸**（天保、都龙、田蓬）
- 强化与东盟跨境贸易通道
- 天保口岸成为全国第三大陆路榴莲进口口岸
- 2025 年口岸货值 **69.5 亿元**，同比增长 136.7%

### 4. 现代农业 + 康养旅居双轮驱动

- 高原特色现代农业提质增效，构建"1+4+N"产业体系
- 康养旅居产业融合发展
- 普者黑景区业态重塑、坝美景区生态化提升
- 仙人洞、六郎城入选首批"旅居云南"高质量发展试点
- 打造"三七之乡 康养文山"品牌

## 三、项目支撑

- 规划 **2845 个重点项目**
- 总投资规模接近 **万亿**
