# 文山州"十五五"规划纲要核心摘要

## 规划概览
- **正式名称**：《文山壮族苗族自治州国民经济和社会发展第十五个五年规划纲要》
- **审议通过**：2026年1月22日，州第十五届人大六次会议
- **正式印发**：2026年4月10日，由州人民政府印发实施

## 发展定位
- 突出产业、项目、干部、环境"四个重点"
- 持续壮大"四大重点产业"

## 四大产业蓝图

### 1. 绿色铝
- **目标**：打造"绿色铝谷"核心区
- **产业体系**："1+1+3+N"（电解铝+再生铝+3个园区+N个项目）
- **关键指标**：铝合金转化率提升至100%、铝材占比至70%以上

### 2. 三七中药材
- **目标**：打造千亿级产业，成为云南"第二个烟草产业"
- **发力方向**：科研、绿色种植等"七个体系"
- **新赛道**：拓展"药食两用"

### 3. 高原特色现代农业
- **目标**：全国优质绿色农产品生产及加工基地、粤港澳大湾区"菜篮子"基地
- **产业体系**："1+4+N"（粮食+香料、三七、肉牛、蔬菜+N个特色产业）

### 4. 康养旅游旅居
- **品牌**：打造"三七之乡·康养文山"品牌
- **重点项目**：
  - 推进普者黑、坝美等景区升级
  - 与广西、贵州共建红色旅游圈
  - 加快麻栗坡边境旅游试验区建设

## 其他关键领域

| 领域 | 重点方向 |
|------|----------|
| 开放合作 | 打造云南面向东南开放先行区，发展"金色榴莲+银色铝品"边境贸易 |
| 基础设施 | 文蒙铁路、富宁港、清洁能源基地、新型储能、现代水网 |
| 城乡民生 | 新型城镇化、乡村振兴、"幸福文山" |
| 生态文明 | 石漠化治理排头兵、碳达峰 |
| 安全稳定 | 平安文山、法治文山、铸牢中华民族共同体意识 |
