# 文山三七：守护道地药材的生态底色

## 破解连作障碍：老地复种技术突破

三七在生长过程中，根系向土壤分泌自毒物质，土壤微生物群落失衡，导致传统模式下种过三七的土地需间隔**20 年左右**才能再次种植。

文山州采取"政府引导 + 科研攻关 + 企业推广"模式，昆明理工大学联合云南七丹药业等单位，攻克连作障碍重大技术难题，构建起：

> **"病原阻断 → 土壤调理 → 微生物重构 → 综合防控"** 生态种植体系

**"十四五"期间**，全州推广三七老地复种超 **5 万亩**。

| 种植模式 | 规模 | 地位 |

| 林下三七生态种植 | **8,000 余亩** | 全省前列 |

## 全链条升级

2025 年以来，文山州联合龙头企业启动产地仓建设，实现全链条管控：

> 种子种苗 → 订单种植 → 规范种植 → 公开购销 → 产地初加工 → 产地仓储 → 地标赋能

**关键成果**：数智云药（文山）交易中心交易额突破 **31 亿元**。
