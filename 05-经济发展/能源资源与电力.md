# 能源资源与电力

## 总体概况

文山州能源结构以清洁水电为主，风电、光伏为重要补充，是典型的 **"绿电+先进制造"** 模式。截至 2025 年，清洁能源装机占比近 **99.8%**，基本实现全绿电运行。

## 电力装机

| 指标 | 数据 |
|------|------|
| 在运电站 | **212 座** |
| 总装机容量 | **707.31 万 kW**（含分布式光伏） |
| 清洁能源装机 | **705.16 万 kW** |
| 清洁能源占比 | **99.8%** |
| 年发电能力 | 约 **130 亿 kWh** |

### 各电源装机详情

| 电源类型 | 装机（万 kW） | 占比 | 说明 |
|----------|-------------|------|------|
| **水电** | **185.74** | 26.3% | 传统主力，马鹿塘水电站 40 万 kW 为代表 |
| **风电** | **182.18** | 25.8% | 分布在广南、丘北等县 |
| **光伏** | **324.58** | 45.9% | 集中式 273.71 + 分布式 50.87 万 kW |
| **其他** | **14.81** | 2.1% | 生物质 5.15、余热余压 9.66 万 kW |

## 发电量结构（2025 年）

2025 年规上工业发电量 **110.83 亿 kWh**，同比增长 19.0%。

| 电源 | 发电量（亿 kWh） | 占比 |
|------|-----------------|------|
| 水力发电 | 61.00 | ~55% |
| 风力发电 | 29.78 | ~27% |
| 太阳能发电 | 15.75 | ~14% |
| 火力发电 | 4.31 | ~4% |

## 2026年Q1发电量

2026年一季度，全州规模以上工业发电量 **31.92 亿 kWh**，同比增长 **30.2%**。清洁电力占比 **95.3%**，比上年同期提高 **1.5 个百分点**。

> 来源：云南日报 2026-04-29《文山州一季度经济运行态势良好》

## 绿色铝产业支撑

| 指标 | 数据 |
|------|------|
| 电解铝产能 | **343 万吨**（占全国近 1/10） |
| 铝加工产能 | **407.4 万吨** |
| 2025 年绿色铝产值 | **1000 亿元**（首个千亿级产业） |
| 电解铝绿电使用比例 | **超过 80%** |
| 每吨铝减排 CO₂ | 约 **8-10 吨**（相比火电炼铝） |
