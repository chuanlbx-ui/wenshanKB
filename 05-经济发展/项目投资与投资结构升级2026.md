# 文山州项目投资与投资结构升级（2026年）

> **数据来源**：文山发布网《奋进文山》综合报道（2026-06-03） + 央视网新闻发布会报道（2026-06-03）

---

## 一、"十四五"投资总盘点

| 指标 | 数据 |
|------|------|
| 累计实施重点项目 | **1876 个** |
| 完成投资总额 | **3880.46 亿元** |
| 对比"十三五" | 逆势正增长 |
| 产业投资占比变化 | **32% → 62%** |
| 工业投资占比变化 | **19.6% → 42.6%** |
| 民间投资活力 | 持续增强 |
| 累计争取上级资金 | **352.48 亿元** |

> **投资结构实现历史性优化**：彻底改变以往依靠基建投资拉动的旧模式，产业投资成为拉动经济增长的绝对主力。

## 三、2026年"项目攻坚年"

### 3.1 攻坚机制

- 全州紧扣**"项目攻坚年"**，细化**九大攻坚任务**
- 州级领导包保 + 清单化推进 + 常态化调度
- 形成"开工一批、建设一批、投产一批、储备一批"良性循环

| 指标 | 数据 |

## 六、关键洞察

1. **投资驱动力切换**：产业投资占比从 32% 跃升至 62%，工业投资从 19.6% 跃升至 42.6%，标志着文山经济从基建拉动型向产业驱动型成功转型。
2. **项目梯度储备充足**："十五五"储备项目总投资超 1.6 万亿，为持续增长提供坚实底座。
3. **能源基础设施同步**：新能源并网 520 万千瓦 + 国家级储能示范 + 天然气管网贯通，为绿色铝等产业提供稳定、低成本的能源保障。
4. **交通瓶颈加速突破**：2026-2030 年富宁港→文蒙铁路→文靖铁路梯次投运，对外开放通道全面升级。
