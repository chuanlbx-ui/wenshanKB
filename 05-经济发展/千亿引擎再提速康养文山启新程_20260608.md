# 千亿引擎再提速 康养文山启新程

> **来源**：云南网·文山频道 | **发布时间**：2026-06-08
> **原标题**：新时代 新征程 新伟业丨千亿引擎再提速 康养文山启新程

## 一、绿色铝：以"绿电+先进制造"提速千亿产业

### 产业地位

| 指标 | 数据 |
|------|------|
| 电解铝产能占全省 | **52.8%** |
| 电解铝产能占全国 | **7.7%** |
| 2025年绿色铝产值 | 突破**1000亿元**（锚点值1016亿） |
| 铝加工产能 | **407.4万吨** |
| 合金化率 | **69.92%** |

### 重点企业动态

- **云南神火新材料**：研发生产 **0.0045毫米** 超薄铝箔坯料，依托绿电保障
- **云南创格新材料**（砚山产业园区）：6台45吨熔炼炉配电磁搅拌+炉外精炼技术，产品覆盖新能源汽车、机器人制造、储能领域

### 在建/谋划重点项目

- 中铝20万吨绿色低碳铸轧
- 砚山智铝二期
- 云南神火超薄铝箔
- 文山华疆再生铝循环利用

十五五目标：力争**十五五末**铝产业综合产值达 **1800亿元**（锚点：当前1016亿元）。

## 四、高原特色农业：以"菜篮子"擦亮金名片

### 产业基础

- 粤港澳大湾区"菜篮子"基地，全年可生产优质生态蔬菜
- 蔬菜种植面积稳居全省前列

### 十五五产业布局

"1+4+N"现代高原特色农业产业体系：
- **1**：稳住粮食生产基本盘
- **4**：三七中药材、蔬菜、水果、肉牛四大优势特色产业
- **N**：茶叶、花卉、甘蔗、畜禽、水产、八角、草果等特色集群

重点推进项目：砚山蔬菜供应基地、蓝莓产业园、广南铁皮石斛种植加工、畜禽精深加工。

十五五目标：力争**十五五末**以绿色食品为代表的现代农业总产值超 **550亿元**。
