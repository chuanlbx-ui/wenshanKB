# 外贸发展

## 2025年概况

- 外贸进出口总值：**40.2 亿元**（出口 9.89 亿 / 进口 30.27 亿），增长 **42.9%**（来源：2025年统计公报）
- 进出口额超 200 万元以上企业达 **53 户**（来源：文山州发改委巡察整改通报 2026-06-04）
- 成功举办中越边交会、"文山·睦邻"交流合作周等品牌活动
- 招引省外产业项目 140 个，到位资金 158.7 亿元

## 主要出口品类

| 品类 | 亮点 |
|------|------|
| 绿色铝产品 | 铝制品出口实现新突破 |
| 三七及中药材 | 产品辐射 40 余国 |
| 高原果蔬 | 粤港澳大湾区蔬菜占全省 13% |
| 鲜切花 | 2026年一季度实现产业破零，出口46万枝/70万元（日本） |

## 口岸建设

- 天保口岸国际货场投入使用，成为全国 **第三大陆路榴莲进口口岸**
- 3 个口岸功能完善，整体通关效率提升 **70%** 以上
- 榴莲进口实现新突破

## 2026年一季度动态

- 文山天华园艺（马关基地）菊花出口日本，填补鲜切花对外出口产业空白
- 一季度出口鲜切花 46 万枝，带动 300 名群众就地就近就业
