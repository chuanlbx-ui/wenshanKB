# 大湾区"菜篮子"基地建设

> **来源**：科技日报 2026-06-07 | **记者**：赵汉斌

## 一、战略定位

文山州是云南通往粤港澳大湾区最便捷的通道，将打造粤港澳大湾区"菜篮子"基地作为高原特色现代农业发展的重要抓手。依托冷链物流，菜心、生姜、辣椒等新鲜食材仅需十余小时即可从田间直达大湾区市场。

## 二、产业规模（2025年）

| 指标 | 数值 |
|------|------|
| 蔬菜种植面积 | **281.57 万亩** |
| 蔬菜总产量 | **227.6 万吨** |
| 辣椒种植规模 | **全省第一、全国前列** |

## 三、砚山标准化基地

- 砚山等地规模化蔬菜基地连片发展
- 建成 **11.3 万亩** 连片标准化基地
- 带动群众持续增收

## 四、产业体系

加快构建 **"1+4+N"高原特色现代农业产业体系**：

| 板块 | 内容 |
|------|------|
| 1（粮食） | 粮食安全基础 |
| 4（四大优势） | 蔬菜、肉牛奶牛、香料佐料 |
| N（多元特色） | 地方特色产业 |

推动一二三产深度融合，目标：全国优质绿色现代农产品生产基地 + 绿色食品加工基地。

## 五、自然禀赋

文山州海拔适中、雨量充沛，孕育品类丰富、品质优良的高原农产品。紧邻粤港澳大湾区消费大市场，区位优势明显。
