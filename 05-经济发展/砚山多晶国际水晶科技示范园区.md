# 砚山多晶国际水晶科技示范园区一期投产

> 来源：云南日报（云新闻） | 发布时间：2026-06-13
> 记者：李彤 | 通讯员：王思入 李文雄
> 地点：文山州砚山县文砚大道（子马组团）

## 一、项目概况

| 指标 | 数据 |
|------|------|
| 项目名称 | 多晶国际水晶科技示范园区 |
| 投资主体 | **云南金石科技有限公司** |
| 占地面积 | **784 亩**（分两期推进） |
| 产品方向 | 精品特A水晶钻石 |
| 全面建成年产能 | **3 万吨** |
| 全面建成年产值 | **28 亿元** |

## 二、一期投产详情

| 指标 | 数据 |
|------|------|
| 一期产能目标 | 年产 **1.8 万吨** |
| 首批投产时间 | **2026年5月1日** |
| 首批投产规模 | **100 个机组** |
| 日产量 | 约 **10 吨** |
| 第二批投产计划 | **100 个机组**，预计 **2026年6月底至7月初** 逐步投产 |
| 电力保障 | **110千伏**电力接入后产能将进一步提升 |

## 三、运营模式

创新推行 **"边调试、边生产"** 并行模式：

- 设备安装、工艺打磨与试产工作同步推进
- 设备调试车间同步进行精细化组装、参数优化与性能检测
- 生产车间自动化设备有序运转，批量试产、质量检验与工艺完善同步进行
- 经过打磨塑形、清水清洗、精细抛光等标准化流程，下线水晶坯料质感温润通透

> "我们5月1日开始投产的100个机组，日产约10吨。目前技术、人员配备齐全，110千伏电一通，产能还将提升。第二批100个机组计划6月底7月初逐步投产。"
> —— 云南金石科技有限公司厂区生产负责人 张建文

## 四、配套设施

| 配套项目 | 进展 |
|----------|------|
| 厂区市政道路 | 已全部完工 |
| 生活污水管网 | 已全部完工 |
| 13公里工业污水管网 | 已全部完工 |
| 32座塔基 | 已全部完工 |
| 1.2公里电缆沟 | 已竣工，高压线已全部拉通 |
| 剩余1.2公里电缆 | 计划6月20日前完工，6月21日验收，6月25日送电 |

> "目前32座塔基已全部完工，1.2公里的电缆沟也已竣工，塔基的高压线已全部拉通。剩余的1.2公里电缆，计划于6月20日前全部完工，6月21日完成验收，于6月25日投入送电，助推企业一期达产。"
> —— 项目开发建设指挥部办公室专职副主任 农斌

## 五、营商环境

项目高效推进得益于砚山县持续优化的营商环境提供的坚实保障。项目指挥部统筹推进，确保基础设施配套与生产需求同步匹配。
